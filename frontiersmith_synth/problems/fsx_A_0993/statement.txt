# Sampler Quilt: No Two Blocks Repeat

## Problem

You are piecing a sampler quilt: a `B x B` grid of blocks, each block a `b x b`
grid of half-square-triangle cells. Every cell is split by one diagonal into two
triangles: a **diagonal orientation** `d in {0,1}` (`0` = `\`, `1` = `/`) and two
**colors** `c1,c2` from a palette of `c` colors (`c1 != c2`).

A block's **pattern** is the `b x b` matrix of its cells' diagonal orientations
(colors do not affect pattern identity). Two blocks are **dihedrally equivalent**
if one pattern is obtained from the other by a rotation and/or reflection of the
square (the 8 symmetries of the square). Let `M` be the number of distinct
equivalence classes among the `B*B` block patterns you place. A **quota** `Q`
sets how many classes are worth crediting.

Adjacent blocks share a seam. At a horizontal seam between block `(r,k)` and
`(r,k+1)`, row `i` of the left block's rightmost cell-column meets row `i` of
the right block's leftmost cell-column, for every `i`; the seam **continues**
there iff the two cells' diagonal orientations are equal (the fabric's bias
looks unbroken across the stitch). Vertical seams between `(r,k)` and `(r+1,k)`
work the same way, column by column. Let `S` be the total number of continuing
seam positions over all seams.

Finally, the dye lot must stay balanced: `2*B*B*b*b` triangles are colored in
all; for every color `k` its total triangle count must lie within a tolerance
`tol` of the even share, checked as `|count_k * c - 2*B*B*b*b| <= tol * c`.

## Input

One line: `B b c Q tol W1` — grid size, block size, palette size, distinctness
quota, color tolerance, and the per-quota-class weight (see Scoring).

## Output

Print exactly `B*B*b*b` triples `d c1 c2`, in this fixed order: for block-row
`r = 0..B-1`, block-col `k = 0..B-1`, cell-row `i = 0..b-1`, cell-col
`j = 0..b-1` — one triple per cell, whitespace-separated (any layout).

## Feasibility

Every `d` in `{0,1}`; every `c1,c2` in `[1,c]` with `c1 != c2`; the color-balance
inequality above must hold for every color `1..c`; the output must contain
exactly `B*B*b*b` triples and nothing else. Any violation scores `0`.

## Objective

Maximize `F = W1 * min(M, Q) + S`.

## Scoring

The checker computes a fixed reference `Bref = W1 + Smax`, where
`Smax = 2*B*(B-1)*b` is the maximum possible `S` (every seam position
continuing). Your score for a test is
`ratio = min(1000, 100 * F / max(1, Bref)) / 1000`,
so a quilt with only 1 distinct pattern class but every seam continuing scores
exactly `0.1`; real improvement scores higher, capped at `1.0`. Per-test ratios
are averaged.

## Constraints

`2 <= B <= 12`, `3 <= b <= 7`, `3 <= c <= 8`, `1 <= Q <= B*B`, `0 <= tol <= 10000`,
`0 <= W1 <= 100000`. Time limit: 3 s. Memory limit: 512 MB.

## Example

`B=2 b=3 c=3 Q=2 tol=6 W1=10`: `Smax = 2*2*1*3 = 12`, `Bref = 10+12 = 22`.
Make every border cell (the 8 cells of each `3x3` block other than the exact
center) `d=0` in all 4 blocks — this guarantees every one of the 12 seam
positions continues, `S = 12 = Smax`, since a block's right/bottom edges are
then identical (all `0`) to its neighbors' left/top edges. Only the center cell
is free: set it `0` for the two blocks on the main diagonal and `1` for the
other two. The all-`0` block and the block with a `1` center are **not**
dihedrally equivalent (the center is fixed by every symmetry, so its value is
an invariant) — `M = 2`, which meets the quota exactly. Colors are assigned in
a balanced round-robin so each of the 3 colors totals 24 triangles, well inside
`tol=6`. `F = 10*min(2,2) + 12 = 32`, `ratio = min(1000, 100*32/22)/1000 =
0.1454`.

Two failure modes bracket this: an independently chosen, neighbor-blind
pattern per block maximizes `M` but leaves seams matching only by chance (`S`
collapses toward half of `Smax`); rotating or reflecting one fixed tile to line
seams up everywhere maximizes `S`, but every copy lands in the *same*
equivalence class (`M` collapses to `1` — rotation/reflection is exactly what
the equivalence quotients out). Fix the shared seam pattern along each border
first, then vary only the cells free of any border.
