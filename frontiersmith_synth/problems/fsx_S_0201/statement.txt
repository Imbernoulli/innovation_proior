# Static on the Long Arm: Interstellar Relay Channel Assignment

## Problem

The **Long Arm** is a chain of deep-space relay stations that carries traffic from the
inner colonies out to the frontier. Every station transmits continuously, and the mission
council has leased only a small block of **`C` broadcast channels** for the entire array.
Whenever two stations whose signals overlap are tuned to interfering channels, their
crosstalk corrupts the relayed packets and burns mission budget.

You must assign each station a channel from the leased block so that the **total
interference cost** across the array is as small as possible.

Two stations are said to **overlap** if they appear as an interfering pair in the input;
each such pair `(u, v)` carries a positive **interference weight** `w`. Given the channels
`c_u` and `c_v` finally assigned to the two stations, the pair contributes:

- `w`            if `c_u == c_v`            (co-channel: full crosstalk),
- `floor(w / 2)` if `|c_u - c_v| == 1`      (adjacent-channel: leakage crosstalk),
- `0`            otherwise                  (well separated: clean).

Minimizing the total interference cost over all interfering pairs is an instance of the
frequency-assignment / minimum-conflict graph-labeling problem: it is NP-hard, the optimum
is unknown in general, and greedy, sequential min-conflict, and local-search / random-restart
strategies all diverge in quality.

## Input

The first line contains three integers:

```
N M C
```

- `N` — number of relay stations, numbered `1..N`.
- `M` — number of interfering pairs.
- `C` — number of leased broadcast channels (channels are numbered `1..C`).

Each of the next `M` lines describes one interfering pair:

```
u v w
```

meaning stations `u` and `v` overlap with interference weight `w` (`1 <= w`). The same
unordered pair may appear more than once; each listed occurrence contributes independently.
It is guaranteed that `u != v` and `M >= 1`.

## Output

Print `N` lines. The `i`-th line contains a single integer `c_i` in `1..C`: the channel
assigned to station `i`.

Assigning every station channel `1` is always valid.

## Feasibility

An output is feasible iff it contains exactly `N` integers, each in the range `1..C`, with
no extra trailing tokens. Any infeasible output scores `0`.

## Objective

Let `F` be the total interference cost of your assignment:

```
F = sum over interfering pairs (u,v) of:
      w            if c_u == c_v
      floor(w/2)   if |c_u - c_v| == 1
      0            otherwise
```

You are **minimizing** `F`.

## Scoring

Let `B` be the interference cost of the trivial assignment that tunes **every** station to
the same channel (every overlapping pair is co-channel, so `B = sum of all w`). The score
for a test is

```
ratio = min(1.0, 0.1 * B / max(1, F))
```

The all-same-channel submission (`F = B`) scores `0.1`; cutting interference to a tenth of
`B` or better caps the score at `1.0`. Reaching `F = 0` (a perfectly clean assignment) also
caps at `1.0`. The overall score is the mean ratio across all tests.

## Constraints

- `2 <= N <= 200`.
- `1 <= M <= 4000`.
- `2 <= C <= 6`.
- `1 <= w <= 100`.
- Time limit: 5 s. Memory limit: 512 MB.

## Example

Input:

```
4 5 2
1 2 10
2 3 8
3 4 6
1 4 4
1 3 5
```

Every station on channel `1` gives `B = 10 + 8 + 6 + 4 + 5 = 33`.

Consider the assignment (station 1 -> 4 in order):

```
1
2
1
2
```

Pair costs: `(1,2)` channels `1,2` -> `floor(10/2)=5`; `(2,3)` `2,1` -> `floor(8/2)=4`;
`(3,4)` `1,2` -> `floor(6/2)=3`; `(1,4)` `1,2` -> `floor(4/2)=2`; `(1,3)` `1,1` -> `5`.
So `F = 5 + 4 + 3 + 2 + 5 = 19`.

Score for this test: `min(1.0, 0.1 * 33 / 19) = min(1.0, 0.173684) = 0.173684`.
