# Reef Hydrophone Channel Assignment: Silencing Survey Interference

## Problem

The *Coral Sentinel* survey deploys a large array of autonomous **hydrophones** across a
reef to log fish choruses and boat traffic. Each hydrophone pings on one of `K` acoustic
**channels**. When two hydrophones whose survey cones overlap ping on channels that are
too close together, their recordings smear into each other and corrupt the survey.

For every interfering pair of hydrophones `(u, v)` the survey team knows two numbers: an
**interference weight** `w` (how badly that overlap hurts the data) and a **required
channel separation** `d` (how many channels apart the two pings must be to be clean). If
you assign channel `c_u` to `u` and `c_v` to `v`, that pair contributes an interference
penalty of

```
w * max(0, d - |c_u - c_v|)
```

So a pair on the *same* channel pays the full `w * d`; each extra channel of separation you
buy back shaves off `w` of penalty, until at separation `d` the pair is completely clean.

You must assign every hydrophone a channel in `1..K` so that the **total interference**
summed over all pairs is as **small** as possible. Only `K` channels are licensed, so you
cannot simply spread every hydrophone onto its own frequency. This is a minimum-interference
frequency-assignment problem (a weighted generalization of graph coloring): it is NP-hard,
the optimum is unknown, and greedy, local-search, and randomized-restart strategies all
diverge.

## Input

The first line contains three integers:

```
n m K
```

- `n` — number of hydrophones, numbered `1..n`.
- `m` — number of interfering pairs.
- `K` — number of available channels (channels are numbered `1..K`).

Each of the next `m` lines describes one interfering pair:

```
u v w d
```

meaning hydrophones `u` and `v` (`u != v`) interfere with weight `w` (`1 <= w`) and require
a channel separation of `d` (`1 <= d`). A pair `(u, v)` may appear more than once (with
possibly different `w`, `d`); every listed pair contributes its penalty independently.

## Output

Print `n` integers `c_1 c_2 ... c_n`, separated by whitespace, where `c_i` is the channel
assigned to hydrophone `i`. Each `c_i` must satisfy `1 <= c_i <= K`.

Printing `1` for every hydrophone is always valid.

## Feasibility

An output is feasible iff it consists of exactly `n` integers and each lies in `1..K`.
Any infeasible output scores `0`.

## Objective

Let `F` be the total interference of your assignment:

```
F = sum over all pairs (u, v, w, d) of  w * max(0, d - |c_u - c_v|)
```

You are **minimizing** `F`.

## Scoring

Let `B` be the interference of the trivial all-on-channel-1 assignment (`c_i = 1` for all
`i`), i.e. `B = sum over all pairs of w * d` (every pair pays its full same-channel
penalty). The score for a test is

```
ratio = min(1.0, 0.1 * B / max(1, F))
```

An all-on-one-channel submission (`F = B`) scores `0.1`; driving the total interference
down to a tenth of the baseline (`F <= B/10`), or to zero, caps the score at `1.0`. The
overall score is the mean ratio across all tests.

## Constraints

- `2 <= n <= 3000`.
- `1 <= m <= 40000`.
- `2 <= K <= 12`.
- `1 <= w <= 20`, `1 <= d <= 3`.
- Time limit: 5 s. Memory limit: 512 MB.

## Example

Input:

```
4 4 3
1 2 5 2
2 3 4 1
3 4 6 2
1 3 2 1
```

Baseline (all on channel 1): every pair pays `w * d`, so
`B = 5*2 + 4*1 + 6*2 + 2*1 = 10 + 4 + 12 + 2 = 28`.

Consider the assignment `c = [1, 3, 1, 3]`:

- pair `(1,2)` w=5 d=2: `|1-3|=2 >= 2` -> `5*max(0,2-2)=0`
- pair `(2,3)` w=4 d=1: `|3-1|=2 >= 1` -> `0`
- pair `(3,4)` w=6 d=2: `|1-3|=2 >= 2` -> `0`
- pair `(1,3)` w=2 d=1: `|1-1|=0` -> `2*max(0,1-0)=2`

So `F = 2`, and the score is `min(1.0, 0.1 * 28 / 2) = min(1.0, 1.4) = 1.0`.

A valid worked output:

```
1 3 1 3
```
