We are given up to n = 2000 boxes. Box i has dimensions w_i, d_i, h_i, and may be rotated so that any one dimension becomes its vertical height while the other two form its rectangular base. A box can be placed on another only when both base dimensions are strictly smaller than the base dimensions below it. Each input box is treated as usable once in a stack. We must output the maximum possible total stack height. Dimensions are as large as 10^9, so the answer requires 64-bit arithmetic. The solution is scored by exact equality with the optimal maximum height.
