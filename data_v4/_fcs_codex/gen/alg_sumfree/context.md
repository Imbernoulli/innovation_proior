We are given a multiset `S` of `n` positive integers, with `1 <= n <= 100000` and values up to `10^9`. We must output a sub-multiset `T` using occurrences from `S` such that no chosen values `a, b, c` satisfy `a + b = c`, allowing `a` and `b` to have the same value. Any valid output is accepted, and the score is `|T|`, so the goal is to find a large valid subset, not necessarily prove optimality.
