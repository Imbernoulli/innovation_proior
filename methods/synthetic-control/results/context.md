## Aggregate Intervention Problem

A policy, conflict, migration shock, or institutional change sometimes affects one city, state, country, firm, or region at a known point in time. The estimand is a post-event gap for that treated aggregate unit, but the untreated version of the same unit is never observed. The core design problem is therefore not a missing coefficient; it is a missing outcome path.

## Available Comparisons

The available evidence usually consists of panel data: one treated unit, several untreated units, pre-event outcomes, post-event outcomes, and predictors measured before the event. Each untreated unit is an imperfect comparison. Some share economic structure, some share outcome history, and some share covariates, but none may be close enough on all relevant dimensions.

## Existing Designs

Earlier comparative case studies choose a control region or a small set of comparison regions and examine before-after differences. Difference-in-differences formalizes that move by differencing out common shocks, but it relies on the comparison path being credible after observed and unobserved differences are accounted for. Regression can add covariates, but it can also hide extrapolation behind fitted coefficients.

## Credibility Requirements

A credible counterfactual must be chosen without using the treated unit's post-event outcomes. It must show that the comparison object resembles the treated unit before the event, especially on outcomes and predictors that forecast later outcomes. It should also make visible which untreated evidence carries the comparison, so readers can judge whether the comparison is substantively plausible.

## Evaluation Criteria

The design should produce a clear post-event treatment-effect path, a transparent pre-event fit assessment, and an inference procedure suitable for a small number of aggregate units. It should avoid relying solely on large-sample standard errors, because the main uncertainty is whether the untreated units can reproduce the missing counterfactual path for the treated unit.
