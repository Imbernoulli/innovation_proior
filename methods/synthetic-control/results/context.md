## Aggregate Intervention Problem

A policy, conflict, migration shock, or institutional change sometimes affects one city, state, country, firm, or region at a known point in time. The estimand is a post-event gap for that treated aggregate unit, but the untreated version of the same unit is never observed. The core design problem is therefore not a missing coefficient; it is a missing outcome path.

## Available Comparisons

The available evidence usually consists of panel data: one treated unit, several untreated units, pre-event outcomes, post-event outcomes, and predictors measured before the event. Each untreated unit is an imperfect comparison. Some share economic structure, some share outcome history, and some share covariates.

## Existing Designs

Earlier comparative case studies choose a control region or a small set of comparison regions and examine before-after differences. Difference-in-differences formalizes that move by differencing out common shocks, relying on the chosen comparison path. Regression can add covariates to the comparison.

## Research Question

How can the untreated counterfactual path for a single treated aggregate unit be constructed from a panel of untreated comparisons?
