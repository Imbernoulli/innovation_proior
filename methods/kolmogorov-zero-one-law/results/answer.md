# Kolmogorov's zero-one law

## The problem it solves

Many probabilities of *limiting* events — convergence of a series of independent terms, the long-run frequency of digits, the strong-law limit `Sₙ/n → c`, the radius of convergence of a random power series, `{Aₙ infinitely often}` — come out equal to exactly `0` or exactly `1`, never anything strictly between. Before this law each was a separate computation. The zero-one law identifies the one structural feature responsible and proves the dichotomy once, for every event with that feature.

## Key idea

All such events share one property: **they are insensitive to any finite prefix of the sequence** — changing finitely many of the `Xₙ` does not change whether the event occurs. The events with this property form the **tail σ-field**

`𝔗 = ⋂ₙ σ(X_{n+1}, X_{n+2}, …)`.

When the `Xₙ` are independent, a tail event `A` is independent of every finite prefix `σ(X₁,…,Xₙ)`; the prefixes generate the entire sequence σ-field `σ(X₁,X₂,…)`, and independence extends from these generators to that whole σ-field. But the tail σ-field is *contained* in `σ(X₁,X₂,…)`, so `A` is independent of itself. Independence of an event with itself gives `P(A) = P(A∩A) = P(A)²`, whose only roots in `[0,1]` are `0` and `1`.

## Theorem

**Zero-or-one law (general form).** Let `x₁, x₂, …` be random variables on a probability space with completely additive `P`, let `f` be a Baire function of `(x₁, x₂, …)`, and set `A = {f(x) = 0}`. If for every `n`

`P_{x₁,…,xₙ}{f(x) = 0} = P{f(x) = 0}`

(equivalently, `A` is independent of `σ(x₁,…,xₙ)` for each `n`), then

`P(A) ∈ {0, 1}`.

**Special case (the usual statement).** If the `Xₙ` are mutually independent, then every event `A` in the tail σ-field `𝔗 = ⋂ₙ σ(X_{n+1}, X_{n+2}, …)` has `P(A) ∈ {0, 1}`.

**Corollary.** Any `𝔗`-measurable random variable `Y` is almost surely constant: `P(Y = c) = 1` for `c = inf{ y : P(Y ≤ y) = 1 }`.

## Proof

### Route 1 — uniqueness of the measure extension

Let `𝔉` be the field of events describable through finitely many coordinates (the cylinder field); its Borel extension is `B𝔉 = σ(x₁, x₂, …)`, and `A ∈ B𝔉` since `f` is a Baire function of all coordinates.

If `P(A) = 0`, done. Otherwise `P(A) > 0`, and `P_A(·) = P(A∩·)/P(A)` is itself a probability measure (non-negative, `P_A(E)=1`, completely additive, inherited from `P`). For any `B ∈ 𝔉`, `B` lies in some `σ(x₁,…,xₙ)`, and the hypothesis gives independence `P(A∩B) = P(A)P(B)`, so

`P_A(B) = P(A∩B)/P(A) = P(A)P(B)/P(A) = P(B)`.

Thus the completely additive measures `P_A` and `P` agree on the field `𝔉`. By the **uniqueness** of the extension of a completely additive set function from a field to its Borel extension, they agree on all of `B𝔉`. Since `A ∈ B𝔉`, set `B = A`:

`P_A(A) = P(A)`, but `P_A(A) = P(A∩A)/P(A) = P(A)/P(A) = 1`,

so `P(A) = 1`. In all cases `P(A) ∈ {0,1}`. ∎

The decisive ingredient is **countable additivity**: it is what makes the extension unique, hence what transports the equality `P_A = P` from finitely-described events out to the limiting event `A`. Without it (finite additivity only) the extension is not unique and a tail event can carry a value in `(0,1)`.

### Route 2 — independence of the tail from itself (π-system form)

Set `Fₙ = σ(X₁,…,Xₙ)`, `F∞ = σ(X₁,X₂,…)`, `Tₙ = σ(X_{n+1}, X_{n+2}, …)`, `T = ⋂ₙ Tₙ`.

- `Fₙ` is generated by the π-system `{X₁ ≤ x₁, …, Xₙ ≤ xₙ}`; `Tₙ` by the π-system `{X_{n+1} ≤ y₁, …, X_{n+k} ≤ yₖ}`, `k ∈ ℕ`. By independence of the variables, `P(A∩B) = P(A)P(B)` for any such generators, so by the π–λ theorem `Fₙ` and `Tₙ` are independent. Since `T ⊆ Tₙ`, `Fₙ ⟂ T` for every `n`.
- `⋃ₙ Fₙ` is a π-system (an increasing union of σ-fields) generating `F∞`. `T` independent of every `Fₙ` is `T` independent of this π-system, so by π–λ, `T ⟂ F∞`.
- But `T ⊆ F∞`, so for `A ∈ T`, `A` is independent of itself: `P(A) = P(A∩A) = P(A)²`, hence `P(A) ∈ {0,1}`. ∎

The two routes are the same fact: "unique extension of a measure from a field" and "extension of independence from a generating π-system" are both the statement that countable additivity pins a measure down from its values on a generating, intersection-closed family.

### Corollary (a.s. constant)

If `Y` is `T`-measurable, then `{Y ≤ y} ∈ T`, so `F_Y(y) = P(Y ≤ y) ∈ {0,1}` for all `y`. Let `x₀ = inf{c : P(Y ≤ c) = 1}` and `x₁ = ` smallest `c` with `P(Y > c) = 0`. If `x₀ < x₁`, pick `ε > 0` with `x₀ < x₁ - ε < x₁`; then `P(Y > x₁-ε) = 1` and `P(Y ≤ x₁-ε) = 1`, forcing `P(E) = 2`, impossible. If `x₀ > x₁`, then `P(Y ≤ x₁) = 1`, contradicting the infimum defining `x₀`. So `x₀ = x₁ =: c`, giving `P(Y < c) = P(Y > c) = 0`, i.e. `P(Y = c) = 1`. ∎

## Why each choice is forced

- **Independence is essential.** Without it the front block need not be independent of the tail blocks, `T ⟂ F∞` fails, self-independence is lost, and a tail event (e.g. eventual absorption of a Markov chain in one state vs. another) can have probability in `(0,1)`.
- **The tail σ-field is the exact class.** It is precisely the events insensitive to every finite prefix — the common feature of all the `0`-`1` phenomena. An event depending on an early coordinate (e.g. `{X₁ = 1}`, probability `1/2`) is not in `T`, and the law correctly says nothing about it.
- **Countable additivity / the continuity axiom is essential.** It is the lever transporting an equality from finitely-generated events to a limiting event; on a finitely-additive set function the conclusion fails.
- **Conclusion is `{0,1}` and nothing between.** `A ∩ A = A` is a trivial set identity; independence turns it into the numeric `P(A) = P(A)²`, whose only fixed points in `[0,1]` are `0` and `1`.

## Canonical instances (each a tail event of an independent sequence)

- **Convergence of `∑ₙ Xₙ`** (independent `Xₙ`): changing finitely many terms cannot change convergence ⇒ tail event ⇒ probability `0` or `1`.
- **Strong-law limit:** `{Sₙ/n → c}` is a tail event; `limsup Sₙ/n` is `T`-measurable ⇒ almost surely a deterministic constant.
- **Radius of convergence of `∑ₙ Xₙ zⁿ`:** `R = 1/limsupₙ |Xₙ|^{1/n}` is `T`-measurable (a `limsup` ignores any finite block) ⇒ almost surely a constant.
- **Normal numbers (Borel):** `{ (1/n)∑_{k≤n} d_k → 1/2 }` for i.i.d. fair bits is a tail event ⇒ probability `0` or `1`; Borel's strong law identifies it as `1`.
- **`{Aₙ infinitely often}`** for independent `Aₙ`: a tail event ⇒ probability `0` or `1` (the Borel–Cantelli dichotomy is the special case decided by `∑ P(Aₙ)`).
- **Percolation on `ℤ²`:** `{∃ infinite open cluster}` is invariant under changing finitely many edges ⇒ tail event of the independent edge variables ⇒ probability `0` or `1`.
