In a long-haul coherent WDM optical link, the received SNR on each channel is set by two impairments: the ASE noise added by inline amplifiers, and the Kerr nonlinearity of the fiber. ASE is straightforward to write down from gain and noise figure, but the Kerr effect couples every frequency to every other through four-wave mixing, and in a dense WDM comb the number of mixing products is too large to enumerate. Existing approaches are either too slow or too restricted. A full split-step simulation numerically integrates the NLSE or Manakov equation and is essentially exact, yet each configuration takes minutes to hours, and it returns only a number rather than a differentiable formula. Classical FWM-among-CW-tones analysis gives closed-form phase-matching efficiencies, but it is built for a handful of unmodulated carriers, not for hundreds of modulated channels with unequal powers, bandwidths, and flexible-grid spacing. Perturbation analysis is the right starting point, yet prior closed forms covered only identical spans and equally spaced channels, not the flexible-grid case a network optimizer needs.

The way forward is to exploit what uncompensated propagation does to the waveform. In modern coherent systems, chromatic dispersion is not undone optically along the fiber; it is undone digitally at the receiver. Because dispersion is linear and invertible, it is left to run free over the line, and each channel's pulses spread over thousands of symbol slots within a short distance. By the central-limit tendency the field statistics approach those of a complex Gaussian process whose only relevant descriptor is its average power spectral density. The signal Gaussianizes after launch, and the Kerr nonlinearity then acts on that Gaussian field. The four-wave mixing generated by such a field produces an enormous number of uncorrelated in-band beat contributions, and their sum is itself additive Gaussian noise, statistically independent of ASE. That justifies adding variances in the denominator of the SNR.

The method is the Gaussian-Noise (GN) model of nonlinear propagation. It treats the uncompensated WDM signal as a Gaussian process, applies a first-order regular perturbation to the NLSE or Manakov equation, and averages over the random data symbols. The result is a closed-form double integral, the GN reference formula, that maps the transmitted PSD directly to the NLI PSD on each channel. For identical spans the formula includes the classical FWM efficiency factor per span and a coherent multi-span phased-array factor; for network planning the large-dispersion limit averages the phased-array term and NLI accumulates linearly with the number of spans. Because the spectrum enters the integrand cubed, the NLI power on a channel scales as the cube of its launch power, giving a clean P_NLI = eta * P^3 law with eta determined only by the link geometry and spectral shapes. That cubic law yields an optimum launch power where ASE is exactly twice the NLI power, P_opt = (P_ASE / (2 * eta))^(1/3), and it makes per-channel SNR a cheap, differentiable function suitable for power and spectrum optimization.

For fast evaluation, the double integral is further collapsed by modeling each Nyquist channel as a flat-top rectangle and keeping only self-channel interference and cross-channel interference, dropping the negligible multi-channel-interference terms. Each SCI or XCI island then evaluates in closed form to an inverse-hyperbolic-sine expression, so the NLI on a channel becomes a sum over pairs of channels rather than a full 2D integral. This rectangular approximation is what makes the GN model callable millions of times inside a launch-power or routing optimizer.

```python
import numpy as np

SPM_WEIGHT = 16.0 / 27.0
XPM_WEIGHT = 2.0 * 16.0 / 27.0

class Channel:
    def __init__(self, f0, baud_rate, power, roll_off=0.0):
        self.f0 = f0
        self.baud_rate = baud_rate
        self.power = power
        self.roll_off = roll_off

class Span:
    def __init__(self, length, alpha, beta2, gamma, beta3=0.0):
        self.length = length
        self.alpha = alpha
        self.beta2 = beta2
        self.beta3 = beta3
        self.gamma = gamma

def effective_length(alpha, length):
    return (1.0 - np.exp(-alpha * length)) / alpha

def asymptotic_length(alpha):
    return 1.0 / alpha

def ase_power(gain, noise_figure, nu, bandwidth):
    h = 6.62607015e-34
    F = 10 ** (noise_figure / 10.0)
    return F * (gain - 1.0) * h * nu * bandwidth

def interaction_factor(df, baud_cut, baud_pump, beta2, L_eff, L_a):
    abs_beta2 = abs(beta2)
    right = df + baud_pump / 2.0
    left = df - baud_pump / 2.0
    val = (np.arcsinh(np.pi**2 * L_a * abs_beta2 * baud_cut * right) -
           np.arcsinh(np.pi**2 * L_a * abs_beta2 * baud_cut * left)) / 2.0
    val *= L_eff**2 / (2.0 * np.pi * abs_beta2 * L_a)
    return val

def coefficient_matrix(channels, span):
    n = len(channels)
    L_eff = effective_length(span.alpha, span.length)
    L_a = asymptotic_length(span.alpha)
    eta = np.zeros((n, n))
    for i, cut in enumerate(channels):
        for j, pump in enumerate(channels):
            df = pump.f0 - cut.f0
            w = SPM_WEIGHT if i == j else XPM_WEIGHT
            p = interaction_factor(df, cut.baud_rate, pump.baud_rate, span.beta2, L_eff, L_a)
            eta_density = span.gamma**2 * w * p / (cut.baud_rate * pump.baud_rate**2)
            eta[i, j] = cut.baud_rate * eta_density
    return eta

def nli_power(channels, span, n_spans):
    eta = coefficient_matrix(channels, span) * n_spans
    P = np.array([c.power for c in channels])
    Pcut = np.outer(P, np.ones(len(P)))
    Ppump = np.outer(np.ones(len(P)), P)
    return np.sum(Pcut * Ppump**2 * eta, axis=1)

def channel_snr(channels, span, n_spans, p_ase):
    P = np.array([c.power for c in channels])
    return P / (p_ase + nli_power(channels, span, n_spans))

def optimum_launch_power(eta_self, p_ase):
    return (p_ase / (2.0 * eta_self))**(1.0 / 3.0)
```
