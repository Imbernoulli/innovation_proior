I start with the wrong picture on purpose: a sphere in `R^{n+1}`, a continuous map into `R^n`, and the wish to find `x` with `g(x)=g(-x)`. If I stare at it in local coordinates, it looks like I have `n` equations in something like `n` degrees of freedom, so maybe this is just a disguised intermediate value theorem. But that is too local. A coordinate chart sees one patch of the sphere and forgets that the point I am comparing with `x` is not nearby at all; it is the antipodal point on the other side. The condition is not local equality of nearby values. It is a global symmetry condition.

The one-dimensional case tells me what survives. If `g : S^1 -> R`, then `h(x)=g(x)-g(-x)` satisfies `h(-x)=-h(x)`. If `h` were never zero, its sign would be a continuous map from the connected circle to `{+1,-1}`, hence constant, but the antipodal rule forces the sign to flip. So some zero exists. That is not merely a sign trick; it says that the negation of the theorem produces an odd object. In one dimension the odd object has nowhere to live without crossing zero.

Now in higher dimension the same difference is vector-valued: `h(x)=g(x)-g(-x) in R^n`, still with `h(-x)=-h(x)`. If no antipodal pair has the same image, then `h(x)` never vanishes. I can divide by its length and get `F(x)=h(x)/|h(x)|`, a continuous map `S^n -> S^{n-1}` with `F(-x)=-F(x)`. So the theorem is really saying: there is no continuous odd map from `S^n` down to `S^{n-1}`.

That already feels like the right shape of the obstruction. A usual collapse from `S^n` to `S^{n-1}` is easy if I ignore antipodes; projection followed by normalization works away from two poles and can be patched in many non-symmetric ways. What is forbidden is not dimension-lowering by itself, but dimension-lowering while preserving the antipodal involution. Every pair `{x,-x}` is being asked to remain a pair of opposite outputs. The collapse must respect a global two-sheeted symmetry all the way around the sphere.

I need an invariant that detects that symmetry. Ordinary degree applies to maps `S^m -> S^m`, not to `S^n -> S^{n-1}`, so the normalized map has the wrong dimensions for degree. If I try to invent a coordinate proof, I lose the topology again. The better move is to look at an equator. Inside `S^n` there is an `S^{n-1}` bounding a hemisphere. Restrict `F` to that equator. Now I have an odd self-map `u : S^{n-1} -> S^{n-1}`. Degree can see this map.

What should the degree of an odd self-map be? I test the familiar examples. The identity is odd and has degree `1`. The antipodal map is odd and has degree `(-1)^n` on `S^{n-1}`, which is still odd as an integer. More generally, oddness means the map descends through the quotient by antipodes, `S^m -> RP^m`. With `Z/2` coefficients, the double cover `S^m -> RP^m` has a transfer exact sequence, and an odd self-map gives a natural map of that exact sequence to itself. Starting in dimension zero, where the induced map is forced to be an isomorphism, exactness carries that nonzero action step by step up through projective space and finally back to the top homology of the sphere. On `H_m(S^m; Z/2)`, the induced map is multiplication by `deg(u)` mod `2`. Since the induced map is nonzero, `deg(u)` is `1` mod `2`. So every odd self-map of a sphere has odd degree.

This is exactly the invariant I need: not the precise integer degree, only its parity. Antipodal symmetry forces the top-dimensional count to be odd. It cannot be washed away by a homotopy that respects the antipodal quotient.

But the same equatorial map `u` comes from restricting `F : S^n -> S^{n-1}` to the boundary of a hemisphere. Let `D^n` be one closed hemisphere bounded by that equator. Then `F|_{D^n}` is an extension of `u` across the disk. A map from `S^{n-1}` to itself that extends over `D^n` cannot carry nonzero top degree: on homology, its induced map factors as `H_{n-1}(S^{n-1}) -> H_{n-1}(D^n) -> H_{n-1}(S^{n-1})`, and the middle group is zero. So the degree of `u` is `0`.

Now the contradiction is sharp. The equatorial restriction is odd, so its degree is odd. The same restriction bounds through a hemisphere, so its degree is zero. The conflict is not visible in any one coordinate patch. It appears only when I keep the antipodal symmetry and ask what the map does globally to the equator's top homology.

This also explains why the proof wants the normalized difference, not the original map `g`. The original values live in `R^n`, and equal antipodal values are what I am trying to force. Subtracting `g(-x)` from `g(x)` converts equality into a zero, and the absence of zero converts the problem into a direction field. The direction field is automatically odd. Normalizing strips away every irrelevant magnitude and leaves only the topological data: a symmetry-respecting direction assigned to each point of the sphere.

The local-coordinate instinct fails because it asks where `n` scalar functions vanish at once. The invariant approach asks what happens if they do not vanish at once. If the vector never vanishes, it points everywhere; if it points everywhere, it defines an odd lower-dimensional sphere map; if that map exists, its equator restriction must both have odd degree and bound a disk. The obstruction is global before it is algebraic.

The low-dimensional cases now fit the same picture instead of sitting apart. For `n=1`, the target of the normalized odd map would be `S^0`, two points. A connected circle cannot map continuously to two points while flipping antipodes. For `n=2`, a hypothetical map `S^2 -> R^2` separating antipodes would give an odd `S^2 -> S^1`; restricting to the equator gives an odd loop in `S^1`, hence a loop of odd winding number, but because it extends over a hemisphere it is nullhomotopic and has winding number zero. In higher dimensions, winding number becomes degree, and the same contradiction remains.

So the proof lands in a compact form. Suppose `g : S^n -> R^n` is continuous and no antipodal pair has equal image. Then `h(x)=g(x)-g(-x)` is nonzero and odd, hence `F(x)=h(x)/|h(x)|` is an odd map `S^n -> S^{n-1}`. Choose an equatorial `S^{n-1}` in `S^n` and call the restriction `u`. Since `u` is odd, `deg(u)` is odd. Since `u` extends over the hemisphere by `F`, `deg(u)=0`. That is impossible. Therefore the assumption was false, and some antipodal pair satisfies `g(x)=g(-x)`.

The theorem is not a clever recombination of fixed-point folklore. Its distinctive step is choosing the invariant that can hear antipodal symmetry. The proof does not find the pair by moving coordinates around. It notices that if every pair were separated, the sphere would admit a continuous antipode-respecting collapse to one lower dimension, and degree mod `2` says such a collapse cannot exist.
