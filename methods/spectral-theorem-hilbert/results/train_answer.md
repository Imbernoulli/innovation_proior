The canonical result I present is the spectral theorem for bounded normal operators on a complex Hilbert space. In finite-dimensional linear algebra a normal matrix can be diagonalized: there is an orthonormal basis of eigenvectors and the operator is a weighted sum of orthogonal projections onto eigenspaces. The picture is clean because the spectrum is finite and each spectral value carries its own eigenspace. When I move to an infinite-dimensional Hilbert space this picture has to change. The operator may have no eigenvectors at all while still having a rich spectrum. The multiplication-by-coordinate operator on L^2([0,1]) is the classic warning sign: it is bounded and self-adjoint, yet Mf = lambda f forces f to vanish almost everywhere, so there are no nonzero eigenvectors. The theorem rescues diagonalization by replacing eigenvectors with spectral subspaces and sums with integrals.

The key conceptual move is to focus on the projection onto the part of the Hilbert space whose spectral values lie in a given Borel set. For a bounded normal operator N with spectrum K, the spectral theorem promises a unique projection-valued measure E on the Borel subsets of K such that N is recovered as the integral of the coordinate function against E. In symbols, N = int_K z dE(z). For a self-adjoint operator A this becomes A = int_{sigma(A)} lambda dE(lambda). This formula is the direct infinite-dimensional descendant of the finite-dimensional identity A = sum_j lambda_j P_j. The difference is that the labels are no longer just eigenvalues; they are points in a compact spectrum, and the weights are no longer individual rank-one projections but projection-valued measures over Borel sets.

I construct the spectral resolution in four stages. First, because N is normal, the C*-algebra generated by N and N* is commutative. The continuous functional calculus gives a unique unital star-homomorphism from continuous functions on the spectrum into bounded operators, sending the coordinate function to N. This is the safe algebraic starting point. Second, I turn vector states into scalar measures. For each vector x, the map f -> <f(N)x,x> is a positive linear functional on continuous functions, so by the Riesz representation theorem it is integration against a finite positive Borel measure mu_x. Polarization gives complex measures mu_{x,y} for pairs of vectors. Third, I extend the calculus to bounded Borel functions by defining the operator g(N) through its matrix coefficients <g(N)x,y> = int g dmu_{x,y}. The bound |<g(N)x,y>| <= ||g||_infty ||x|| ||y|| guarantees that this defines a genuine bounded operator, and a monotone-class argument preserves multiplication and adjunction. Fourth, I define E(B) = 1_B(N). Because indicator functions are real and idempotent, each E(B) is an orthogonal projection, and the multiplicativity of the calculus gives E(B)E(C) = E(B intersect C).

Countable additivity follows from the scalar measures. If B_1, B_2, ... are disjoint with union B, then for any vector x the squared norm of the tail projection E(B) - E(union_{n<=m} B_n) applied to x equals mu_x(B \\ union_{n<=m} B_n), which tends to zero by ordinary countable additivity. Thus E(B)x = sum_n E(B_n)x in norm. The projection-valued measure is therefore strongly countably additive, which is exactly the right infinite-dimensional analogue of adding orthogonal eigenspace projections.

The theorem also gives a bounded Borel functional calculus: for every bounded Borel function f on the spectrum, f(N) = int f dE is a bounded operator with norm at most the supremum norm of f, and the map f -> f(N) is a unital star-homomorphism. In particular N* corresponds to conjugation of the coordinate function. Uniqueness comes from the same scalar measures. If another projection-valued measure F produced the same operator, then it would produce the same continuous functional calculus, hence the same integrals against all continuous functions for every scalar measure <F(.)x,y>. Riesz uniqueness forces those scalar measures to agree on all Borel sets, and since the matrix coefficients determine the projections, E and F coincide.

When the spectrum is finite, the spectral theorem collapses to ordinary diagonalization. Each singleton {lambda} yields the projection P_lambda = E({lambda}), and the integral becomes the finite sum sum_lambda lambda P_lambda. When the spectrum is continuous, the integral absorbs the multiplication-by-coordinate example: E(B) is multiplication by the indicator of B, and integrating the coordinate function recovers the multiplication operator. The same theorem therefore spans finite matrices, compact operators with countable eigenexpansions, and operators with purely continuous spectrum.

I verify the finite-dimensional case numerically with a small Python illustration. The script constructs a random Hermitian matrix, computes its eigenvalues and eigenprojectors, checks that the matrix equals the sum of eigenvalue-weighted projectors, and then approximates a spectral projection onto an interval by summing the projectors for eigenvalues inside that interval.

```python
import numpy as np

np.random.seed(0)
n = 5
A = np.random.randn(n, n) + 1j * np.random.randn(n, n)
A = (A + A.conj().T) / 2  # make A Hermitian

eigvals, eigvecs = np.linalg.eigh(A)
P = [np.outer(v, v.conj()) for v in eigvecs.T]  # orthogonal spectral projections

reconstruction = sum(lam * p for lam, p in zip(eigvals, P))
print("Reconstruction error:", np.max(np.abs(A - reconstruction)))

a, b = -1.0, 1.0
mask = (eigvals >= a) & (eigvals <= b)
interval_projection = sum(P[i] for i in range(n) if mask[i])
print("Is interval projection idempotent/hermitian?",
      np.allclose(interval_projection @ interval_projection, interval_projection),
      np.allclose(interval_projection, interval_projection.conj().T))

# verify spectral measure property E(B)E(C)=E(B cap C) on two intervals
B = (eigvals >= a) & (eigvals <= b)
C = (eigvals >= 0.0) & (eigvals <= 2.0)
PB = sum(P[i] for i in range(n) if B[i])
PC = sum(P[i] for i in range(n) if C[i])
P_inter = sum(P[i] for i in range(n) if B[i] and C[i])
print("Projection intersection property:", np.allclose(PB @ PC, P_inter))
```

The output confirms that the finite atomic picture is exactly the spectral theorem in action. In the infinite-dimensional setting the same logic persists, but the atomic sums become integrals and the eigenprojectors become the projection-valued measure. The theorem is usually called the spectral theorem for bounded normal operators on Hilbert space, and it is the foundation on which functional calculus, quantum-mechanical observables, and spectral analysis of differential operators are built.
