# Spectral theorem for bounded normal operators

## Theorem

Let `H` be a complex Hilbert space and let `N in B(H)` be normal. Put `K=sigma(N)`, a compact subset of `C`. There is a unique projection-valued measure

`E: Borel(K) -> B(H)`

such that

`N = int_K z dE(z)`.

More explicitly:

1. `E(emptyset)=0`, `E(K)=I`, and each `E(B)` is an orthogonal projection.
2. `E(B cap C)=E(B)E(C)` for Borel sets `B,C subset K`.
3. If `B_1,B_2,...` are disjoint Borel sets, then

   `E(union_n B_n)x = sum_n E(B_n)x`

   for every `x in H`, with convergence in norm.
4. For every bounded Borel function `f:K -> C`,

   `f(N) := int_K f(z) dE(z)`

   is a bounded operator, `||f(N)|| <= ||f||_infty`, and `f -> f(N)` is a unital `*`-homomorphism.

In particular, `N* = int_K conjugate(z) dE(z)`.

If `A=A*`, then `sigma(A) subset R` and the formula becomes

`A = int_{sigma(A)} lambda dE(lambda)`.

If the spectrum is finite and `P_lambda=E({lambda})`, this reduces to the matrix formula

`A = sum_lambda lambda P_lambda`.

## Proof

Since `N` is normal, the unital C*-algebra generated by `N` is commutative. The continuous functional calculus gives a unique unital `*`-homomorphism

`Phi:C(K)->B(H)`

such that `Phi(z)=N`, where `z` is the coordinate function on `K`.

For `x,y in H`, define a bounded linear functional on `C(K)` by

`L_{x,y}(f)=<Phi(f)x,y>`.

For `x=y`, this functional is positive. By the Riesz representation theorem, there is a finite positive Borel measure `mu_x` on `K` such that

`<Phi(f)x,x> = int_K f dmu_x`

for all `f in C(K)`. Polarization gives finite complex Borel measures `mu_{x,y}` satisfying

`<Phi(f)x,y> = int_K f dmu_{x,y}`.

For a bounded Borel function `g`, the form

`s_g(x,y)=int_K g dmu_{x,y}`

is bounded, with `|s_g(x,y)| <= ||g||_infty ||x|| ||y||`. Hence there is a unique bounded operator, denoted `Psi(g)`, such that

`<Psi(g)x,y>=int_K g dmu_{x,y}`.

This extends `Phi`. A monotone-class argument, starting from continuous functions and using bounded pointwise convergence, shows that `Psi` is a unital `*`-homomorphism from bounded Borel functions on `K` into `B(H)`.

Define

`E(B)=Psi(1_B)`.

Then `E(B)^*=E(B)` and `E(B)^2=E(B)`, so `E(B)` is an orthogonal projection. Also

`E(B)E(C)=Psi(1_B)Psi(1_C)=Psi(1_{B cap C})=E(B cap C)`.

For disjoint `B_n` and `B=union_n B_n`, let `S_m=union_{n<=m}B_n`. For every `x in H`,

`||(E(B)-E(S_m))x||^2 = <E(B \ S_m)x,x> = mu_x(B \ S_m)`,

which tends to zero by countable additivity of `mu_x`. Thus `E` is strongly countably additive.

Since `Psi` extends the continuous functional calculus and the coordinate function is continuous on `K`,

`int_K z dE(z)=Psi(z)=Phi(z)=N`.

For self-adjoint `A`, the spectrum is real, giving the real integral formula. In the finite atomic case, `1_B(A)` is the sum of the eigenspace projections whose eigenvalues lie in `B`, so the operator integral is exactly the usual orthogonal diagonalization.

Finally, suppose `F` is another projection-valued measure with `N=int_K z dF(z)`. Since `F` is projection-valued, it also gives `N*=int_K conjugate(z) dF(z)`. Therefore polynomial integrals in `z` and `conjugate(z)` agree with the same polynomials in `N` and `N*`; by uniform approximation they agree with the continuous functional calculus on `C(K)`. For every `x,y`, the scalar measures `<E(.)x,y>` and `<F(.)x,y>` therefore have the same integrals against every continuous function on `K`. Riesz uniqueness gives equality of these scalar measures on all Borel sets. Hence `<E(B)x,y>=<F(B)x,y>` for all `x,y` and all Borel `B`, so `E(B)=F(B)`. The spectral resolution is unique.
