I want diagonalization to survive, but the word "eigenvector" is already too narrow. In a finite-dimensional Hilbert space a normal matrix gives me an orthonormal eigenbasis, and if I stop looking at the individual vectors and look at the eigenspaces instead, the formula is really `N=sum_j lambda_j P_j`. The projections `P_j` are the stable part of the story. The basis inside each eigenspace is irrelevant. Once I notice that, the calculus is automatic: `f(N)=sum_j f(lambda_j)P_j`. The operator is not merely decomposed into vectors; it is decomposed by orthogonal pieces of the space, one piece for each spectral value.

Now I test the same idea where an eigenbasis cannot possibly be the right object. On `L^2([0,1])`, multiplication by the variable, `(Mf)(t)=t f(t)`, is bounded and self-adjoint. If `Mf=lambda f`, then `(t-lambda)f(t)=0` almost everywhere, so `f` must vanish away from the one point `lambda`; in `L^2` that means `f=0`. There are no eigenvectors. But the operator is not mysterious. If I ask for the part of the space with spectral values in an interval or a Borel set `B`, I know exactly what it is: functions supported on `B`. The projection is multiplication by `1_B`. This is the same diagonal idea, except the diagonal is no longer a list. It is a variable over a measure space.

So the finite formula should not become an infinite list of eigenvectors. It should become a rule `B -> E(B)`, where `E(B)` is an orthogonal projection and disjoint spectral sets give orthogonal subspaces. The old finite formula is the atomic case: `E(B)=sum_{lambda_j in B} P_j`. For multiplication by `t`, `E(B)` is multiplication by `1_B`. In both cases the operator is recovered by integrating the coordinate function:

`N = int z dE(z)`.

This is the right replacement for diagonalization. The spectrum is not a set of labels for basis vectors. It is the measurable space over which the Hilbert space is resolved.

I still need a way to construct `E` from an abstract operator. Polynomials are the first safe objects. If `N` is normal, then `N*N=NN*`, so the algebra generated by `N` and `N*` is commutative. That matters because functions of one spectral variable commute pointwise; a non-normal operator would still remember an order of multiplication, and a scalar functional calculus would be the wrong language. For a normal operator, a polynomial `p(z, conjugate z)` should become `p(N,N*)`. Completing this idea gives a continuous functional calculus on `sigma(N)`: every continuous `f` on the compact spectrum has an operator `f(N)`, and the map is a unital `*`-homomorphism with `z(N)=N`.

But continuous functions are not enough. They let me make smooth or uniform-limit functions of `N`, but they do not give the exact projection onto a spectral set. The function I need for a set `B` is `1_B`, and that is usually discontinuous. If I stop at continuous calculus, I can approximate an interval from inside and outside, but I have not produced the projection itself. The missing move is measure theory.

For each vector `x`, the expression `<f(N)x,x>` is a positive linear functional on `C(sigma(N))`: if `f>=0`, then `f(N)` is positive. By the Riesz representation theorem, that positive functional is integration against a finite positive Borel measure `mu_x`. For two vectors, polarization gives a complex measure `mu_{x,y}` satisfying

`<f(N)x,y> = int f dmu_{x,y}`

for continuous `f`. This is already the scalar shadow of the spectral resolution. Each vector sees the operator as an ordinary measure on the spectrum.

Now I can try to define a bounded Borel function of `N` by the same formula:

`<g(N)x,y> = int g dmu_{x,y}`.

I have to check this is really an operator, not just a formal sesquilinear expression. If `g` is bounded, the integral is controlled by `||g||_infty ||x|| ||y||`, using the positivity of the diagonal measures and Cauchy-Schwarz for these forms. So for each bounded Borel `g` there is a unique bounded operator with those matrix coefficients. The construction agrees with the continuous calculus because it was built from the same scalar measures, and it remains multiplicative because the continuous calculus was multiplicative and bounded pointwise limits preserve the identities through a monotone-class argument.

Now the projection appears without being separately invented. For a Borel set `B`, define

`E(B)=1_B(N)`.

Since `1_B` is real-valued and idempotent, `E(B)^*=E(B)` and `E(B)^2=E(B)`. Thus `E(B)` is an orthogonal projection. Since `1_B 1_C=1_{B cap C}`, I get `E(B)E(C)=E(B cap C)`. Disjoint sets automatically give orthogonal projections. The Hilbert space geometry is doing exactly what the finite eigenspaces did, but now the labels are measurable sets.

Countable additivity is also a Hilbert-space statement. If `B_1,B_2,...` are disjoint and `B=union_n B_n`, then for a vector `x` the squared norm of the tail is measured by `mu_x`. More concretely, the projection onto `union_{n>m} B_n` has norm on `x` squared equal to `mu_x(union_{n>m} B_n)`, and that tends to zero by countable additivity of the scalar measure. Therefore

`E(B)x = sum_n E(B_n)x`

with convergence in norm. The measure is projection-valued, and its additivity is strong enough to act on vectors, not merely on scalar matrix coefficients.

At this point the old diagonal formula has become an integral. The coordinate function `z` belongs to the bounded Borel functions on `sigma(N)` because `N` is bounded, and the continuous calculus already says `z(N)=N`. Therefore

`N = int_{sigma(N)} z dE(z)`.

For a self-adjoint operator `A`, the spectrum lies on the real line, so this reads

`A = int_{sigma(A)} lambda dE(lambda)`.

The finite-dimensional formula is recovered when the measure has only atoms: if the spectral values are `lambda_j`, then `E(B)=sum_{lambda_j in B}P_j`, and the integral of `z` is exactly `sum_j lambda_j P_j`. The multiplication example is recovered when the projections are multiplication by indicators. The same sentence covers both cases because the real object is neither a basis nor a list; it is the projection-valued measure.

Uniqueness has to come from the same scalar measures. If another projection-valued measure `F` gives the same operator and the same functional calculus for the coordinate function, then every continuous function of the coordinate has the same operator under `E` and `F`. Taking matrix coefficients gives the same integrals of every continuous function against the scalar measures `<E(.)x,y>` and `<F(.)x,y>`. Riesz uniqueness forces those scalar measures to agree on all Borel sets. Since all matrix coefficients agree, the projections agree. There is no room for a second spectral resolution.

The conceptual shift is now clean. I do not rescue diagonalization by searching harder for eigenvectors. Continuous spectrum says that search can be empty. I rescue diagonalization by changing the unit of decomposition from a vector at a point to a projection over a measurable set. The Hilbert space supplies orthogonality, the commutative algebra of a normal operator supplies the functional calculus, and measure theory supplies the passage from continuous functions to sharp spectral cuts. Linear algebra becomes integration.
