We want to know which sparse Boolean functions plain SGD on a plain two-layer network can actually learn. The data are $x \sim \mathrm{Unif}(\{+1,-1\}^d)$ with $d$ arbitrarily large, but the label depends on only $P$ hidden coordinates: $f^*(x) = h^*(z)$ with $z = x_I$ for an unknown latent subset $I \subseteq [d]$, $|I| = P$. The sharpening case is a pair of $3$-sparse targets that look almost identical, $h^*(z) = z_1 z_2 z_3$ and $h^*(z) = z_1 + z_1 z_2 + z_1 z_2 z_3$: both depend on three coordinates, both carry a degree-$3$ term, and so both are hopeless for any fixed-feature method, which needs $\Omega(d^3)$ samples to express a degree-$3$ component over an unknown subset. Yet watching networks train suggests SGD treats them completely differently — the staircase gets learned, the bare monomial just sits there. If that is real, some structural property of $h^*$ decides learnability, and the goal is to name that property exactly, with both a sufficiency and a matching necessity direction, at a sample budget of $O(d)$ uniform over the choice of $I$.

The existing options each fail in a specific way. Lazy/NTK training freezes the first layer at a random init and fits only the readout; the feature map is then chosen before seeing which coordinates matter, so it spends its budget uniformly over all $d$ inputs and pays $\Omega(d^k)$ for a degree-$k$ component — it cannot adapt to $I$, and learns the staircase and the bare monomial at the same prohibitive cost. The prior layerwise coordinate-descent construction proves staircases are learnable in principle, but it needs depth growing with the degree and a non-standard optimizer, gives only a "polynomial in $d$" rate with no finer separation, covers only single nested chains, and supplies sufficiency with no necessity. Mean-field analyses keyed to rotational symmetry reduce the dynamics for isotropic Gaussian data, but the reduction is driven by symmetry of the *inputs*, not sparsity of the *target*, so $d$ never really shrinks and no structure-tied sample-complexity statement comes out. What we need is something that stays nonlinear so the features can move toward $I$, that collapses the problem along the sparsity, and that pins down necessity as sharply as sufficiency.

I propose Mean-Field two-layer SGD for sparse Boolean functions, together with the characterization it yields: the Merged-Staircase Property. The first design choice is the parametrization, and it carries more weight than it looks. Writing the output with the $1/\sqrt{N}$ normalization makes the weights barely move as the width grows — the network linearizes around its init and becomes a fixed kernel machine, frozen before it sees $I$. That is exactly the regime to escape, so I take the mean-field $1/N$ scaling
$$\hat f(x;\Theta) = \frac{1}{N}\sum_{j=1}^N a_j\,\sigma(\langle w_j, x\rangle), \qquad \Theta = (a_j, w_j)_{j\in[N]}.$$
With $1/N$ out front each neuron contributes $O(1/N)$ to the output but the weights are free to travel an $O(1)$ distance, so the dynamics stays genuinely nonlinear and the features can adapt to the hidden subset. Train by square loss with one-pass batch-SGD, anisotropic step $H_k = \mathrm{diag}(\eta^a_k, \eta^w_k I_d)$ and regularization $\Lambda$, drawing fresh samples every step.

The $N$ neurons are exchangeable — the output sees them only through their empirical distribution $\hat\rho^{(N)} = \frac1N \sum_j \delta_{\theta_j}$ — so I track that measure rather than $N(d+1)$ coordinates. The population risk is a functional of $\rho$ alone, $R(\rho) = R_\# - 2\int E[y\,\sigma]\,a\,d\rho + \iint E[\sigma\sigma']\,aa'\,d\rho\,d\rho$, and in the wide-and-small-step limit one-pass SGD becomes a Wasserstein gradient flow on $R$: the empirical measure converges to $\rho_t$ solving a continuity equation $\partial_t\rho_t = \nabla_\theta\cdot(\rho_t H(t)\nabla_\theta\psi(\theta;\rho_t))$, the neurons a gas of particles descending the risk while conserving mass. This is clean but still lives in $\mathbb{R}^{d+1}$, and sparsity has not yet been used.

The reduction is where sparsity enters. Split every input $x = (z, r)$ and weight $w = (u, v)$, with $u \in \mathbb{R}^P$ aligned to the signal $z$ and $v \in \mathbb{R}^{d-P}$ aligned to noise $r$. At init the coordinates of $w^0$ are iid and symmetric, so flipping any sign of the noise block leaves the law unchanged, and averaging over the symmetric $v^0$ makes $\hat f(x;\rho_0)$ independent of $r$. This survives training: the pushforward $\rho_t^{\#r}$ under a fixed sign flip of $v$ solves the same PDE with the same initial condition, because the flow couples weights only through $\langle w, x\rangle$ and the flip can be absorbed into the $x$-average; by uniqueness $\rho_t^{\#r} = \rho_t$ for all $t$, so the network stays independent of the irrelevant directions throughout. The remaining dependence on noise is through $\langle v^t, r\rangle$, a sum of $d-P$ independent bounded terms, hence approximately Gaussian for large $d$: $\langle v^t, r\rangle \approx \|v^t\|_2\, G$, $G \sim N(0,1)$, with Berry–Esseen controlling the error by $\max_i |v_i^t|/\|v^t\|_2$, which is tiny at init and stays small up to times of order one. So the entire $(d-P)$-dimensional noise weight collapses to a single scalar $s^t = \|v^t\|_2$ acting as a Gaussian smoothing width. Introduce effective parameters $\bar\theta = (\bar a, \bar u, \bar s) \in \mathbb{R}^{P+2}$ and the effective network
$$\hat f(z;\bar\rho_t) = \int \bar a\,E_G\!\big[\sigma(\langle \bar u, z\rangle + \bar s\,G)\big]\,\bar\rho_t(d\bar\theta),$$
a two-layer network in the fixed dimension $P$, free of $d$. Its closed dynamics is the dimension-free Wasserstein gradient flow on $R(\bar\rho) = E_z[(h^*(z) - \hat f(z;\bar\rho))^2]$ with particle drift
$$\dot{\bar a} = \xi^a E_{z,G}[(f^*-\hat f)\sigma] - \xi^a\lambda^a\bar a,\quad \dot{\bar u} = \xi^w \bar a\, E_{z,G}[(f^*-\hat f)\sigma' z] - \xi^w\lambda^w\bar u,\quad \dot{\bar s} = \xi^w\bar a\,E_{z,G}[(f^*-\hat f)\sigma' G] - \xi^w\lambda^w\bar s.$$
The init is the load-bearing part: drawing first-layer coordinates at scale $1/\sqrt d$ — equivalently $\sqrt d\, w^0_{j,l}$ having a fixed law — gives $\bar u^0 = O(1/\sqrt d) \to 0$, $\bar s^0 = m^w_2$, $\bar a^0 \sim \mu_a$. The flow *starts with the signal weights exactly at the origin*, and learnability is precisely whether the flow can push $\bar u$ off it. That this dimension-free flow is a faithful proxy is quantitative, via the standard mean-field PDE as a bridge: $\|\hat f(\cdot;\Theta^k) - \hat f(\cdot;\bar\rho_{k\eta})\| \le e^{KT^7}\{\sqrt{(P+\log d)/d} + \sqrt{\log N/N} + \sqrt{(d+\log N)/b}\,\sqrt\eta\}$ for $k \le T/\eta$, $T = \eta n/b$ bounded, so taking $d, N, 1/\eta$ large makes the flow govern SGD.

Necessity is where the example separation lives. From the $\bar u^0 = 0$ start, a coordinate $i$ leaves the origin only if it gets a nonzero push $\dot{\bar u}_i = \bar a\, E_z[(h^* - \hat f)\sigma'(\langle\bar u,z\rangle)z_i] - \text{reg}$. Take $h^* = z_1 + z_1 z_2 z_3$ and, for intuition, set $\hat f = 0$, $\bar s = 0$: coordinate $1$ correlates with the $z_1$ term and moves, but coordinates $2$ and $3$ enter only through $z_1 z_2 z_3$, and their equations are symmetric under swapping $2\leftrightarrow 3$ from an identical zero start, so $\bar u_2^t = \bar u_3^t =: u_{23}^t$. Integrating out $z_{23} = z_2 + z_3$ gives $\dot u_{23} = \bar a\, u_{23}\, E_{z_1}[z_1\sigma''(z_1 u_1 + r)]$, a *homogeneous linear* ODE in $u_{23}$ from $u_{23}^0 = 0$, so $u_{23}^t \equiv 0$: the degree-$3$ piece is permanently missed and the risk is bounded below by $\hat h(\{1,2,3\})^2 > 0$. The general mechanism, made rigorous without the $\hat f = 0$ cheat, is this: order the supports greedily, let $\bar S_*$ be the largest subset addable one-new-coordinate-at-a-time, and let $\Omega$ be the leftover new coordinates that never appear "alone." For $i \in \Omega$ I bound three correlations — the network's own output contributes $\le K(1+t)|\bar u_i^t|$ (the readout grows at most linearly, $|\bar a^t| \le K(1+t)$); an already-ordered support $S$ with $i \notin S$ contributes $\le \|\sigma''\|_\infty|\bar u_i^t|$; and crucially a leftover support has *another* fresh coordinate $j \in \Omega \cap S$, contributing $\le \|\sigma''\|_\infty|\bar u_j^t|$, so the push on $i$ is gated by some other zero-coordinate. With $m_\Omega^t = \max_{i\in\Omega}|\bar u_i^t|$ this gives $|\dot{\bar u}_i^t| \le K(1+t)^2 m_\Omega^t$, and since every $\Omega$-coordinate starts at zero, Gronwall forces $m_\Omega^t \equiv 0$. Those coordinates are frozen, every monomial touching them is unlearnable, and $R(\bar\rho_t) \ge \sum_{S\ \text{unlearnable}}\hat h(S)^2 > 0$ regardless of activation, schedule, regularization, or noise width. So the supports $\{S_1,\dots,S_m\}$ must admit an ordering with $|S_i \setminus \bigcup_{j<i} S_j| \le 1$ — the Merged-Staircase Property — and $z_1 z_2 z_3$, leaping straight to degree $3$, is the cleanest violator.

Sufficiency runs the same mechanism in reverse, layerwise: Phase 1 trains only $\bar u$ (set $\xi^a = 0$), Phase 2 freezes $\bar u$ and trains only $\bar a$ (set $\xi^w = 0$). Taking $\mu_w = \delta_0$ gives $\bar s^t \equiv 0$, killing the smoothing term. In Phase 1 the staircase $h^* = \sum_k \alpha_{1..k} z_1\cdots z_k$ induces a cascade: $\bar u_k$ has no first-order driver until $\bar u_1\cdots\bar u_{k-1}$ is nonzero, captured to leading order by
$$\frac{d}{dt}\,\hat u_k^t = \bar a\,\alpha_{1..k}\,m_k\!\prod_{j<k}\hat u_j^t, \qquad \hat u_k^0 = 0,\qquad m_r = \sigma^{(r)}(0),$$
from the Taylor identity $E_z[\chi_S(z)\sigma^{(i)}(\langle u,z\rangle)] = m_{|S|+i}\prod_{k\in S} u_k\,(1 + O(t)) + O(t^L)$, where only the support-matching monomial survives orthogonality. Solving from the bottom: $\hat u_1 \sim t$, $\hat u_2 \sim t^2$, $\hat u_3 \sim t^4$ — exponents $1, 2, 4, \dots$ doubling each step, so $|u_k^t| = \Theta(t^{2^{k-1}})$. The weights light up sequentially, lower degree first, each new stair an order slower than the one below: the network climbs the staircase, and a Gronwall bound $\dot\Delta_k \le K(t^{2^{k-1}} + \Delta_k)$ confirms the true trajectory tracks the cascade with strictly higher-order error. Phase 2 is then convex — the output is linear in $\bar a$, a kernel regression with $K^{T_1}(z,z') = E_{a\sim\mu_a}[\sigma(\langle\bar u^{T_1}(a),z\rangle)\sigma(\langle\bar u^{T_1}(a),z'\rangle)]$, and $\frac{d}{dt}\|g_t\|_2^2 = -g_t^\top K^{T_1} g_t \le -\lambda_{\min}(K^{T_1})\|g_t\|_2^2$, so the risk decays as $e^{-\lambda_{\min}(K^{T_1})t}$ provided the kernel is positive definite. In the Fourier basis $K^{T_1} = D(M + \Delta)D$ with $M = (E_a[a^{\beta(S)+\beta(S')}])$, $\beta(S) = \sum_{k\in S} 2^{k-1}$. Since $\beta$ ranges over all of $\{0,\dots,2^P-1\}$ as $S$ ranges over subsets, $M$ is exactly the Gram matrix of $1, X, X^2,\dots,X^{2^P-1}$ in $L^2([-1,1])$, which are linearly independent, so $\lambda_{\min}(M) > 0$; taking $T_1$ small enough that $\Delta$ cannot kill it gives $\lambda_{\min}(K^{T_1}) > 0$ and exponential convergence. Two ingredients are doing the work and dictate the recipe. The randomness of the readout $a \sim \mu_a$ is what makes the per-neuron features $a^{\beta(S)}$ linearly independent and $M$ full rank — fix all $a$ equal and the neurons collapse and the kernel degenerates, so neuron diversity is essential. And the cascade $\dot{\hat u}_k = \bar a\,\alpha_{1..k}\,m_k\prod\hat u_j$ dies the instant any $m_k = \sigma^{(k)}(0) = 0$, so a symmetric activation (odd like $\tanh$, killing the even derivatives) breaks the chain; I need $\sigma^{(r)}(0) \neq 0$ for $r = 0,\dots,P$, which the shifted sigmoid $\sigma(x) = (1 + e^{-x+0.5})^{-1} = \mathrm{sigmoid}(x - 0.5)$ provides, the shift moving evaluation off the logistic's symmetric center. Vanilla staircases are learnable with no genericity; general MSP supports are learnable almost surely over the Fourier coefficients (exact symmetries like $z_1 + z_2 + z_1 z_3 + z_2 z_4$ can collapse weights into a zero-risk-free subspace, which random coefficients break with probability one). Finally the separation is genuine: any linear method needs $\min(n,q) = \Omega(d^k)$ for a degree-$k$ component, and a subspace-projection bound over all coordinate-permutations of a staircase gives $\min(n,q) \ge (\eta/2)\binom{d}{\lfloor\eta P/2\rfloor} = d^{\omega(1)}$ for slowly-growing $P$ — superpolynomially more than the $O(d)$ this network uses.

The concrete artifact is a two-layer network in the $1/M$ mean-field scaling, no bias, scaled Gaussian first layer with $\sqrt d\, w_{j,l} \sim N(0,1)$ so the signal coordinates start near the origin saddle, readout weights from $\mathrm{Unif}([-1,1])$ for diversity, shifted-sigmoid activation, trained by plain SGD with $\eta = 1/2$ on the square loss with fresh batches:

```python
import torch
import torch.nn as nn


def build_model(config) -> nn.Module:
    class TwoLayerMeanField(nn.Module):
        def __init__(self, d: int, M: int) -> None:
            super().__init__()
            self.fc1 = nn.Linear(d, M, bias=False)   # no bias
            self.fc2 = nn.Linear(M, 1, bias=False)
            nn.init.normal_(self.fc1.weight, mean=0.0, std=d ** -0.5)
            with torch.no_grad():
                self.fc2.weight.uniform_(-1.0, 1.0)               # a_j ~ Unif([-1,1])
            self.M = M

        @staticmethod
        def _shifted_sigmoid(u: torch.Tensor) -> torch.Tensor:
            return torch.sigmoid(u - 0.5)            # sigma(x) = (1 + e^{-x+0.5})^{-1}

        def forward(self, x: torch.Tensor) -> torch.Tensor:
            h = self._shifted_sigmoid(self.fc1(x))
            return self.fc2(h).view(-1) / self.M     # 1/M mean-field normalization

    return TwoLayerMeanField(d=config.d, M=config.width)


def get_optimizer(model: nn.Module, config) -> torch.optim.Optimizer:
    return torch.optim.SGD(model.parameters(), lr=0.5, momentum=0.0)   # plain SGD, eta = 1/2


def train_step(model, optimizer, x, y) -> float:
    optimizer.zero_grad(set_to_none=True)
    pred = model(x).view(-1)
    loss = ((pred - y) ** 2).mean()                  # square loss, one-pass fresh batch
    loss.backward()
    optimizer.step()
    return float(loss.item())
```
