# Synthesis — Algorithmic Lovász Local Lemma (Moser / Moser–Tardos)

## Sources actually read this run (all in refs/)
- **Primary 1**: Moser 2009 STOC, "A constructive proof of the Lovász Local Lemma", arXiv 0810.4812 (LaTeX source `src/moser-cplll08.tex`, PDF+txt in refs/). The single-author SAT-only version. Uses the recursive `locally_correct` + `solve_lll` with abort-after-`log m + 2`-invocations, the **assignment table / indirect-assignment** device, the **composite witness** counting, and the **witness-encoding injection** giving `≤ m·2^{u(k-1)}` witnesses of size `u`. Neighborhood bound `|Γ⁺(C)| ≤ 2^{k-5}`.
- **Primary 2**: Moser–Tardos 2010 JACM, "A constructive proof of the general Lovász Local Lemma", arXiv 0903.0544 (LaTeX `src/moser-tardos-lop4.tex`). The definitive general version: `sequential_lll` resample algorithm, **witness tree** `τ_C(t)`, **coupling/τ-check** Lemma (Pr[τ occurs] ≤ ∏ Pr[[v]]), **Galton–Watson** branching process Lemma (p_τ = (1-x(A))/x(A) · ∏ x'([v])), giving E[N_A] ≤ x(A)/(1-x(A)). Plus parallel version, derandomization, lopsided version.
- **Self-account (secondary records of the STOC talk)**: Fortnow/GASARCH computationalcomplexity.org "A Kolmogorov Complexity Proof of the Lovász Local Lemma" (refs/fortnow-kolmogorov-lll.html) — records that Moser "gave one of the best STOC talks ever" and "came up with [the proof] while preparing the talk", and that it "can be written as a Kolmogorov complexity argument". And Terence Tao "Moser's entropy compression argument" (refs/tao-entropy-compression.html) — the canonical crystallization of the information-theoretic framing Moser presented orally: the Fix/Solve algorithm, the incompressibility argument (the log of the execution plus the final state losslessly encodes the random bits; if the algorithm ran too long it would compress n+Mk random bits into n+O(|S|)+M(k-C+O(1)) bits, contradiction), and the "fix-one-break-another is fine because each break is cheap to log" point.
- **Antecedent / non-constructive LLL**: Yufei Zhao, *Probabilistic Methods in Combinatorics* ch.6 (refs/zhao-pm6-lll.pdf+txt) — the original Erdős–Lovász (1975) proof by induction on |S| with conditional probabilities, symmetric form `e·p·(d+1) ≤ 1`, general form `Pr[A] ≤ x(A)∏(1-x(B))`, SAT incarnation `|Γ(C)| ≤ 2^{k-2}`.
- **Antecedents (via web research, noted)**: Beck 1991 (first constructive, hypergraph 2-coloring, `2^{k/48}`); Alon 1991 (randomized, `2^{k/8}`); Srinivasan 2008 (`2^{k/4}`); Moser 2008 (`2^{k/2}`). All worked far below the existential `2^{k}/e` threshold by a "freeze a large independent core, brute-force the rest" two-phase strategy.

## The research question (pre-method)
LLL proves existence non-constructively: if bad events A each have Pr[A] ≤ p, each dependent on ≤ d others, and e·p·(d+1) ≤ 1, then Pr[no bad event] > 0. But that probability can be exponentially small (∏(1-x(A))), so naive sampling won't find the good point in poly time. GOAL: an efficient algorithm that actually *finds* an assignment avoiding all bad events, ideally under the *same* (or nearly the same) condition as the existential lemma.

## Key lineage limitations
- Erdős–Lovász induction proof: gives only positive (tiny) probability; the conditional-probability induction is non-algorithmic — no procedure falls out.
- Beck/Alon/Srinivasan/earlier-Moser: constructive but only up to `2^{k/c}` for c=48,8,4,2 — an exponential gap below the existential `2^{k}/e`. They froze a maximal independent "dangerous" sub-structure and brute-forced it; the freezing wasted most of the slack.

## The method (what the trace must land on)
1. **Algorithm** (dead simple): random assignment; while some clause/event A violated, resample vbl(A) afresh; repeat. (Moser–Tardos `sequential_lll`; Moser SAT `solve_lll` + recursive `locally_correct`.)
2. **Why "fix one break another" is not fatal**: when you resample vbl(A), every clause NOT sharing a variable with A is untouched; only neighbors Γ(A) can break, and the recursion fixes those locally. After a recursive call on D returns, the set of violated clauses strictly shrank (Moser Lemma: locally_correct(D) leaves Γ⁺(D) all satisfied without un-satisfying anything outside vbl(τ)).
3. **Termination via incompressibility (talk framing)**: log each resample by which neighbor index (k-O(1) bits) it was; the log + current state losslessly reconstruct all the random bits consumed (k bits per resample). A run of M resamples encodes Mk random bits in ≈ M(k-O(1)) + O(n) bits — impossible for M large. So M = O(n) expected. The slack `d ≤ 2^{k-O(1)}` is exactly "each Fix index costs < k bits so the log is shorter than the entropy".
4. **Combinatorial form (paper)**: composite witness W (forest of recursion trees, connected by shared variables); injective encoding into the infinite (2d)-ary tree I via high/low children + glueing-edge 2-coloring ⇒ ≤ m·(4ed)^u ≤ m·2^{u(k-1)} witnesses of size u (using Knuth's count (2ed)^u of u-vertex subtrees, ·2^u colorings ⇒ (4ed)^u, and 4ed = 4e·2^{k-5} < 2^{k-1}). A fixed size-u witness is consistent with a random table with prob 2^{-ku}. So E[#size-u consistent witnesses] ≤ m·2^{u(k-1)}·2^{-ku} = m·2^{-u}; summing u ≥ log m + 2 gives ≤ 1/2.
5. **Moser–Tardos tight bound (definitive)**: witness tree τ_C(t); coupling τ-check ⇒ Pr[τ occurs] ≤ ∏_{v} Pr[[v]] ≤ ∏ x'([v]); Galton–Watson process produces τ with prob p_τ = (1-x(A))/x(A) ∏ x'([v]); since ∑ p_τ ≤ 1, E[N_A] = ∑_τ ∏ x'([v]) = x(A)/(1-x(A)) ∑ p_τ ≤ x(A)/(1-x(A)). Total expected resamples ≤ ∑_A x(A)/(1-x(A)). This matches the existential LLL condition exactly — no constant lost.

## Design-decision → why
- **Resample only vbl(A), keep everything else**: minimal disturbance ⇒ only Γ(A) can break; the locality is what bounds the recursion fan-out by d.
- **Pick lexicographically-first / arbitrary violated event**: analysis is selection-independent (MT remark). Lex order in Moser SAT is only to make the recursion tree reconstructible from shape+labels (the "natural ordering"), which is needed so the witness can be encoded without storing the order.
- **Abort after log m + 2 invocations (Moser SAT)**: a too-long single correction = a large consistent composite witness, which is exponentially unlikely; abort+restart costs ≤ 2 expected restarts.
- **Indirect assignment / assignment table**: decouples "which sample" from "when sampled" so the coupling/consistency argument is clean — the t-th sample of a variable is a fixed table entry, reused identically by the τ-check.
- **(2d)-ary tree, high vs low children, glueing-edge coloring**: the encoding must be injective and store only one global root; high children encode the path from an interior vertex up to a tree root (rotations), low children encode genuine recursion, glueing edges (2-coloring) mark where one recursion tree connects to the next in the composite witness. 2d children because each node has ≤ d real neighbors but needs both a "low" and a "high" copy.
- **Witness tree attaches C(i) to the DEEPEST eligible vertex (MT)**: forces labels on each level to be an independent set in G ⇒ tree is proper ⇒ coupling argument and Galton–Watson factorization go through.
- **x'(B) := x(B)∏_{C∈Γ(B)}(1-x(C))**: exactly the per-vertex factor that makes the Galton–Watson tree-probability telescope into x(A)/(1-x(A)).
