# Context: first-order conditions and duality for inequality-constrained optimization

## Research question

I want a usable first-order theory of optimality — and, if possible, a duality theory — for the
finite-dimensional constrained problem

```
minimize    f(x)
subject to  g_i(x) ≤ 0,   i = 1,…,m
            h_j(x) = 0,   j = 1,…,p
```

with f, g_i, h_j differentiable on ℝⁿ. For *equality only* (the g_i absent) the answer is two
centuries old: at a local minimum the gradient of f is a linear combination of the constraint
gradients, ∇f + Σ ν_j ∇h_j = 0. The open question is what replaces this when **inequalities** are
present. Inequalities change the geometry in a way equalities do not: the feasible set is no
longer a smooth manifold but a region with a boundary; at a candidate optimum some inequality
constraints are *active* (g_i = 0, the point sits on that wall) while others are *slack*
(g_i < 0, the wall is far away and irrelevant); and the set of feasible directions is a cone, not
a subspace.

## Background

**Lagrange multipliers for equality constraints (Lagrange, *Méchanique analitique*, 1788).**
For min f(x) subject to h_1(x)=⋯=h_m(x)=0, the multiplier rule states that at a local solution x̄,
under a regularity assumption, there exist λ_1,…,λ_m with

```
∇f(x̄) + Σ_j λ_j ∇h_j(x̄) = 0.
```

The standard regularity assumption (LICQ) is that ∇h_1(x̄),…,∇h_m(x̄) are linearly independent.
The geometric content is that −∇f lies in the span of the constraint gradients: there is no
feasible first-order direction (one keeping all h_j = 0) along which f decreases. Existence can be
proved elementarily by a quadratic penalty: minimize φ_k(x) = f(x) + ½‖x−x̄‖² + (k/2)‖h(x)‖² over a
small ball; the unconstrained minimizers x_k converge to x̄, ∇φ_k(x_k)=0 gives
∇f(x_k) + (x_k−x̄) + Σ_j λ_j^k ∇h_j(x_k) = 0 with λ_j^k = k h_j(x_k), and under linear independence
the λ^k stay bounded, so a limit point yields true multipliers (Haeser–dos Santos 2022, an
elementary modern proof of Lagrange's rule).

**The principle of virtual work and the entry of inequalities (Fourier 1798; Ostrogradsky 1838;
Farkas 1895–1901).** In mechanics, Lagrange's multipliers express equilibrium under *reversible*
(equality) constraints: the virtual work of the applied forces vanishes for every admissible
displacement. Fourier extended the principle to *irreversible* displacements — systems subject to
inequality constraints — and found that equilibrium requires the virtual work to be *nonpositive*
for every feasible displacement (the "Fourier inequality"). Ostrogradsky (1838) derived
equilibrium conditions for such systems and reached a conclusion already containing the sign rule:
the multipliers λ, μ,… must have signs opposite to the corresponding constraint differentials, i.e.
the one-sided constraints force a *sign restriction* on their multipliers, but his coordinate
change only worked when the number of constraints did not exceed the number of variables. Farkas set
out to put this on a clean footing for any number of constraints, and in doing so isolated the
purely algebraic fact that turned out to be the engine of the whole subject.

**Farkas' lemma / the Minkowski–Weyl theory of polyhedral cones (Farkas 1901; Weyl 1935).** The
load-bearing algebraic fact about homogeneous linear inequalities: a linear inequality bᵀx ≤ 0
holds for **all** x satisfying a system Ax ≥ 0 **if and only if** b is a nonnegative combination of
the negatives of the rows of A, i.e. b = −Aᵀt for some t ≥ 0. Geometrically, the set of vectors b
such that bᵀx ≤ 0 on the cone {Ax ≥ 0} is precisely the polar cone generated by the negative rows of
A. The lemma is the standard algebraic instrument for reasoning about which vectors are forced to
lie inside a cone cut out by linear inequalities, and the sign restriction t ≥ 0 is intrinsic to the
statement.

**Convex sets and the separating-hyperplane theorem.** Two nonempty disjoint convex sets C, D in
ℝⁿ can be separated by a hyperplane: there exist a ≠ 0 and b with aᵀx ≤ b for all x ∈ C and
aᵀx ≥ b for all x ∈ D. (For the case where the Euclidean distance between C and D is attained, the
hyperplane orthogonal to the segment joining the two closest points separates them.) This is the
standard geometric instrument available once a set is known to be convex. A function f is convex if
f((1−θ)x⁰+θx) ≤ (1−θ)f(x⁰)+θf(x); a differentiable convex
f satisfies the first-order inequality f(x) ≥ f(x⁰) + ∇f(x⁰)ᵀ(x−x⁰), so its graph lies above every
tangent. (For maximization one works with concave functions, where the inequality reverses.)

**Linear programming and its duality (Dantzig 1947; Gale, Kuhn & Tucker 1951).** Dantzig's linear
program — maximize cᵀx subject to Ax ≤ b, x ≥ 0 — comes with a *dual* program — minimize bᵀu
subject to Aᵀu ≥ c, u ≥ 0 — and the duality theorem says the two share the same optimal value
whenever either is feasible and bounded. The mechanism is a saddle point: form the bilinear
Lagrangian φ(x,u) = cᵀx + uᵀ(b − Ax); then x⁰ solves the primal and u⁰ the dual exactly when (x⁰,u⁰)
is a saddle point, φ(x,u⁰) ≤ φ(x⁰,u⁰) ≤ φ(x⁰,u) for all x ≥ 0, u ≥ 0. The bilinear symmetry of φ in
x and u is what produces the primal–dual symmetry. The same circle of ideas connects LP to
two-person zero-sum games (von Neumann & Morgenstern 1944): a saddle point is a game solution.

## Baselines

The prior treatments of inequality-constrained optimality, against which a new theory would be
measured:

- **The classical multiplier rule (equality constraints).** Core idea: ∇f + Σ ν_j ∇h_j = 0 under
  linear independence of the ∇h_j. Handles only equality constraints; the setting where each h_j = 0
  defines a smooth manifold and the multipliers are unrestricted in sign.

- **Fourier–Ostrogradsky equilibrium conditions.** Core idea: for irreversible (inequality)
  constraints, equilibrium ⟺ virtual work nonpositive for all feasible displacements, with
  multipliers sign-restricted opposite to the constraint differentials. Algorithm/math: a
  coordinate change to "generalized" variables, then the conclusion that the transformed
  coefficients vanish. Developed within the mechanical interpretation (a potential V must exist),
  with Ostrogradsky's coordinate trick applicable when #constraints ≤ #variables.

- **Farkas' programme.** Core idea: a rigorous theory of homogeneous linear inequalities (the
  lemma) sufficient to derive the equilibrium conditions for any number of constraints. Developed
  inside analytical mechanics for the linear (first-order) inequality system; supplies the algebraic
  engine connecting sign-restricted multipliers to polyhedral cones.

- **Linear-programming duality (Gale–Kuhn–Tucker).** Core idea: primal max cᵀx and dual min bᵀu
  coincide; the saddle point of the bilinear Lagrangian solves both. Linearity of the objective and
  constraints enters through the bilinear form; the dual is read off by symmetry of that form, and
  the equivalence between optimum and saddle point rests on linear algebra.

## Evaluation settings

This is a theory question, so the natural yardstick is not a benchmark dataset but a set of
problem classes and self-consistency checks that any proposed conditions must pass:

- **Specialization to the equality case.** With the inequalities absent, any proposed first-order
  conditions must reduce exactly to Lagrange's ∇f + Σ ν_j ∇h_j = 0.
- **Specialization to linear programming.** Applied to max cᵀx s.t. Ax ≤ b, x ≥ 0, the conditions
  and any duality statement must reproduce the known LP duality theorem and its saddle point.
- **Quadratic programs.** The immediate next class: objective and constraints that are (negative
  semidefinite / convex) quadratics, e.g. max Σ c_i x_i − Σ q_ij x_i x_j subject to linear
  inequalities — the proving ground for extending duality beyond the linear case.
- **Test problems with active and slack constraints.** Problems where the optimum lies on the
  boundary of some constraints and the interior of others, to check that the slack-constraint rule
  behaves.
- **Singular boundary geometries.** Configurations with cusps or other irregular boundary points
  (e.g. the region (1−x₁)³ − x₂ ≥ 0, x₁ ≥ 0, x₂ ≥ 0 near (1,0)), to probe exactly when the
  first-order conditions can break and a regularity hypothesis is needed.
- **The mechanical sanity check.** A spring/contact system whose equilibrium minimizes a (convex)
  potential energy subject to non-penetration inequalities, where the multipliers should come out
  as physical contact forces — nonnegative, and nonzero only where there is contact.
