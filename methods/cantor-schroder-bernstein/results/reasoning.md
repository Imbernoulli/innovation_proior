I have two injections and I want a bijection. What about inverting? $g$ is injective, so on its image $g[B]$ it has a genuine inverse $g^{-1} : g[B] \to B$, and that *is* a bijection ŌĆö onto $B$, even. The trouble is it's only a partial function on $A$: it's undefined on $A \setminus g[B]$, the very elements that have no $g$-preimage. So $f$ is total but not onto; $g^{-1}$ is onto but not total. That suggests the shape of the answer right away: use $f$ on part of $A$ and $g^{-1}$ on the rest. Set $C = $ chain of $(A\setminus g[B])$ under $\psi = g\circ f$, so $C$ satisfies the structural identity

Let me make sure I can hit this with the *gluing lemma* too, since that's the route the picture suggested and it's worth having both. So three constructions, one map: the fixed-point cut $C$ with $h = f$ on $C$ and $g^{-1}$ off it; the orbit/graph partition that *explains* the cut by where each component originates; and the shell-by-shell approximation. Each is grounded in nothing but $f$, $g$, images, the partial inverse $g^{-1}$ on $g[B]$, Dedekind's chain-of-a-seed (or the naturals, for the shells), and the disjoint-gluing lemma. None well-orders anything, none invokes Choice; only excluded middle, to classify each element's orbit. Let me also close the loop with the *fundamental lemma* form, because it shows the theorem is "really" a statement about embedding a set into a sandwiched subset, which is conceptually the cleanest. The lemma: if $h_0 : a \to e$ is a bijection and $e \subseteq d \subseteq a$, then $a \approx d$. The theorem reduces to it: given injections $f,g$, the composite $g\circ f : A \to A$ is a bijection onto $g[f[A]]$, and $g[f[A]] \subseteq g[B] \subseteq A$ ŌĆö so with $a = A$, $e = g[f[A]]$, $d = g[B]$, the lemma gives $A \approx g[B]$; compose with $g^{-1} : g[B] \approx B$ and get $A \approx B$. And the lemma itself is the same chain argument in miniature: let $c$ be the chain of $a\setminus d$ under $h_0$ (smallest $h_0$-closed set containing $a\setminus d$), define $h^*(x) = h_0(x)$ on $c$ and $h^*(x) = x$ on $a\setminus c$. By the structural identity $c = (a\setminus d)\cup h_0[c]$, this $h^*$ is a bijection $a \to d$: on $c$ it bijects $c$ onto $h_0[c] = c\setminus(a\setminus d)$, the identity fixes $a\setminus c$, and the pieces $\{c, a\setminus c\}$ partition $a$ while $\{h_0[c], a\setminus c\}$ partition $d$ ŌĆö exactly the same partition-and-glue, with "$\mathrm{id}$" playing the role $g^{-1}$ played before. Seeding the chain at $a\setminus d$ instead of $d\setminus e$ is Dedekind's least-fixed-point variant; seeding at $d\setminus e$ is Zermelo's greatest ŌĆö the same two canonical maps showing up yet again, from the same single idea.

Let me write the result down cleanly, as the theorem and the proof I'll stand behind ŌĆö the chain/fixed-point version, since it's the shortest airtight one and carries the orbit intuition inside it.

> **Theorem (CantorŌĆōSchr├ČderŌĆōBernstein).** Let $A, B$ be sets. If there exist injections $f : A \to B$ and $g : B \to A$, then there exists a bijection $h : A \to B$.

**Proof.** Define the operator $m : \wp(A) \to \wp(A)$ by $m(X) = A \setminus g[\,B \setminus f[X]\,]$. For $X\subseteq Y$, $f[X]\subseteq f[Y]$, so $B\setminus f[X]\supseteq B\setminus f[Y]$, so $g[B\setminus f[X]]\supseteq g[B\setminus f[Y]]$, so $m(X)\subseteq m(Y)$: $m$ is monotone (the two complements cancel). Let
$$ C \;=\; \bigcap\{\,X\subseteq A : m(X)\subseteq X\,\}. $$
For each $X$ with $m(X)\subseteq X$ we have $C\subseteq X$, hence $m(C)\subseteq m(X)\subseteq X$; intersecting over all such $X$, $m(C)\subseteq C$, and applying $m$ once more, $m(C)\subseteq C \Rightarrow m(m(C))\subseteq m(C)$, so $C\subseteq m(C)$; thus $m(C)=C$. Unfolding, the complement of this fixed-point equation reads
$$ A\setminus C \;=\; g[\,B\setminus f[C]\,]. \tag{$\dagger$} $$
Define $h : A \to B$ by
$$ h(x) = \begin{cases} f(x), & x\in C,\\[2pt] g^{-1}(x), & x\in A\setminus C. \end{cases} $$
*Well defined and total.* On $C$, $f$ applies. On $A\setminus C$, by $(\dagger)$ every $x$ is $g(b)$ for some $b\in B\setminus f[C]$, so $x\in g[B]$ and $g^{-1}(x)$ is defined and single-valued ($g$ injective).
*Injective.* If $f(x)=f(x')$ with $x,x'\in C$, then $x=x'$. If $g^{-1}(x)=g^{-1}(x')$ with $x,x'\in A\setminus C$, apply $g$: $x=x'$. If $f(x)=g^{-1}(x')$ with $x\in C$, $x'\notin C$, put $b=f(x)$; then $x'=g(b)=g(f(x))\in g[f[C]]\subseteq C$ by $(\dagger)$'s reading $C\supseteq g[f[C]]$ ŌĆö contradiction. So $h$ is injective.
*Surjective.* Given $b\in B$, consider $g(b)$. If $g(b)\in A\setminus C$ then $h(g(b))=g^{-1}(g(b))=b$. If $g(b)\in C$, then since $g(b)\in g[B]$ it cannot lie in $A\setminus C = g[B\setminus f[C]]$ (which consists of $g$-images of points *outside* $f[C]$); injectivity of $g$ forces $b\in f[C]$, say $b=f(x)$ with $x\in C$, and $h(x)=f(x)=b$. So $h$ is onto.
Hence $h$ is a bijection $A\to B$. $\qquad\blacksquare$

The causal chain, start to finish: I wanted to upgrade two embeddings into one bijection; the only consistent shape was "$f$ here, $g^{-1}$ there," which turned "where to cut" into a self-referential equation $C = A\setminus g[B\setminus f[C]]$; the operator is monotone because its two complements cancel, so a canonical least fixed point exists and is exactly Dedekind's chain of the $g$-preimage-less seed under $g\circ f$; that chain *is* the set of elements whose backward $f,g$-trace originates in $A$, which is why $f$ must be used there and $g^{-1}$ elsewhere; and the structural identity $C = (A\setminus g[B])\cup g[f[C]]$ is precisely what makes the splice total, injective, and onto ŌĆö with no well-ordering and no choice, only excluded middle to read off each element's orbit.
