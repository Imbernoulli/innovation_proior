Quantum mechanics, as it was handed down in the 1920s, is almost entirely a Hamiltonian theory. The state is a wave function or a vector, the dynamical variables are canonical coordinates and momenta tied together by the commutator [q, p] = iℏ, and time evolution is generated by a Hamiltonian operator Ĥ. That is perfectly adequate whenever a Hamiltonian exists, but it is not adequate for every physical system. Some systems, notably electrodynamics formulated as direct action-at-a-distance between charges with a built-in time delay, are specified only by an action S = ∫ L dt and have no Hamiltonian to quantize. More broadly, the action formulation of classical mechanics is in many respects the deeper one: it packages all equations of motion into the single scalar statement δS = 0 and is manifestly Lorentz-invariant, whereas the Hamiltonian formalism singles out a time axis the moment it is written. The question, then, is whether quantum mechanics can be stated directly in terms of the action functional S[x(t)] rather than operators and a Hamiltonian, and whether such a formulation can reproduce both the classical least-action principle and Schrödinger's equation.

The existing formulations do not answer this. Schrödinger's wave mechanics and Heisenberg's matrix mechanics are built on the Hamiltonian side and never give S an operational role. The Wentzel–Kramers–Brillouin approximation does put the classical action in the phase, but it is only an approximation tied to a single classical path and its neighborhood. Dirac came closest in 1933 when he observed that the transformation function ⟨q_{t+dt}|q_t⟩ "corresponds to" exp[(i/ℏ) L dt], and that the classical least-action principle emerges in the ℏ → 0 limit of the quantum composition law. But Dirac stopped at the word "corresponds"; he never fixed the constant of proportionality, never assembled the infinitesimal hint into a single self-contained finite-time object, and never derived Schrödinger's equation from the infinitesimal kernel. What is needed is a construction that closes those three gaps.

The method is the Feynman path integral, also called the sum-over-histories formulation of quantum mechanics. The central postulate is that the amplitude, or propagator, to go from one space-time point to another is obtained by summing complex contributions from every path that connects the two endpoints. Each path x(t) contributes with equal magnitude and with a phase equal to its classical action divided by ℏ: Φ[x(t)] ∝ exp[(i/ℏ) S[x(t)]], where S[x(t)] = ∫_{t_a}^{t_b} L(ẋ, x) dt. Because the magnitudes are equal, all the discriminating power lies in the phase; interference between paths is the sole source of wave behavior. The structure mirrors the quantum law for composite alternatives: amplitudes multiply along a single history, since the action of a whole path is the sum of slice actions and phases add in the exponent, and amplitudes add across different possible intermediate positions, which becomes an integral over all paths in the continuum limit.

In practice the integral is defined by time slicing. Divide the interval [t_a, t_b] into N steps of size ε = (t_b − t_a)/N, fix x_0 = x_a and x_N = x_b, and integrate over the intermediate positions x_1, ..., x_{N−1}. The short-time action for one slice is S(x_{i+1}, x_i) ≈ ε [ (m/2)((x_{i+1} − x_i)/ε)² − V(x_{i+1}) ], and each slice carries a normalization factor 1/A with A = (2π i ℏ ε / m)^{1/2}. That normalization is not chosen by hand; it is forced by the requirement that the kernel reduce to the identity as ε → 0. With A fixed, expanding the one-slice update of the wave function to first order in ε reproduces Schrödinger's equation exactly: iℏ ∂ψ/∂t = −(ℏ²/2m) ∂²ψ/∂x² + V(x)ψ. The equivalence to ordinary quantum mechanics is therefore derived, not assumed. In the classical limit ℏ → 0, the phase S/ℏ oscillates rapidly and neighboring paths cancel everywhere except near the path where S is stationary, δS = 0; the sum over histories collapses onto the classical least-action trajectory, with paths within ΔS ∼ ℏ of it contributing constructively.

A few subtleties are worth keeping in mind. The dominant paths are continuous but nowhere differentiable: the typical step size is of order (ℏε/m)^{1/2}, so the effective velocity diverges as ε → 0. For a scalar potential V(x) this jaggedness only affects higher-order terms, but for velocity-linear terms such as a magnetic coupling (e/c) A(x)·ẋ the slice action must be evaluated at the midpoint, S = ε L((x_{i+1}−x_i)/ε, (x_{i+1}+x_i)/2), because endpoint and midpoint differ at order ε and would otherwise shift the Hamiltonian by a term proportional to ℏ∇·A. Because the Lagrangian couples only neighboring instants, the full multi-integral factorizes at any chosen time into a past part and a future part; the past part is exactly the wave function ψ(x, t), which therefore carries all information needed for future predictions and obeys Schrödinger's equation as shown above.

```python
import numpy as np

def evolve_path_integral(psi0, xs, t_final, N, m, hbar, V):
    """
    Evolve a 1-D wavefunction psi0(x, 0) -> psi(x, t_final) by applying the
    short-time Feynman kernel N times.  Each step uses the Trotterized kernel
    K(x', x) ~ (1/A) exp[ i m (x'-x)^2 / (2 hbar eps) ] exp[ -i eps V(x)/hbar ],
    implemented stably in momentum space via FFT.
    """
    eps = t_final / N
    dx = xs[1] - xs[0]
    p = np.fft.fftfreq(len(xs), d=dx) * 2 * np.pi * hbar

    # Diagonal phase factors: half potential, kinetic, half potential
    V_phase = np.exp(-0.5j * eps * V(xs) / hbar)
    T_phase = np.exp(-1j * eps * p**2 / (2 * m * hbar))

    psi = psi0.copy()
    for _ in range(N):
        psi = V_phase * psi
        psi = np.fft.ifft(T_phase * np.fft.fft(psi))
        psi = V_phase * psi
    return psi

def gaussian_packet(x, x0, p0, sigma):
    norm = (2 * np.pi * sigma**2) ** (-0.25)
    return norm * np.exp(-(x - x0)**2 / (4 * sigma**2)) * np.exp(1j * p0 * x / hbar)

def free_exact(x, t, x0, p0, sigma, m, hbar):
    """Exact freely spreading Gaussian wave packet."""
    alpha = 1 + 1j * hbar * t / (2 * m * sigma**2)
    norm = (2 * np.pi * sigma**2) ** (-0.25) * alpha ** (-0.5)
    return norm * np.exp(-(x - x0 - p0 * t / m)**2 / (4 * sigma**2 * alpha)) \
                 * np.exp(1j * p0 * (x - x0 - p0 * t / (2 * m)) / hbar)

if __name__ == "__main__":
    m, hbar = 1.0, 1.0
    xs = np.linspace(-15, 15, 1024)
    x0, p0, sigma = 0.0, 2.0, 0.5
    t_final = 1.0

    psi0 = gaussian_packet(xs, x0, p0, sigma)
    psi_num = evolve_path_integral(psi0, xs, t_final, N=200, m=m, hbar=hbar,
                                   V=lambda x: 0.0 * x)
    psi_ex = free_exact(xs, t_final, x0, p0, sigma, m, hbar)

    err = np.trapezoid(np.abs(psi_num - psi_ex)**2, xs)
    print(f"Free-particle L2 error: {err:.2e}")
    print(f"Numerical center: {np.trapezoid(xs * np.abs(psi_num)**2, xs):.3f}")

    # Harmonic oscillator: V(x) = 0.5 x^2
    psi_ho = evolve_path_integral(psi0, xs, t_final, N=200, m=m, hbar=hbar,
                                  V=lambda x: 0.5 * x**2)
    print(f"HO final probability at x=0: {np.abs(psi_ho[len(xs)//2])**2:.4f}")
```
