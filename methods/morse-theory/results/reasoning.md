I want a way to understand a smooth manifold without cutting it apart by hand. A manifold is locally Euclidean everywhere, so the local charts by themselves do not tell me where the global holes, components, and tunnels are. A real-valued smooth function feels like the simplest probe I can put on the manifold. I can imagine it as a height, and I can let the water level rise: `M^a={x:f(x)<=a}`. If this is going to say anything precise, most levels had better be boring. Otherwise I have just replaced one hard object by a continuum of hard objects.

So I look first at a band `f^{-1}([a,b])` with no critical points. If `df` never vanishes there, then after choosing a Riemannian metric the gradient is never zero. I do not want merely to move uphill; I want a clock whose time is exactly the change in `f`. The vector field `X=grad f/||grad f||^2` does that, because `df(X)=<grad f,grad f>/||grad f||^2=1`. On a compact band the flow exists for the finite time I need. If a point starts on level `a`, flowing for time `t` puts it on level `a+t`. This means the whole regular band is swept out by trajectories that cut every level once. Nothing can be born, merge, or close inside such a band, because the flow itself identifies the levels and carries one sublevel set to another, with the region below `a` left fixed after a cutoff. That already explains half of the picture: topology is not changing at regular values.

But that only pushes all the work onto the exceptional levels. At a critical point the gradient clock stops, so the regular-flow argument fails exactly there. If I try to classify critical points just by saying "minimum, saddle, maximum," that works on surfaces but it is too pictorial in higher dimensions. The differential vanishes, so the first local information is the Hessian. At a critical point the Hessian is a genuine quadratic form on the tangent space. Its signs give descending and ascending directions. If it has a kernel, I am in trouble: a flat direction can hide higher-order behavior, as in `x^3`, and a small perturbation can remove or split the event. A single topological rule cannot be determined by such a point. I need the Hessian to have no kernel.

Now I test whether the Hessian signs are only approximate information or the whole local story. Taylor expansion says

`f(p+y)=f(p)+quadratic part+higher order terms`.

If the higher order terms remain, I cannot cleanly read off a handle. The necessary local statement is stronger: after a smooth change of coordinates, a nondegenerate critical point is exactly quadratic. That is the Morse lemma. In coordinates `(u,v)` with `u in R^lambda` and `v in R^{n-lambda}`,

`f=c-||u||^2+||v||^2`,

where `c=f(p)` and `lambda` is the number of negative Hessian directions. This is the moment where the Hessian index stops being a diagnostic number and becomes a topological dimension.

Let me cross the critical level in this local model. Below `c`, the inequality is `-||u||^2+||v||^2 <= -epsilon`, or `||v||^2 <= ||u||^2-epsilon`. Near the origin that forces `u` to be nonzero and large enough; the central negative disk is missing. Above `c`, the inequality is `-||u||^2+||v||^2 <= epsilon`, or `||v||^2 <= ||u||^2+epsilon`, and now the set includes the disk in the negative directions with the positive directions thickened around it. The new core is the disk `D^lambda` in the descending directions, and its normal thickening is `D^{n-lambda}` in the ascending directions. The boundary by which it was already visible below is `S^{lambda-1} x D^{n-lambda}`. So the local event is not "a mysterious singularity"; it is the attachment of a `lambda`-handle `D^lambda x D^{n-lambda}`.

I should be careful not to pretend the local chart alone proves the global statement. Outside a small neighborhood of the critical point, there are no critical points in a sufficiently thin band around `c`. The regular-flow argument applies there. So I can push the lower sublevel set up to the boundary of the local chart, perform the one local quadratic change inside the chart, and then push out again. All the global motion away from `p` is product-like; the only non-product piece is the quadratic saddle. That is why the whole topology changes by precisely the handle seen in the normal form.

The index now has a direct meaning. If `lambda=0`, the handle is `D^0 x D^n`, an `n`-disk born from nothing along an empty attaching sphere; that is a new component beginning at a minimum. If `lambda=1`, the handle is `D^1 x D^{n-1}`, attached along two copies of `D^{n-1}`; it can connect components or create a tunnel, depending on the attaching map. If `lambda=n`, the handle is `D^n x D^0`, which caps off along `S^{n-1}`. The same formula covers all dimensions without changing the language.

There is still a nuisance: several critical points could sit at the same value. The mechanism is local, so I want to cross one event at a time. For a generic smooth function the critical points are nondegenerate, and by a small adjustment of values or by taking sufficiently narrow bands when the values are distinct, the critical levels can be separated. Then the manifold is reconstructed by starting below the minimum and crossing the critical values in order. Between them, flow says nothing changes. At each one, the Morse lemma says which handle is attached. The whole manifold is therefore built from handles, and after collapsing each handle `D^lambda x D^{n-lambda}` along its `D^{n-lambda}` factor, I get the homotopy type of a CW complex with one `lambda`-cell for each critical point of index `lambda`.

So the method is not recombining unrelated topological tricks. It is one filtration idea made rigid. Watch the sublevel sets of a generic smooth function. The gradient flow proves regular levels are topologically inert. Nondegeneracy makes critical points isolated and stable. The Morse lemma turns each critical point into the exact quadratic model `c-||u||^2+||v||^2`. The negative eigenspace of the Hessian gives the index, and the index gives the handle dimension. All topology enters only when the rising sublevel set crosses a critical value, and the change there is the corresponding handle attachment.
