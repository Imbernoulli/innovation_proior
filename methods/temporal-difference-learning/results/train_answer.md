We want to learn a prediction of a future-dependent quantity from experience with an incompletely known dynamical system: a game position that forecasts a win, a process state that forecasts a later cost, a random-walk state that forecasts which boundary it will reach. The defining difficulty is that the data do not arrive as input-label pairs; they arrive as a temporal sequence, each observation $x_t$ followed by another observation and only much later by a final outcome $z$. The natural supervised answer is to treat $z$ as the label for every observation in the sequence and run the delta rule, $\Delta w_t = \alpha\,(z - P_t)\,\nabla_w P_t$, where $P_t = P(x_t, w)$. This is the thing that almost works, and its two failures are exactly what point the way out. First, the update has the wrong temporal shape: the target $z$ is not known when the early predictions are made, so the learner must retain every observation or gradient until the sequence ends, spending sequence-length-dependent memory. Second, and more deeply, the realized final outcome is a poor target for an earlier state. If a novel game position leads straight into a position I already know loses most of the time, but this particular episode ends in a lucky win, the supervised update against the win raises the value of the novel position — exactly backwards. What I actually want to predict is the expected outcome from the current state, and the one sampled outcome that happened to follow includes all the randomness that occurred after the state I am trying to evaluate. The continuation is a cleaner standard than the eventual return. The Widrow-Hoff baseline cannot exploit this, the Rescorla-Wagner model collapses a whole trial into one reinforcement term and cannot handle within-trial timing, Samuel's checker player adjusted earlier evaluations toward later ones but had no terminal grounding and admitted self-consistent useless value functions, and dynamic programming bootstraps correctly but normally needs a known transition model. So the open question is whether the relation between successive observations can itself supply a training signal before the outcome is known.

I propose temporal-difference learning: a family of prediction methods that update each prediction from the error in its own temporal consistency rather than from the final label. The starting move is an exact identity. Define the terminal outcome to be one more prediction, $P_{m+1} = z$. Then the whole supervised error telescopes into a sum of successive prediction differences,
$$z - P_t = P_{m+1} - P_t = \sum_{k=t}^{m} (P_{k+1} - P_k).$$
Substituting this into the total weight change over a sequence and swapping the order of the two sums turns the delayed supervised update into an incremental one,
$$\sum_{t=1}^{m} \alpha\,(z - P_t)\,\nabla_w P_t = \sum_{k=1}^{m} \alpha\,(P_{k+1} - P_k)\sum_{t=1}^{k}\nabla_w P_t,$$
so at temporal step $k$ I need only the new prediction difference $P_{k+1} - P_k$ and a running sum of past gradients. For a linear predictor this produces the same per-sequence weight change as Widrow-Hoff, but the work is spread through the sequence instead of being held back until the outcome appears, which already kills the memory problem. Equal credit to every past prediction is only one endpoint, though. A more local credit-assignment rule should weight recent predictions more strongly, which I get with an exponentially decaying eligibility trace $e_t = \sum_{k=1}^{t}\lambda^{t-k}\nabla_w P_k$ for $0 \le \lambda \le 1$. The exponential form is the load-bearing choice because it, and essentially only it, collapses into an online recursion,
$$e_t = \nabla_w P_t + \lambda\,e_{t-1},$$
giving the update $\Delta w_t = \alpha\,(P_{t+1} - P_t)\,e_t$. At $\lambda = 1$ the trace is the full running sum and I recover the incremental rearrangement of the final-outcome rule; at $\lambda = 0$ the trace is just the current gradient and the update becomes $\Delta w_t = \alpha\,(P_{t+1} - P_t)\,\nabla_w P_t$. That endpoint is the conceptual break: the final outcome has been replaced by the next prediction. This is bootstrapping — improving one estimate using another estimate — and because the target $P_{t+1}$ depends on $w$ it is not gradient descent on any fixed supervised objective. It is nonetheless principled, because the quantity I am after is the expected outcome and the next prediction can be a lower-variance estimate of that expectation than the sampled return. The one danger this introduces is that if predictions are trained only to agree with other predictions, a constant useless function satisfies the consistency constraint perfectly. The cure is to anchor the boundary: in a terminal episode the last prediction is pinned to the actual external outcome, and in a cumulative-signal problem the observed immediate signal enters directly. That anchored boundary lets real information propagate backward through the differences.

For the general cumulative case I rewrite the target rather than starting over. If the return is a discounted sum $z_t = c_{t+1} + \gamma c_{t+2} + \gamma^2 c_{t+3} + \cdots$, it obeys the recursion $z_t = c_{t+1} + \gamma z_{t+1}$, so correct predictions must satisfy $P_t = c_{t+1} + \gamma P_{t+1}$, and the learning signal is the violation of exactly that equation, the temporal-difference error
$$\delta_t = c_{t+1} + \gamma P_{t+1} - P_t.$$
The trace must now decay by the same discount that scales future credit, so $e_t = \nabla_w P_t + \gamma\lambda\,e_{t-1}$, and the weight update is $w \leftarrow w + \alpha\,\delta_t\,e_t$; an undiscounted final-outcome task is the special case $\gamma = 1$ with intermediate signals zero and the terminal prediction pinned to the outcome. What makes this combination work is that it merges the two prior families that each had one half: like Monte Carlo learning it samples actual trajectories and needs no transition model, and like dynamic programming it bootstraps from current successor estimates — it samples the actually observed successor and bootstraps from the current prediction there, instead of waiting for the complete return (Monte Carlo) or sweeping over a known model's successors (DP). The conditioning interpretation falls out of the same error: Rescorla-Wagner's "actual minus predicted reinforcement" $\lambda - \bar V$ becomes, in real time, $\lambda_{t+1} + \gamma \bar V_{t+1} - \bar V_t$, immediate signal plus successor prediction minus current prediction, so a prediction can act as a secondary reinforcer and move credit backward before any primary outcome arrives.

The bootstrapped endpoint is also provably sound in the clean linear case, which is what convinced me it was not merely a heuristic. Let the data come from an absorbing Markov chain with nonterminal transition matrix $Q$ and expected terminal contribution $h_i$ when state $i$ exits; since $Q^n \to 0$, the ideal predictions are $v = h + Qv = (I - Q)^{-1}h$. Writing $X$ for the matrix of state observation vectors and $D = \mathrm{diag}(d_i)$ for expected visit counts, the expected TD(0) recursion across sequences is $X^\top \bar w_{n+1} = \alpha X^\top X D h + [I - \alpha X^\top X D (I - Q)] X^\top \bar w_n$, whose fixed point is exactly $(I - Q)^{-1}h$ provided the powers of $M = I - \alpha X^\top X D (I - Q)$ vanish. The whole convergence question is therefore whether $M^n \to 0$. The key fact is that $B = D(I - Q)$ is positive definite: its symmetric part $S = B + B^\top$ has positive diagonal $S_{ii} = 2 d_i(1 - p_{ii})$ and nonpositive off-diagonal entries, and using $d^\top(I - Q) = \mu^\top$ each row is weakly diagonally dominant with a strict row in every connected component (absorption forbids a closed nonterminal component), so the diagonal-dominance lemma makes $S$, and hence $B$, positive definite. Lifting through $X$, every eigenvalue $\mu = a + bi$ of $A = X^\top X D(I - Q)$ has $a > 0$, the eigenvalues of $M$ are $1 - \alpha\mu$ with $|1 - \alpha\mu|^2 = 1 - 2\alpha a + \alpha^2(a^2 + b^2)$, and this is below one exactly when $0 < \alpha < 2a/(a^2 + b^2)$. Taking $\epsilon$ as the minimum of that bound over the finitely many eigenvalues, every $0 < \alpha < \epsilon$ drives the expected predictions to $(I - Q)^{-1}h$. There is a second guarantee that explains why finite data is used differently: presented a fixed training set repeatedly, Widrow-Hoff converges to the best fit of the raw observed returns, while TD(0) converges to the certainty-equivalence predictions $(I - \hat Q)^{-1}\hat h$ of the empirical Markov model — so if state $A$ is seen once ending in $0$ but always transitions to a well-sampled state $B$ worth $0.75$, outcome-fitting drags $A$ toward $0$ while TD lets $A$ inherit $B$'s value through the transition structure.

The method in full is small: keep a predictor, keep an eligibility trace, and after each sampled transition form the error from immediate signal plus discounted successor prediction minus current prediction, then nudge every currently eligible weight by it. With $\lambda = 0$ this is pure one-step bootstrapping; with $\lambda = 1$ in an undiscounted finite episode it recovers the incremental supervised sequence update; and the values between them trade lower-variance bootstrapping against longer-range return propagation.

```python
import numpy as np

class TDLambda:
    def __init__(self, n_features, alpha=0.1, gamma=1.0, lam=0.0):
        self.w = np.zeros(n_features)
        self.e = np.zeros(n_features)
        self.alpha = alpha
        self.gamma = gamma
        self.lam = lam

    def predict(self, x):
        return float(self.w @ x)

    def step(self, x, reward, x_next):
        v = self.predict(x)
        v_next = 0.0 if x_next is None else self.predict(x_next)
        delta = reward + self.gamma * v_next - v
        self.e = self.gamma * self.lam * self.e + x
        self.w += self.alpha * delta * self.e
        return delta

    def end_episode(self):
        self.e.fill(0.0)
```
