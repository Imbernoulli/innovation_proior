# Answer: Supyire Numerals

Supyire numerals here are additive, with **na** marking "+". The system uses the places **5, 10, 20, 80, 400**.

## Basic Values

| value | Supyire |
|---:|---|
| 1 | niŋkin |
| 2 | shuunni |
| 3 | taanre |
| 4 | sicyɛɛre |
| 5 | kaŋkuro |
| 10 | kɛ |
| 20 | beɲjaaga |
| 80 | ŋkuu |
| 400 | kampwoo |

The values 6-9 are fused "5 + n" compounds:

| value | Supyire | calculation |
|---:|---|---:|
| 6 | baani | 5 + 1 |
| 7 | baashuunni | 5 + 2 |
| 8 | baataanre | 5 + 3 |
| 9 | baaricyɛɛre | 5 + 4 |

The multiplied place forms are:

| value pattern | form |
|---|---|
| 20 x n | **be-/bee-** + multiplier, as in beeshuunni = 40 and beetaanre = 60 |
| 80 x n | **ŋkwuu** + multiplier, as in ŋkwuu shuunni = 160, ŋkwuu taanre = 240, ŋkwuu sicyɛɛre = 320 |
| 400 x n | **kampwɔhii** + multiplier, as in kampwɔhii shuunni = 800, kampwɔhii taanre = 1200, kampwɔhii sicyɛɛre = 1600 |

Free single-place forms use **beɲjaaga** for 20, **ŋkuu** for 80, and **kampwoo** for 400.
