# β-VAE

## Problem

Learn, fully unsupervised, an interpretable *disentangled* latent representation of image data — each latent coordinate sensitive to a single independent generative factor (position, scale, rotation, lighting, …) and invariant to the others — using a model that provides a proper inference network, and provide a way to *quantify* disentanglement so models can be compared and the method tuned.

## Key idea

Take the variational autoencoder and reweight its objective. Frame the goal as "maximise reconstruction quality subject to a budget on how far the posterior may stray from an isotropic, factorised prior." The KL term against `p(z) = N(0, I)` simultaneously caps the latent channel's capacity and rewards independence across coordinates; pressing on it harder forces the model into the most *efficient* code, which — since the data is generated by conditionally independent factors — is a disentangled one. Casting the constrained problem as a KKT Lagrangian makes the entire modification a single scalar `β` multiplying the KL:

`L(θ, φ; x, β) = E_{q_φ(z|x)}[log p_θ(x|z)] − β · D_KL(q_φ(z|x) ‖ p(z))`.

`β = 1` recovers the standard VAE. `β > 1` strengthens the capacity/independence constraint and induces disentanglement, at the cost of blurrier reconstructions and possibly dropping factors tied to small pixel changes — so `β` is tuned, not maximised. Across datasets the comparable quantity is the normalised `β_norm = β·M/N` (`M` latents, `N` pixels). `β` cannot be learned directly: it is the exchange rate `δG/δε = β*(ε)` between reconstruction and the capacity budget `ε`, set from outside via the disentanglement metric or visual inspection.

## Disentanglement metric

Measures independence *and* interpretability together. Over a batch of `L` pairs: pick a target factor `y` uniformly; for each pair sample two factor-vectors that share the value of factor `y` and differ randomly in all others; render the images and infer their latent means `z_{1,l} = μ(x_{1,l})`, `z_{2,l} = μ(x_{2,l})`; form `z_diff^l = |z_{1,l} − z_{2,l}|` and average over the batch, `z_diff^b = (1/L) Σ_l z_diff^l`. Train a low-VC-dimension *linear* classifier to predict `y` from `z_diff^b`; its accuracy is the score. A disentangled code puts the shared factor in a single low-variance coordinate that a weak linear rule can read off; the deliberately weak classifier ensures the score reflects the representation, not the classifier.

## Architecture and objective

Diagonal-Gaussian encoder `q_φ(z|x) = N(μ(x), σ²(x))`, isotropic-Gaussian prior `p(z) = N(0, I)`, reparameterisation `z = μ + σ·ε`. The KL has a closed form, per latent dimension: `KL = −½(1 + log σ² − μ² − σ²)`. On 2D shapes: FC 1200/1200 encoder–decoder, 10 latents, Adagrad 1e-2, Bernoulli decoder, `β = 4`. On CelebA/chairs/faces: four 32-channel 4×4 stride-2 convs then FC 256, 32 latents, mirror deconv decoder, Adam 1e-4.

## Code

```python
import torch
from torch import nn
import torch.nn.functional as F

def reparametrize(mu, logvar):
    std = logvar.div(2).exp()
    eps = torch.randn_like(std)
    return mu + std * eps


class BetaVAE(nn.Module):
    def __init__(self, z_dim=10, nc=3):
        super().__init__()
        self.z_dim = z_dim
        self.encoder = nn.Sequential(
            nn.Conv2d(nc, 32, 4, 2, 1), nn.ReLU(True),
            nn.Conv2d(32, 32, 4, 2, 1), nn.ReLU(True),
            nn.Conv2d(32, 64, 4, 2, 1), nn.ReLU(True),
            nn.Conv2d(64, 64, 4, 2, 1), nn.ReLU(True),
            nn.Conv2d(64, 256, 4, 1),  nn.ReLU(True),
            nn.Flatten(),
            nn.Linear(256, z_dim * 2),
        )
        self.decoder = nn.Sequential(
            nn.Linear(z_dim, 256), nn.Unflatten(1, (256, 1, 1)), nn.ReLU(True),
            nn.ConvTranspose2d(256, 64, 4), nn.ReLU(True),
            nn.ConvTranspose2d(64, 64, 4, 2, 1), nn.ReLU(True),
            nn.ConvTranspose2d(64, 32, 4, 2, 1), nn.ReLU(True),
            nn.ConvTranspose2d(32, 32, 4, 2, 1), nn.ReLU(True),
            nn.ConvTranspose2d(32, nc, 4, 2, 1),
        )

    def forward(self, x):
        dist = self.encoder(x)
        mu, logvar = dist[:, :self.z_dim], dist[:, self.z_dim:]
        z = reparametrize(mu, logvar)
        return self.decoder(z), mu, logvar


def reconstruction_loss(x, x_recon, distribution="bernoulli"):
    B = x.size(0)
    if distribution == "bernoulli":
        return F.binary_cross_entropy_with_logits(x_recon, x, reduction="sum") / B
    return F.mse_loss(x_recon, x, reduction="sum") / B


def kl_divergence(mu, logvar):
    # closed-form KL( N(mu, sigma^2) || N(0, I) ), summed over latents, averaged over batch
    klds = -0.5 * (1 + logvar - mu.pow(2) - logvar.exp())
    return klds.sum(1).mean(0)


# training
beta = 4.0  # beta = 1 recovers the standard VAE; beta > 1 induces disentanglement

for x in loader:

    recon = reconstruction_loss(x, x_recon, distribution="bernoulli")
    total_kld = kl_divergence(mu, logvar)
    loss = recon + beta * total_kld
    optimizer.zero_grad(); loss.backward(); optimizer.step()
```
