I start with the thing the scalar theory silently destroys. A return is not born as a number. It is a random sum, `Z(x,a) = sum_t gamma^t R_t`, with randomness entering through rewards, transitions, and possibly the policy. The number I normally call `Q(x,a)` is only `E Z(x,a)`. That expectation is useful because it satisfies Bellman's equation and gives me a contraction, but it is also a collapse. If one future means survival and another means immediate failure, the mean may describe neither future.

So the first insight is to stop asking how to attach uncertainty to a value estimate and instead ask what object Bellman's recursion is already acting on before expectation. If I remove the outer averaging from the scalar equation, I get a random-variable recursion:

`Z(x,a) =_D R(x,a) + gamma Z(X',A')`.

The equality has to be equality in distribution. I am not matching one sampled trajectory to another sampled trajectory; I am matching laws. The right side is a compound random variable made from the immediate reward, the random next state-action, and the next return law. That immediately changes the problem. I no longer need a scalar norm on value vectors. I need a distance between distributions, and then a worst-case lift of that distance over all state-action pairs.

My first instinct is to hope that any familiar distribution discrepancy will do, but the discount factor tells me otherwise. Discounting does a geometric operation: it shrinks the support along the return axis. A metric that only cares about overlap or likelihood will not necessarily shrink when two point masses move closer together. Two distinct atoms have maximal total-variation distance whether they sit at `0` and `1` or at `0` and `gamma`. KL and Kolmogorov have similar vertical habits. The contraction cannot come from a metric that is blind to the ground distance between returns.

This points me to the fixed-point method for recursive distributions. Roesler's Quicksort analysis treats a limiting random variable as the fixed point of a distributional map and proves contraction in a Wasserstein/Mallows metric. That is the geometry I need because it measures transport along the value axis. For cdfs `F` and `G`, `d_p(F,G)` is the best `L_p` distance over couplings, equivalently the `L_p` distance between inverse cdfs under a shared uniform draw. If I multiply both random variables by `gamma`, the horizontal transport distance scales by `gamma`. If I add the same reward random variable to both, the distance does not increase.

Now the evaluation proof almost writes itself. I define `P^pi Z(x,a) =_D Z(X',A')` and `T^pi Z(x,a) =_D R(x,a) + gamma P^pi Z(x,a)`. For two candidate return laws, at a fixed state-action pair,

`d_p(T^pi Z_1, T^pi Z_2) <= gamma d_p(P^pi Z_1, P^pi Z_2)`.

The successor operator is only a mixture over next state-actions, so I couple the same successor draw on both sides and then use an optimal conditional coupling for that successor. The remaining distance is no larger than the worst state-action distance between `Z_1` and `Z_2`. Taking the supremum over state-actions gives

`bar d_p(T^pi Z_1, T^pi Z_2) <= gamma bar d_p(Z_1, Z_2)`.

That is the fixed-point theorem I wanted. For a fixed policy, the whole return distribution is a legitimate Bellman object: the operator has a unique fixed point, iteration converges geometrically in maximal Wasserstein, and the fixed point is the actual random return law. This is more than learning a confidence interval around `Q`. It says the law itself is the dynamic-programming quantity, and `Q` is only its first moment.

I need to be careful with moments. It is tempting to say the Wasserstein proof automatically gives every summary I might want, but variance is not just `d_2`. The squared coupling distance contains a mean term and a variance term, so variance needs its own recursion. Under policy evaluation, the variance component propagates with `gamma^2`, matching Sobel's older moment result. That reinforces the distinction: moment recursions are shadows of the full law, not replacements for it.

Then I try to carry the same idea to control, and this is where the scalar analogy breaks. In the usual Bellman optimality operator, the max returns a number. If several actions are optimal, the scalar value does not remember which one I picked. A return distribution does remember. Two optimal policies can share the same mean value and have different return laws, so there is not generally a single optimal return distribution.

To define a control operator I must select a greedy policy according to `E Z`. That selection rule is now part of the operator. The mean still behaves because expectation commutes with the distributional backup and gives the ordinary scalar optimality operator. But the distribution does not inherit the contraction. A tiny perturbation in one action's mean can flip the greedy action. The backup then does not move the old distribution a little; it replaces it with the distribution of another action.

The two-state example makes the wall concrete. From the second state, one action gives a terminal zero and another gives `epsilon +/- 1`, so the second action is uniquely better in mean. If I perturb only that second action to `-epsilon +/- 1`, the distance to the true optimal law is only `2 epsilon` in `d_1`, but the greedy action flips to the zero action. The backed-up law at the predecessor becomes a point mass at zero instead of the bimodal law. For small `epsilon`, the post-backup distance is about one. The operator expands. With tie-breaking it can even cycle; with multiple optimal policies, iterates can approach laws generated by nonstationary sequences of optimal policies rather than a stationary optimal law.

This is the second major insight: making the full return distribution the Bellman object solves the fixed-policy contraction problem and exposes a control problem that scalar values hide. The paper is not saying "use distributions because uncertainty estimates are nice." It is saying that policy evaluation has a stronger Bellman fixed point in distribution space, while greedy control is intrinsically discontinuous there.

The practical algorithm adds another wall. If I represent each return law by probabilities on fixed atoms `z_i`, the exact backup sends those atoms to `r + gamma z_i`. The target support usually no longer matches the prediction support. Wasserstein can compare disjoint supports, so it looks like the principled loss. But sampled reinforcement learning gives me one transition at a time, and the sample Wasserstein objective is not an unbiased stochastic version of the true Wasserstein objective. For a mixture target, the expected distance to sampled components can be a strict upper bound with the wrong gradient. The simple Bernoulli example is damning: the true `d_1` loss has a V-shaped minimum at the right mixture, while the averaged sampled loss is flat.

So the implementable move is to project the sampled backup back onto the fixed atom grid. Each transformed atom `[r + gamma z_j]` is clipped to the support range and its probability is split linearly between the two nearest atoms. Now target and prediction live on the same support, and the loss becomes cross-entropy, like a soft classification target. Later analysis explains why this projection has its own geometry: the categorical projection is a non-expansion in Cramer distance, and the projected Bellman operator contracts there with modulus `sqrt(gamma)`. That does not replace the original Wasserstein theorem; it explains why the fixed-grid approximation can be stable even though the raw theorem alone did not justify the projection.

The final picture is coherent. First, define the random return law as the Bellman object. Second, prove fixed-policy contraction by choosing a transport metric that sees discounting as shrinkage. Third, admit that greedy control breaks distributional contraction because action selection is discontinuous in the law. Fourth, when building a learner, project the sampled transformed support to a finite grid and train the projected target with a sampling-friendly loss. The conceptual advance is the Bellman fixed-point theory for the whole return distribution, and the negative control result is part of that advance rather than a footnote.
