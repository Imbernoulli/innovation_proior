# Context

## Research question

How can we design a reinforcement-learning algorithm that is *provably* efficient — efficient in both runtime and sample complexity — in a setting where the state space is so large that we must use function approximation rather than a lookup table?

The difficulty is specific and sharp. In an episodic Markov decision process MDP(S, A, H, P, r), an agent interacts for K episodes; in each episode it sees an initial state x₁ (chosen adversarially), and at each of the H steps observes a state, picks an action, collects a reward in [0,1], and transitions according to an unknown kernel P_h(·|x,a). Performance is measured by regret,

  Regret(K) = Σ_{k=1}^K [ V*₁(x₁ᵏ) − V^{π_k}₁(x₁ᵏ) ],

the cumulative shortfall against the optimal policy along the *learner's own* trajectory. The agent has no simulator: it cannot query arbitrary state-action pairs, it cannot reset within an episode. It must solve the exploration/exploitation tradeoff using only the data its own policy generates.

When S (the number of states) is small, this is the tabular problem, and it is solved. The trouble is the modern regime — Atari, Go, robotics — where S is effectively infinite, and the agent will never visit the neighborhood of most states even once. There, a guarantee whose sample complexity scales with S is vacuous. What we need is a regret bound that depends on an *intrinsic complexity measure of the function class* — its dimension d — and is independent of S and A. A solution would have to manage exploration (visit informative states) while approximating value functions from sparse, correlated, possibly-misspecified data generated by a moving policy in a dynamical system. Whether such an algorithm even exists has been open, and it remains open even in the most basic case — when the dynamics and rewards are themselves linear, so that only linear function approximation is required.

## Background

**The MDP machinery.** For a policy π, the value and action-value functions are V^π_h(x) = E[Σ_{h'≥h} r_{h'} | x_h=x], Q^π_h(x,a) = r_h(x,a) + E[Σ_{h'>h} r_{h'} | x_h=x, a_h=a]. They satisfy the Bellman equation Q^π_h = r_h + P_h V^π_{h+1}, V^π_h(x) = Q^π_h(x, π(x)), with V^π_{H+1} ≡ 0, where (P_h V)(x,a) := E_{x'∼P_h(·|x,a)} V(x'). There is an optimal policy π* with V*_h(x) = sup_π V^π_h(x), characterized by the Bellman optimality equation Q*_h = r_h + P_h V*_{h+1}, V*_h = max_a Q*_h, and π* is greedy w.r.t. {Q*_h}. So to act optimally it suffices to estimate the optimal action-value functions. Because rewards lie in [0,1], every value function lies in [0, H].

**Value iteration and its least-squares form.** Classical value iteration applies the Bellman optimality operator recursively: Q*_h ← r_h + P_h max_{a'} Q*_{h+1}(·,a'). Two obstacles block this in practice: P_h is unknown, and with a huge state space we cannot iterate over all (x,a). The classical response (Bradtke & Barto 1996; Osband et al. 2014) is least-squares value iteration (LSVI): parametrize the action-value by a linear form w_h^⊤φ(x,a), and replace the Bellman update with a regression — fit w_h to the targets r_h(xᵗ,aᵗ) + max_a Q_{h+1}(x'ᵗ, a) on the observed data. LSVI is the standard algorithm "for the linear setting," but it carries *no* exploration mechanism and *no* sample-complexity guarantee; on its own it can fail to explore the uncertain directions of feature space.

**Optimism, the engine of exploration.** The principle that drives every efficient exploration algorithm is optimism in the face of uncertainty (OFU): maintain a confidence set for the unknown quantity and act as if the most favorable plausible value were true; this automatically draws the agent toward under-explored options, because uncertainty inflates their apparent value. In tabular RL this takes the form of confidence sets on the model or on Q, with a bonus that shrinks as a state-action pair is visited more.

**The self-normalized / elliptical confidence width.** For *linear* estimation there is a precise, geometry-aware notion of uncertainty. If we fit w by ridge regression with Gram matrix Λ = Σ_t φ_t φ_t⊤ + λI, then the prediction error at a query φ is controlled by ‖φ‖_{Λ⁻¹} = √(φ⊤Λ⁻¹φ). This quantity is large exactly along directions in which few samples have been collected; (φ⊤Λ⁻¹φ)⁻¹ behaves like the *effective number of samples along φ*. The error of a ridge predictor on adversarially-chosen inputs is |ŷ(φ) − y(φ)| ≤ ‖φ‖_{Λ⁻¹}·O(V_max √(d + log 1/δ)), a "one-shot" generalization bound where the sample size is hidden inside Λ⁻¹ rather than appearing as 1/n.

**Two tools that make that bound provable.** (i) The *self-normalized tail inequality for vector-valued martingales* (Abbasi-Yadkori, Pál & Szepesvári 2011): if η_t is conditionally R-sub-Gaussian and φ_t is predictable, then with probability 1−δ, ‖Σ φ_s η_s‖²_{Λ_t⁻¹} ≤ 2R² log(det(Λ_t)^{1/2} det(Λ_0)^{−1/2}/δ) for all t simultaneously — the deviation is measured in the data-dependent norm Λ_t⁻¹, hence "self-normalized." It yields a confidence ellipsoid ‖θ̂_t − θ_*‖_{Λ_t} ≤ R√(d log((1+tL²/λ)/δ)) + λ^{1/2}S_θ around the ridge estimate, when ‖θ_*‖≤S_θ. (ii) The *elliptical potential lemma*: for any bounded sequence with ‖φ_t‖≤1 and Λ_t = Λ_0 + Σ_{j≤t} φ_jφ_j⊤ with λ_min(Λ_0)≥1,

  Σ_{j=1}^t φ_j⊤ Λ_{j−1}⁻¹ φ_j ≤ 2 log[det(Λ_t)/det(Λ_0)] ≤ 2d log((λ+t)/λ).

This is the "the learner runs out of directions" bound: the cumulative confidence width grows only logarithmically per dimension, because each informative observation grows det(Λ), and det can grow at most d log T worth. It is what turns per-step uncertainty into a √(dT) total.

**Covering numbers for continuous classes.** When a guarantee must hold uniformly over a continuous function class F (not a single fixed function), one pays log N_ε, the log of the ε-covering number, instead of log|F|. For a linear class {φ⊤θ : ‖θ‖≤B} one grids the parameter ball: log N_ε ≲ d log(B√d/ε), polynomial in d, poly-logarithmic in everything else.

**The empirical state of the field.** Tabular RL is essentially settled: the best model-based regret is Õ(√(H²SAT)) and the best model-free is Õ(√(H³SAT)), both (nearly) matching the minimax lower bound Ω(√(H²SAT)) (Jaksch et al. 2010; Azar et al. 2017; Jin et al. 2018). The lower bound is itself the diagnostic fact that frames the whole problem: information-theoretically, a large state space *cannot* be handled efficiently unless extra structure is exploited — every tabular guarantee scales with √S, and S is exponentially large in the applications. Linear *bandits* (the H=1 case, no transitions) are solved with regret Õ(d√T) for linear bandits and Õ(√(dT)) for linear contextual bandits, with no dependence on the number of arms — function approximation works there. The open gap is the whole region between: genuine multi-step RL, no simulator, function approximation, with a regret free of S.

## Baselines

- **Tabular optimistic RL — UCRL2 (Jaksch, Ortner & Auer 2010).** Build confidence sets around the empirical transition probabilities and rewards from past samples; run extended value iteration to compute an optimistic policy over the confidence set; act greedily. Achieves Õ(D S √(AT)) regret (D the diameter). Core idea — optimism via model confidence sets — is exactly right, but the bound scales linearly in S: it estimates and stores the entire transition matrix, so it is hopeless for large state spaces in both statistics and runtime.

- **Tabular model-free UCB — Q-learning with UCB (Jin, Allen-Zhu, Bubeck & Jordan 2018).** Incremental Q-learning with a UCB exploration bonus b_t = c√(H³ι/t) and a carefully reweighted learning rate α_t = (H+1)/(H+t). Maintains only the Q-table (O(SAH) space), online. Achieves Õ(√(H⁴SAT)) with the Hoeffding bonus and Õ(√(H³SAT)) with a sharper Bernstein bonus. The regret analysis introduces the structure later reused under function approximation: a recursive bound on the per-episode value gap, an Azuma-Hoeffding control of the martingale-difference part, and a bonus-sum part. But it is still tabular: the bonus 1/√t counts visits to a *single* state-action cell, so the bound scales with √S and the method cannot generalize across states.

- **Linear bandits — OFUL (Abbasi-Yadkori, Pál & Szepesvári 2011).** Optimism in the face of uncertainty for the linear bandit y = ⟨x, θ_*⟩ + noise: maintain a confidence ellipsoid C_t for θ_* (from the self-normalized bound), choose (X_t, θ̃_t) = argmax over (D_t × C_{t−1}) of ⟨x,θ⟩, i.e. play the action with the highest optimistic reward; equivalently a ridge estimate plus a β‖x‖_{Λ⁻¹} bonus. Regret Õ(d√T), independent of the number of arms; the analysis is exactly the elliptical-potential argument. The gap: a bandit has no state transition. The world is the same after each pull, so there is no credit-assignment over a horizon. Transplanting OFUL naively into an H-step MDP — treating each step's reward as a separate bandit — makes the optimism compound multiplicatively across steps and yields regret *exponential* in H.

- **Linear RL with extra power — simulator / low-rank / deterministic variants.** Yang & Wang (2019) give a sample-efficient algorithm in the linear-transition-and-reward setting *assuming a simulator* that returns the reward and next state for any queried (x,a) — which removes the exploration problem (uniform random querying already suffices). A separate line (Yang & Wang 2019b) assumes the transition model is parameterized by a small finite matrix, reducing it to estimating that matrix; others assume deterministic transitions (Wen & Van Roy 2013/2017) or low-variance transitions (Du et al. 2019). General function-approximation results (Olive, Jiang et al. 2017) are sample-efficient when a "Bellman rank" is small but are not computationally efficient and give no √T regret. Each buys tractability by removing the actual difficulty — exploration with an infinite-degree-of-freedom stochastic model and no oracle.

## Evaluation settings

This is a theory result; the yardstick is analytical, not a benchmark suite. The setting is the episodic MDP MDP(S, A, H, P, r) with S a measurable space of possibly infinite cardinality, A finite, non-stationary P_h and r_h, rewards in [0,1], adversarial initial states, no simulator, no resets. A feature map φ: S×A → ℝ^d is given, with ‖φ(x,a)‖ ≤ 1. The figure of merit is high-probability regret Regret(K) = Σ_k [V*₁(x₁ᵏ) − V^{π_k}₁(x₁ᵏ)] as a function of d, H, T=KH, expressed up to absolute constants and poly-log factors (Õ). Secondary metrics are runtime and space, also required to be independent of S. The reference points the bound is read against are the tabular minimax rate Ω(√(H²SAT)) and the linear-bandit rate Õ(d√T) (the H=1 special case). A robustness axis is misspecification: a ζ-approximate linear MDP, where the true P_h and r_h are within total-variation / absolute distance ζ of the linear model, with the question being how gracefully the regret degrades.
