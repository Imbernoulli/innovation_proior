## Research question

经典信息的一个默认直觉是：如果我拥有一段信息，就可以把它复制到空白载体上。比特串、文件、数学描述都可以在不改变原件的情况下重复写入另一个介质。No-cloning theorem 问的是：这个直觉能否原封不动搬到量子态上？

问题不是能否复制某些已知态。正交基态可以通过受控操作复制，例如把计算基态 `|0>` 和 `|1>` 复制到一个空白比特上。真正的问题是是否存在一个通用物理过程，对任意未知纯态 `|psi>` 和固定空白态 `|b>` 都实现

`|psi>|b> -> |psi>|psi>`.

No-cloning theorem 的独特洞察是：量子演化的线性与内积保持本身就禁止这种 universal copier。限制不是工程能力不足，而是量子信息结构的数学后果。

## Background

封闭量子系统的可逆演化由幺正变换描述。幺正变换是线性的，并保持态向量之间的内积。线性意味着如果一个过程分别作用在 `|0>` 和 `|1>` 上，那么它作用在叠加态 `a|0>+b|1>` 上的结果必须是对应输出的同样线性组合。

复制映射看起来也很自然：给定一个空白寄存器，把输入态写到空白寄存器中，同时保留原输入。若只考虑一组互相正交的候选态，这可以做到，因为正交态可以被完美区分，并可用条件操作转写。

任意未知量子态则不同。非正交态不能被完美区分；更深层的是，复制任意态要求一个线性、内积保持的物理演化实现一个非线性映射 `|psi> -> |psi>|psi>`。这个映射会把态之间的内积从 `<psi|phi>` 变成 `<psi|phi>^2`，一般不可能保持原内积。

## Baselines

- **经典复制器。** 对经典比特串，复制是允许的，因为状态集合可视为彼此可区分的离散标签。这个模型默认信息可以从载体中读出并重写到另一个载体上。

- **正交态复制器。** 对已知正交集合，可以构造复制操作，例如 `|0>|0> -> |0>|0>` 与 `|1>|0> -> |1>|1>`。局限是它只适用于该正交集合，不能推广到任意叠加态。

- **测量后重制。** 如果先测量未知态再按结果制备副本，非正交未知态的测量结果不能给出完整态信息，并且测量通常扰动态本身。因此这不是 universal copier。

- **状态估计。** 多个同分布样本可以让人估计制备过程，但单个未知态不能被无扰地读出全部振幅。估计不是完美复制。

- **非线性复制假想。** 若允许任意非线性映射，形式上可以写出 `|psi> -> |psi>|psi>`，但标准量子力学的封闭演化不允许这种动力学。

## Evaluation settings

核心判据应覆盖任意未知纯态，而不是某个固定正交基。最小证明只需两个非正交态 `|psi>` 与 `|phi>`：如果同一个幺正复制器能同时复制它们，内积保持会给出

`<psi|phi> = <psi|phi>^2`.

这只允许 `<psi|phi>` 为 `0` 或 `1`。因此两个不同且非正交的态已经足以排除通用复制器。

另一个等价压力测试是叠加态。若复制器能复制 `|0>` 和 `|1>`，线性要求它对 `a|0>+b|1>` 输出 `a|00>+b|11>`；但真正的两份副本应是

`(a|0>+b|1>)(a|0>+b|1>) = a^2|00>+ab|01>+ab|10>+b^2|11>`.

两者一般不同。这直接暴露出复制任意态与叠加原理之间的冲突。

## Proof target

最终论证应陈述：不存在一个固定的线性量子演化，能够把任意未知纯态和固定空白态映射为两份同样的纯态。若允许复制一组候选态，那么这些候选态必须彼此正交或完全相同；任意非正交不同态不能同时被完美复制。

证明应同时展示两条直觉。第一，内积保持说明复制会把态间重叠平方化，从而破坏几何结构。第二，叠加原理说明“复制基向量”并不自动意味着“复制所有叠加态”，因为复制映射本质上不是线性的。

报告重点不是把 no-cloning theorem 说成某个技术禁令，而是说明它如何把“信息天然可复制”的经典直觉改写为量子信息论的基本结构限制：未知量子信息不是可任意读写的标签，而是受线性、内积与可区分性共同约束的物理态。
