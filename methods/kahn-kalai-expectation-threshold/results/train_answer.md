Fix a finite ground set $X$ with $|X|=n$, and let $\mu_p$ be the product measure that includes each element independently with probability $p$. For a nontrivial increasing property $\mathcal F\subseteq 2^X$ the map $p\mapsto\mu_p(\mathcal F)$ is continuous and strictly increasing, so there is a unique threshold $p_c(\mathcal F)$ with $\mu_{p_c}(\mathcal F)=1/2$; below it a random set almost never has the property, above it almost always. This number is the thing one wants, and it is brutally hard: every interesting threshold I know of — Hamilton cycles by Pósa rotations, perfect matchings by second-moment work, hypergraph matchings by the Johansson–Kahn–Vu machinery — cost somebody a long, property-specific argument, and none of those arguments speak to one another. What I want instead is a single computable quantity that predicts $p_c$ for every increasing family at once.

The naive candidate is staring at me. Call $\mathcal F$ *$p$-small* if it has a cheap cover: a family $\mathcal G$ with $\mathcal F\subseteq\langle\mathcal G\rangle=\bigcup_{S\in\mathcal G}\{T:T\supseteq S\}$ and cost $\sum_{S\in\mathcal G}p^{|S|}\le 1/2$, and define the *expectation threshold* $q(\mathcal F)$ to be the largest $p$ for which $\mathcal F$ is $p$-small. The name is exact, because $\sum_{S\in\mathcal G}p^{|S|}=\mathbb E\big[\,|\{S\in\mathcal G:S\subseteq X_p\}|\,\big]$ is the expected number of cover-sets a $p$-random set captures; by the union bound $\mu_p(\mathcal F)\le\mu_p(\langle\mathcal G\rangle)\le\sum_S p^{|S|}\le 1/2$, so $q(\mathcal F)\le p_c(\mathcal F)$ for free. The entire difficulty is the other direction: how far above its expectation threshold can a true threshold sit? It can sit a $\log$ factor up — for perfect matchings and Hamilton cycles in $G_{n,p}$ the cheap cover comes from local structures and gives $q\sim 1/n$, while the truth is $\log n/n$, the coupon-collector cost of killing every isolated or degree-one vertex — so a single $\log$ gap is forced and unavoidable. The question Kahn and Kalai pinned down is whether a single $\log$ is all that ever happens: is $p_c(\mathcal F)\le K\,q(\mathcal F)\log\ell(\mathcal F)$ for a universal $K$, where $\ell(\mathcal F)$ is the size of the largest minimal member (or $2$)?

The reason this resisted a frontal assault is that all the working machinery lives next door, on the *fractional* side. Relax the cover from a $\{0,1\}$ set system to a nonnegative weighting and one gets the fractional expectation threshold $q_f$, which is the value of a linear program; "not weakly $p$-small" then has a dual certificate — a spread probability measure supported on $\mathcal F$ — and that spread measure is exactly what the Alweiss–Lovett–Wu–Zhang robust-sunflower encoding argument eats. So Frankston–Kahn–Narayanan–Park got $p_c(\mathcal F)\le K\,q_f(\mathcal F)\log\ell(\mathcal F)$ by the chain not-fractionally-small $\xrightarrow{\text{LP duality}}$ spread measure $\xrightarrow{\text{ALWZ hitting}}$ random set wins. Drop to the integer condition $q$ and the first arrow vanishes: "not $p$-small" is an integer statement with no LP duality, so nothing hands me a spread measure. The community's expected detour — Talagrand's rounding conjecture $q(\mathcal F)\ge q_f(\mathcal F)/L$ — is itself a hard problem whose baby cases each needed their own paper, so I refuse it. The wall is precisely the missing spread hypothesis, and I will go around it by never asking for spread.

I propose to prove the Kahn–Kalai expectation-threshold conjecture directly by an incremental sampling dichotomy built on one new object, the **minimum fragment**. The whole proof is the contrapositive of a hypergraph reformulation: if $\mathcal H$ (the minimal elements of $\mathcal F$, an $\ell$-bounded hypergraph with $\langle\mathcal H\rangle=\mathcal F$) is *not* $p$-small, then a uniformly random set of size about $(Lp\log\ell)n$ lands in $\langle\mathcal H\rangle$ with high probability. The idea is to sample the random set $W$ in increments and, in lockstep, try to manufacture a cheap cover of $\mathcal H$ out of what the sampling reveals; since "not $p$-small" forbids a cheap cover, the manufacturing must usually fail, and I engineer failure to mean the sample already contains a member. Concretely, for an edge $S\in\mathcal H$ and the sampled $W$, define $T(S,W)$ to be the *smallest* set of the form $S'\setminus W$ with $S'\in\mathcal H$ and $S'\subseteq W\cup S$ — the tiniest leftover of any edge that sits inside the sampled region. It exists because $S$ itself qualifies. Where ALWZ/FKNP fix a chosen edge $\psi(Z)$ and have no automatic control on its remainder, taking the *minimum* buys a containment for free: if $W\cup T$ contains any edge $\hat S\in\mathcal H$ and $T\not\subseteq\hat S$, then $\hat S\setminus W$ is a strictly smaller fragment of $(S,W)$, contradicting minimality, so $T\subseteq\hat S$. That single fact is the load-bearing replacement for spread.

Each round of the construction splits $\mathcal H$ by fragment size. I peel off the *large*-fragment edges, those with $t(S,W)=|T(S,W)|\ge 0.9\ell$, into $\mathcal G(W)$ and cover them by their fragments $\mathcal U(W)=\{T(S,W):S\in\mathcal G(W)\}$ (each $T\subseteq S$, so this is a genuine cover); the *small*-fragment edges become the successor hypergraph $\mathcal H'(W)=\{T(S,W):S\notin\mathcal G(W)\}$, which is $0.9\ell$-bounded, so edges shrink by a constant factor each round. The reason the peeled cover is cheap is the minimum-fragment count, and it is where the freed containment pays off: to count pairs $(W,T)$ with $|T|=m$, first specify $Z=W\cup T$, whose size overshoots $w=Lpn$ by only the *small* $m$, costing a factor $(Lp)^{-m}$ via the binomial-ratio estimate $\binom{n}{w+m}\le\binom{n}{w}(n/w)^m$; then specify $T$ inside an arbitrary edge $\hat S\subseteq Z$, which by the containment must contain $T$, costing only $2^{|\hat S|}\le 2^\ell$. The weight $p^m$ cancels the $(Lp)^{-m}$, leaving a clean $L^{-m}2^\ell$ that sums over $m\ge 0.9\ell$ to $L^{-0.8\ell}$ once $L^{0.1}>2$ — no spread, no pathological-case split, the case analysis FKNP needed simply evaporates because $Z=W\cup T$ overshoots by $t$ rather than $|S|$. Iterating $\gamma=\Theta(\log\ell)$ rounds drives the edge bound below $1$, so the final hypergraph is $\emptyset$ or $\{\emptyset\}$, forcing an exhaustive dichotomy: either the accumulated sample $W=\bigcup_i W_i$ already contains a member (by the capture bookkeeping $(\bigcup_{j\le i}W_j)\cup S_i\in\langle\mathcal H\rangle$, propagated by induction), or every edge was peeled and the union of peeled covers $\mathcal U$ covers $\mathcal H$. The second is forbidden because $\mathcal H$ is not $p$-small, so any cover has $\sum_U p^{|U|}>1/2$, and Markov on the total expected cost kills its probability. The only subtlety is that a constant sampling rate makes the late rounds (where $\ell_i$ is a constant and the per-round bound is weak) too expensive over $\Theta(\log\ell)$ steps, so I boost the rate to $L\sqrt{\log\ell}$ only in the last $O(\sqrt{\log\ell})$ rounds; that costs just a constant factor in the final sample size, keeping $|W|=\Theta(p\log\ell)n$, and drives the exceptional probability to $(\log\ell)^{-c}$. Converting fixed-size sampling back to $\mu_p$ by concentration of $|X_{p'}|$ then yields $p_c(\mathcal F)\le K\,q(\mathcal F)\log\ell(\mathcal F)$.

What follows is the precise statement and full proof.

Let $\ell(\mathcal F)=\max\big(2,\ \text{size of the largest minimal member of }\mathcal F\big)$.

**Theorem (Kahn–Kalai expectation-threshold).** There is a universal constant $K$ such that for every finite set $X$ and every nontrivial increasing $\mathcal F\subseteq 2^X$,
$$p_c(\mathcal F)\ \le\ K\,q(\mathcal F)\,\log\ell(\mathcal F).$$

This follows from the hypergraph reformulation (all logarithms base $2$; floors/ceilings suppressed; $\langle\mathcal H\rangle=\bigcup_{S\in\mathcal H}\{T:T\supseteq S\}$; $\mathcal H$ is *$\ell$-bounded* if every edge has size $\le\ell$):

**Theorem (reformulation).** Let $\ell\ge 2$. There is a universal constant $L$ such that for any nonempty $\ell$-bounded hypergraph $\mathcal H$ on $X$ that is **not** $p$-small, a uniformly random $\big((Lp\log\ell)\,|X|\big)$-element subset of $X$ belongs to $\langle\mathcal H\rangle$ with probability $1-o_{\ell\to\infty}(1)$. (Quantitatively the exceptional probability is $(\log\ell)^{-c}$ for some $c>0$.)

*Proof of the reformulation.* Write $n=|X|$, $w=Lpn$, and let $W\in\binom{X}{w}$.

**Minimum fragment.** For $S\in\mathcal H$ and $W\subseteq X$, let $T=T(S,W)$ be the smallest set (possibly empty) of the form $S'\setminus W$ with $S'\in\mathcal H$ and $S'\subseteq W\cup S$; set $t=|T|$. (It exists: $S$ itself works.) Define
$$\mathcal G(W)=\{S\in\mathcal H: t(S,W)\ge 0.9\ell\},\quad \mathcal U(W)=\{T(S,W):S\in\mathcal G(W)\},\quad \mathcal H'(W)=\{T(S,W):S\in\mathcal H\setminus\mathcal G(W)\}.$$
Each $T(S,W)\subseteq S$, so $\mathcal U(W)$ covers $\mathcal G(W)$ and $\mathcal H\setminus\mathcal G(W)\subseteq\langle\mathcal H'(W)\rangle$; and $\mathcal H'(W)$ is $0.9\ell$-bounded.

**Cost Lemma.** For $W$ uniform from $\binom{X}{w}$,
$$\mathbb E\Big[\sum_{U\in\mathcal U(W)}p^{|U|}\Big]<L^{-0.8\ell}\quad(\text{for }L\text{ a large enough absolute constant}),$$
equivalently $\sum_{W\in\binom{X}{w}}\sum_{U\in\mathcal U(W)}p^{|U|}<\binom{n}{w}L^{-0.8\ell}$.

*Proof.* For $m\ge 0.9\ell$ let $\mathcal U_m(W)=\{T(S,W):t(S,W)=m\}$; every $U\in\mathcal U_m(W)$ has $|U|=m$, so
$$\sum_{W}\sum_{U\in\mathcal U_m(W)}p^{|U|}=p^m\cdot\big|\{(W,T(S,W)):W\in\tbinom{X}{w},\ S\in\mathcal H,\ t(S,W)=m\}\big|.$$
Count the pairs $(W,T)$ in two steps.

- **Step 1 (specify $Z=W\cup T$).** $W,T$ are disjoint and $|T|=m$, so $|Z|=w+m$ and
$$\binom{n}{w+m}=\binom{n}{w}\prod_{j=0}^{m-1}\frac{n-w-j}{w+j+1}\le\binom{n}{w}\Big(\frac{n}{w}\Big)^{m}=\binom{n}{w}(Lp)^{-m},$$
since each factor is $\le n/w=1/(Lp)$ (numerator decreasing, denominator increasing).
- **Step 2 (specify $T$ inside $Z$).** $Z=W\cup T$ contains an edge of $\mathcal H$ (the $S'$ realizing $T$); choose an edge $\hat S\subseteq Z$ as a function of $Z$ only (free given $Z$). Minimality forces $T\subseteq\hat S$: otherwise $\hat S\setminus W\subsetneq T$ would be a strictly smaller fragment of $(S,W)$. So $T$ is one of at most $2^{|\hat S|}\le 2^\ell$ subsets of $\hat S$.

Then $W=Z\setminus T$ is determined, so the number of pairs is $\le\binom{n}{w}(Lp)^{-m}2^\ell$, and the size-$m$ contribution is $\le p^m\binom{n}{w}(Lp)^{-m}2^\ell=\binom{n}{w}L^{-m}2^\ell$. Summing,
$$\sum_{m\ge 0.9\ell}\binom{n}{w}L^{-m}2^\ell=\binom{n}{w}\,2^\ell\,\frac{L^{-0.9\ell}}{1-1/L}\le\binom{n}{w}L^{-0.8\ell},$$
the last inequality holding once $L^{0.1}>2$ (then $2^\ell L^{-0.9\ell}\le L^{-0.8\ell}$ with room to spare). Dividing by $\binom{n}{w}$ gives the lemma. $\qquad\blacksquare$

*(Remark.* Specifying $Z=W\cup T$ instead of $Z=W\cup S$, with $T$ the minimum fragment, is what makes Step 2's containment automatic and removes the "pathological/non-pathological" case split of the earlier sunflower-style arguments.)*

**Iteration.** Let $\ell_i=0.9^i\ell$ and $\gamma=\lfloor\log_{0.9}(1/\ell)\rfloor+1$, so $0.9\le\ell_\gamma<1$. Set $X_0=X$; for $i=1,\dots,\gamma$ let $W_i\in\binom{X_{i-1}}{w_i}$, $X_i=X_{i-1}\setminus W_i$, $w_i=L_ipn$ with
$$L_i=\begin{cases}L & i<\gamma-\sqrt{\log_{0.9}(1/\ell)},\\ L\sqrt{\log\ell} & \gamma-\sqrt{\log_{0.9}(1/\ell)}\le i\le\gamma.\end{cases}$$
Let $\mathcal H_0=\mathcal H$ and $\mathcal H_i=\mathcal H_{i-1}'(W_i)$, with $\mathcal G_i=\mathcal G_i(W_i)$, $\mathcal U_i=\mathcal U_i(W_i)$. Then $\mathcal H_i$ is $\ell_i$-bounded, and $W:=\bigcup_{i=1}^\gamma W_i$ is a uniformly random $(CLp\log\ell)n$-subset for an absolute $C$.

*Capture (by induction on $i$):* for all $S_i\in\mathcal H_i$, $(\bigcup_{j\le i}W_j)\cup S_i\in\langle\mathcal H\rangle$. Indeed $S_i=T(S_{i-1},W_i)=S_{i-1}'\setminus W_i$ for some $S_{i-1}'\in\mathcal H_{i-1}$, so $(\bigcup_{j\le i}W_j)\cup S_i\supseteq(\bigcup_{j\le i-1}W_j)\cup S_{i-1}'\in\langle\mathcal H\rangle$.

*Dichotomy:* since $\ell_\gamma<1$, $\mathcal H_\gamma=\emptyset$ or $\mathcal H_\gamma=\{\emptyset\}$.
- If $\mathcal H_\gamma=\emptyset$: tracing $S=S_0,S_1,\dots$ ($S_i=T(S_{i-1},W_i)$), each $S$ is peeled at some step $j<\gamma$ (i.e. $S_j\in\mathcal G_j$, covered by $\mathcal U_j$), so $\mathcal U:=\bigcup_{i\le\gamma}\mathcal U_i$ covers $\mathcal H$.
- If $\mathcal H_\gamma=\{\emptyset\}$: capture with $S_\gamma=\emptyset$ gives $W=\bigcup_i W_i\in\langle\mathcal H\rangle$.

Hence $\Pr(W\in\langle\mathcal H\rangle)+\Pr(\mathcal E)\ge 1$, where $\mathcal E=\{\mathcal U\text{ covers }\mathcal H\}$.

**Bounding $\Pr(\mathcal E)$.** Because $\mathcal H$ is *not* $p$-small, any cover has $\sum_U p^{|U|}>1/2$; so $\mathcal E\Rightarrow\sum_{U\in\mathcal U}p^{|U|}>1/2$. By the Cost Lemma applied at each $(\ell_i,L_i)$,
$$\mathbb E\Big[\sum_{U\in\mathcal U}p^{|U|}\Big]<\sum_{i\le\gamma}L_i^{-0.8\ell_i}.$$
There is $c>0$ with $\ell_i>\exp(c\sqrt{\log\ell})$ for $i<\gamma-\sqrt{\log_{0.9}(1/\ell)}$, so the early terms sum to $\le 2L^{-0.8\exp(c\sqrt{\log\ell})}$. The late range has $O(\sqrt{\log\ell})$ terms with $\ell_i\ge 0.9$, summing to $O\big((L\sqrt{\log\ell})^{-c'}\big)$. Thus
$$\mathbb E\Big[\sum_{U\in\mathcal U}p^{|U|}\Big]<2L^{-0.8\exp(c\sqrt{\log\ell})}+O\big((L\sqrt{\log\ell})^{-c'}\big)=(\log\ell)^{-c''},$$
and by Markov $\Pr(\mathcal E)\le 2\,\mathbb E[\sum_U p^{|U|}]=(\log\ell)^{-c''}=o_{\ell\to\infty}(1)$. Therefore $\Pr(W\in\langle\mathcal H\rangle)\ge 1-(\log\ell)^{-c''}$. $\qquad\blacksquare$

*Deriving the main theorem from the reformulation.* Let $\mathcal F$ be nontrivial increasing and $q>q(\mathcal F)$. Let $\mathcal H$ be the minimal elements of $\mathcal F$ (so $\langle\mathcal H\rangle=\mathcal F$); $\mathcal H$ is $\ell(\mathcal F)$-bounded and not $q$-small. Pick absolute $C$ so that with $\ell=C\ell(\mathcal F)$ the exceptional probability above is $<1/4$. Set $m=(Lq\log\ell)n$, $p'=2Lq\log\ell$, and choose $K$ so $p:=Kq\log\ell(\mathcal F)\ge p'$. The bound is vacuous if $p'>1$, so assume $p'\le1$. Then
$$\Pr(X_{p'}\in\langle\mathcal H\rangle)\ \ge\ \Pr(X_m\in\langle\mathcal H\rangle)\,\Pr(|X_{p'}|\ge m)\ \ge\ \tfrac34\,\Pr(|X_{p'}|\ge m)\ >\ \tfrac12,$$
where the last step is concentration of $|X_{p'}|\sim\mathrm{Bin}(n,p')$ about its mean $p'n=2m$, valid because $m$ is large: $\mathcal H$ not $q$-small implies $nq>1/2$ (the singleton cover $\{\{x\}\}$ has cost $nq$), so $m=(L\log\ell)\,qn>(L\log\ell)/2$. As $\mathcal F$ is increasing and $p\ge p'$, $\Pr(X_p\in\mathcal F)\ge\Pr(X_{p'}\in\mathcal F)>1/2$, i.e. $p_c(\mathcal F)\le Kq\log\ell(\mathcal F)$. Letting $q\downarrow q(\mathcal F)$ gives $p_c(\mathcal F)\le K\,q(\mathcal F)\log\ell(\mathcal F)$. $\qquad\blacksquare$
