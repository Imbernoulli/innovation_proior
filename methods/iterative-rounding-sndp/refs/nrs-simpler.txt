Abstract
We consider the S URVIVABLE N ETWORK D ESIGN P ROBLEM (SNDP) and the S YMMETRIC T RAVELING S ALESMAN P ROBLEM (STSP). We give simpler proofs of existence of a 12 -edge and 1-edge in

any extreme point of the natural LP relaxations for SNDP and STSP respectively. We formulate a common generalization of both problems and show our results by a new counting argument. We also obtain a
simpler proof of existence of a 21 -edge in any extreme point of the set-pair LP relaxation for the element
connectivity S URVIVABLE N ETWORK D ESIGN P ROBLEM (SNDPelt ).
Keywords: Linear Programming Relaxations, Traveling Salesman, Survivable Network Design, Approximation Algorithms.

Element Connectivity SNDP

In this section, we consider the element connectivity survivable network design problem (SNDPelt ). In this
problem, we are given an undirected graph G = (V, E) with edge-costs c : E → R+ , a set U ⊆ V of
terminals, and connectivity requirements ruv for all undirected pairs u, v ∈ U of terminals. Vertices in
V \ U are called non-terminals. The edges and non-terminals of the graph are called elements. The goal in
SNDPelt is to find the minimum cost subgraph that contains at least ruv element-disjoint paths between u and
v for every u, v ∈ U . Fleischer, Jain and Williamson [5] used iterative rounding to obtain a 2-approximation
algorithm for this problem. Fleischer et al. [5] showed that SNDPelt can be formulated as a suitable integer
program (defined formally below), such that any extreme-point solution to its LP-relaxation contains an
edge with solution value at least half. We give a short proof of this result using a new counting argument
generalizing the results in the previous section.
A set-pair is an ordered tuple (S, S 0 ) where S, S 0 ⊆ V . Let F denote some family of set-pairs. A
two-set function f : F → Z+ is called weakly two-supermodular if for any (S, S 0 ) and (T, T 0 ) ∈ F, at least
one of the following holds:
1. (S ∩ T, S 0 ∪ T 0 ) and (S ∪ T, S 0 ∩ T 0 ) ∈ F, and we have
f (S ∩ T, S 0 ∪ T 0 ) + f (S ∪ T, S 0 ∩ T 0 ) ≥ f (S, S 0 ) + f (T, T 0 ).
2. (S ∩ T 0 , S 0 ∪ T ) and (S ∪ T 0 , S 0 ∩ T ) ∈ F, and we have
f (S ∩ T 0 , S 0 ∪ T ) + f (S ∪ T 0 , S 0 ∩ T ) ≥ f (S, S 0 ) + f (T, T 0 ).
For any set-pair (S, S 0 ) let E(S, S 0 ) = {e = (u, v) ∈ E | u ∈ S, v ∈ S 0 } denote the edges with one
end-point in S and the other in S 0 . For any assignment x : E → R+ and set-pair (S, S 0 ), we abbreviate
x(E(S, S 0 )) by just x(S, S 0 ). The LP-relaxation for SNDPelt considered in [5] is the following.

We first introduce some definitions from [5] that are required for the proof. Define a partial order on
set-pairs where (S, S 0 ) ≤ (T, T 0 ) iff S ⊆ T and T 0 ⊆ S 0 ; in this case we say that (S, S 0 ) is smaller than
(T, T 0 ). We also say that (S, S 0 ) and (T, T 0 ) are comparable if either (S, S 0 ) ≤ (T, T 0 ) or (T, T 0 ) ≤ (S, S 0 );
otherwise they are incomparable.
Set-pairs (S, S 0 ) and (T, T 0 ) are said to pair-cross iff none of the following holds:
C1. S ⊆ T and T 0 ⊆ S 0 , i.e. (S, S 0 ) ≤ (T, T 0 ).
C2. S 0 ⊆ T 0 and T ⊆ S, i.e. (S, S 0 ) ≥ (T, T 0 ).
C3. S ⊆ T 0 and T ⊆ S 0 .
A collection of set-pairs is pair-laminar if no two of them pair-cross.
The following result appears as Corollary 4.6 and Lemma 4.7 in Fleischer et al. [5].
Lemma 3.2 ([5]) Let x be a basic feasible solution to (LPelt ), such that 0 < xe < 1 for all edges e ∈ E.
Then, there exists a pair-laminar family L of set-pairs such that:
1. x is the unique solution to: {x(S, S 0 ) = f (S, S 0 ), ∀(S, S 0 ) ∈ L}.
2. The vectors χE(S,S 0 ) for (S, S 0 ) ∈ L are linearly independent.
3. The poset induced by ≤ on L is a forest; i.e. for any (X, X 0 ), (Y, Y 0 ), (Z, Z 0 ) ∈ L with (X, X 0 ) ≤
(Y, Y 0 ) and (X, X 0 ) ≤ (Z, Z 0 ), the set-pairs (Y, Y 0 ) and (Z, Z 0 ) are comparable.
4. |E| = |L|.
We also refer to set-pairs in L as nodes. For any node (S, S 0 ) ∈ L, its parent is the smallest node (T, T 0 ) ∈
L \ {(S, S 0 )} that is larger than (S, S 0 ) (i.e. satisfying (T, T 0 ) ≥ (S, S 0 )); in this case (S, S 0 ) is called a
child of (T, T 0 ). If there is no node in L \ {(S, S 0 )} that is larger than (S, S 0 ), then node (S, S 0 ) is called a
maximal node. Node (R, R0 ) ∈ L is a descendent of (S, S 0 ) ∈ L iff (R, R0 ) ≤ (S, S 0 ).

the token of any edge (u, v) ∈ E(P, P 0 ) given by rule III is
unused. Let u ∈ P and v ∈ P 0 . For any
T
descendent (T, T 0 ) of (P, P 0 ), we have T 0 ⊇ P 0 3 v; so T 0 {u, v} 6= ∅. Any node (Q, Q0 ) ∈ L that is
not a descendent of P, P 0 ) is incomparable to (P, P 0 ), and we have Q0 ⊇ P 3 u. Thus {u, v} ∩ S 0 6= ∅ for
all (S, S 0 ) ∈ L, i.e. rule III token of edge (u, v) is unassigned. Thus there is a positive amount of unused
tokens. However this implies that |E| > |L| which contradicts Lemma 3.2.
This completes the proof of Theorem 3.1.
