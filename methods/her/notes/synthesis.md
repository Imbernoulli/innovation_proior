# HER synthesis notes (pre-Phase-2)

## Pain point
- Sparse binary reward r(s,a)=-[not at goal]: agent essentially never reaches goal by random exploration in large state spaces -> never sees reward != -1 -> value learning has nothing to bootstrap from. Exploration tricks (VIME, count-based, bootstrapped DQN) don't help because the issue is not state-diversity but that the *reward signal* is absent, not that states are unvisited.
- Reward shaping (Ng et al. 1999 potential-based) gives dense informative reward but: requires domain knowledge/RL expertise; for hard manipulation (Popov 2017) needs 5 carefully weighted terms; shaped rewards can penalize useful exploration and induce degenerate behavior (agent learns not to touch object). Shaped reward is a compromise on the metric we actually care about (binary success).

## Core insight
- A trajectory s_0..s_T that failed to reach goal g DID reach s_T (and every intermediate state). So it is a *successful* demonstration of reaching s_T. If we pretend the goal had been g' = m(s_T), then the same transitions get reward != -1 at the end -> a valid, positive, dense-enough learning signal.
- Requires goal-conditioned value function Q(s,a,g) (UVFA, Schaul 2015a): single net approximates value for all goals, generalizes across goals. So relabeling a transition to a different goal is *legal* input to the same Q.
- Off-policy required: relabeling changes the goal, hence changes the reward and effectively the policy that the data is "about"; only off-policy algos (DQN/DDPG/NAF/SDQN) can learn from data not generated by the current target policy/goal.

## Formalism
- Multi-goal RL: goal space G, each g has predicate f_g: S->{0,1}; reward r_g(s,a)=-[f_g(s')=0]. Mapping m: S->G with f_{m(s)}(s)=1 (every state satisfies some goal). For S=G, f_g(s)=[s=g], m=identity. For object-position goals G=R^3, f_g(s)=[|g - s_object|<=eps], m(s)=s_object.
- Goal-conditioned Bellman (UVFA): Q*(s,a,g)=E[r_g(s,a) + gamma max_a' Q*(s',a',g)]. DDPG actor pi(s||g), critic Q(s,a,g). Target y = r + gamma Q'(s', pi'(s'||g), g). Note g held fixed across the Bellman backup — relabel BOTH the stored transition's g AND recompute r consistently.

## Algorithm (Alg 1)
- Collect episode with original goal g, behavioral policy a_t = pi_b(s_t || g).
- For each transition store (s_t||g, a_t, r_t, s_{t+1}||g) with r_t = r(s_t,a_t,g) [standard ER].
- Sample additional goals G := S(current episode) via strategy; for each g' store (s_t||g', a_t, r'=r(s_t,a_t,g'), s_{t+1}||g') [HER].
- Train off-policy on minibatch from buffer.

## Strategies S
- final: g' = m(s_T) (final achieved state of episode). simplest.
- future: replay with k random states observed *after* t in the same episode. BEST (k=4 or 8). Reason: the most valuable relabel goals are ones reachable in the near future from the current transition — gives signal for "how to get from here to slightly ahead", which is exactly the local credit needed.
- episode: k random states from same episode (not necessarily after t).
- random: k random states from anywhere in training so far.
- k = ratio of HER to normal data. future_p = 1 - 1/(1+k). k too high (>8) degrades: fraction of *real* (unrelabeled, original-goal) data becomes too small -> distribution mismatch, the buffer no longer reflects the actual goal distribution we're evaluated on.

## Why future beats final
- final only gives the endpoint as goal; future gives a spectrum of horizons. Sliding task needs future; final/episode insufficient there. future also automatically forms implicit curriculum: early on, achieved states are near start (easy goals); as policy improves, achieved states approach real goals.

## Implicit curriculum
- relabel goals = states the agent actually reaches; these shift from easy (random-policy reachable) to hard as policy improves. No control over initial-state distribution needed (unlike explicit curriculum).

## DDPG / impl details (Appendix A + baselines code)
- actor & critic: 3 hidden layers, 64 units, ReLU; actor output tanh rescaled to [-5cm,5cm]; add square of preactivations to actor cost to prevent tanh saturation.
- gamma=0.98; clip critic targets to [-1/(1-gamma), 0] (value bounds for negative reward in [-1,0]).
- Adam lr 1e-3; target nets polyak 0.95 (updated each cycle); 200 epochs x 50 cycles x (16 episodes rollout + 40 optim steps), batch 128, buffer 1e6.
- Input normalization: zero-mean unit-std clip [-5,5], stats over all observations.
- Exploration: 20% uniform random action, else gaussian noise std 5% of range.
- bit-flipping toy: S={0,1}^n, action flips i-th bit, r=-[s!=g]. DQN fails n>40; DQN+HER solves up to 50. DQN without HER only n<=13.

## Code structure (baselines her_sampler) -> drives final code
- make_sample_her_transitions(strategy, k, reward_fun): future_p = 1-1/(1+k).
- sample: pick random episode idxs + t_samples; for a future_p fraction, pick future_t in (t, T], set transitions['g'] = ag at future_t (ag = achieved goal = m(state)); recompute r = reward_fun(ag_2, g). 'ag' = achieved goal trajectory, 'ag_2' = achieved goal at next state.
- replay buffer stores full episodes {o, ag, g, u} as (size x T x dim); sampling happens at minibatch time so relabeling is done lazily.

## Load-bearing ancestors
- UVFA (Schaul 2015a): goal-conditioned single-net value approximator; the substrate that makes relabeling legal.
- DQN (Mnih 2015): off-policy value learning, replay buffer, target net.
- DDPG (Lillicrap 2015): off-policy continuous control, actor-critic, target policy.
- Experience replay (Lin 1992): store and resample transitions; off-policy reuse.
- Reward shaping (Ng 1999): the alternative we reject as domain-specific.
- Prioritized ER (Schaul 2015b): orthogonal, combinable.
- Horde / Sutton 2011, Foster&Dayan 2002: multiple value functions / GVFs lineage.
- curriculum (Bengio 2009, Elman 1993): HER as implicit curriculum.
