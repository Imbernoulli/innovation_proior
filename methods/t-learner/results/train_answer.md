We have a table of units — for each one a covariate vector $X \in \mathbb{R}^d$, a binary treatment flag $W \in \{0,1\}$, and a single observed outcome $Y^{\mathrm{obs}} = Y(W)$ — and what we want, for a brand-new unit with covariates $x$, is how much the treatment would move its outcome, not on average over the population but specifically for units that look like $x$. The target is the Conditional Average Treatment Effect $\tau(x) = \mathbb{E}[Y(1) - Y(0) \mid X = x]$, the difference of the two potential outcomes' conditional means. Two obstructions make this harder than ordinary prediction. The first is fundamental: for any single unit we only ever observe the one potential outcome matching the treatment it received, so the per-unit effect $D = Y(1) - Y(0)$ is never in the data — and it is genuinely unidentifiable. A clean counterexample settles this: with $X \sim \mathrm{Unif}[0,1]$, fair-coin $W$, and Rademacher control outcome $P(Y(0){=}1)=P(Y(0){=}{-}1)=0.5$, the world $Y(1)=Y(0)$ (all unit effects zero) and the world $Y(1)=-Y(0)$ (all unit effects $\pm 2$) induce exactly the same law on the observable triple $(Y^{\mathrm{obs}}, X, W)$, so no estimator can tell them apart. But $\tau(x)=0$ in both, so the conditional mean survives. And aiming at $\tau$ is not a consolation prize even if one secretly wants $D$: for an estimator $\hat\tau$ trained independently of a fresh target unit, $\mathbb{E}[(D_i - \hat\tau(x))^2 \mid X_i = x] = \mathbb{E}[(D_i - \tau(x))^2 \mid X_i = x] + \mathbb{E}[(\tau(x) - \hat\tau(x))^2]$, the cross term vanishing because the target residual has conditional mean zero and is independent of the training-based fit, so the first term is irreducible and minimizing unit-effect MSE is identical to minimizing CATE MSE. The second obstruction is confounding: in observational data $W \sim \mathrm{Bern}(e(X))$ with propensity $e(x) = P(W{=}1 \mid X{=}x)$, so treated and control units have systematically different covariates and raw group-mean differences confound treatment with selection.

The existing options each fall short. A single pooled model that folds $W$ in as just another feature and reads off $\hat\mu(x,1) - \hat\mu(x,0)$ gives $W$ no special status, so a flexible learner is free to decide $W$ barely matters — a forest in the typical observational regime, where the covariates strongly predict the outcome and $W$ only weakly does, can run all the way to its leaves without ever splitting on $W$, and on every such path the predicted effect is exactly zero, systematically attenuating a real effect toward null. Transformed-outcome regression builds a single reweighted response $Y^* = Y^{\mathrm{obs}}(W - e(X))/(e(X)(1-e(X)))$ whose conditional mean is $\tau$ and regresses it directly, but dividing by $e(1-e)$ makes it explode in variance exactly where overlap is weak. Causal trees commit to one honest tree-structured partition, elegant and interval-valid but tied to a single model class for the whole problem rather than letting the best-fit learner be chosen per arm. None of these both keeps maximal modeling flexibility and avoids the attenuation collapse.

I propose the T-learner ("T" for two). The lever is an identification fact the assumptions hand us. Under ignorability $(\varepsilon(0), \varepsilon(1)) \perp W \mid X$ (within a covariate cell, treated and control units are exchangeable) and overlap $0 < e_{\min} < e(x) < e_{\max} < 1$ (every cell contains some of both arms), the observed within-arm conditional mean equals the potential-outcome surface: on the event $W{=}1$ we have $Y^{\mathrm{obs}} = Y(1)$, and conditioning further on $W{=}1$ adds nothing once $X$ is fixed, so $\mathbb{E}[Y^{\mathrm{obs}} \mid X{=}x, W{=}1] = \mu_1(x)$, and identically $\mathbb{E}[Y^{\mathrm{obs}} \mid X{=}x, W{=}0] = \mu_0(x)$. No counterfactual appears inside either regression — each arm uses only outcomes it actually observed. So the most direct estimator imaginable works: fit any regressor $\hat\mu_0$ on the control rows $\{(X_i, Y_i): W_i = 0\}$, fit any (possibly different) regressor $\hat\mu_1$ on the treated rows $\{(X_i, Y_i): W_i = 1\}$, and difference them,
$$\hat\tau(x) = \hat\mu_1(x) - \hat\mu_0(x).$$
The causal content lives entirely in the act of splitting by arm and subtracting; the estimation of each surface is delegated to whatever supervised method — trees, forests, boosting, OLS, $k$-NN, a net — best fits that arm's data. This is what makes it a meta-algorithm: it converts an off-the-shelf learner into a CATE estimator. Splitting the arms is the deliberate choice that beats pooling here, because the two surfaces are now estimated in isolation and the attenuation-to-zero collapse cannot happen merely because $W$ lost a greedy split competition. The price is the mirror image: by refusing to pool, the method cannot borrow strength across arms to capture structure both surfaces share, and it pays the full sampling cost in each arm separately — a shared discontinuity must be discovered twice, and if one arm is small its surface artifacts do not cancel in the difference, producing spurious heterogeneity in $\tau$. This is most painful under unbalanced designs, where one arm dwarfs the other. The practical corollary is that base-learner complexity must be matched to each arm's own sample size, and the freedom to use a different learner per arm is not a frill but the right response to arms differing in size and surface complexity.

What makes the method trustworthy is a sharp rate guarantee that also exposes its ceiling. Place the data-generating distribution in $S(a_\mu, a_\tau)$: ignorability holds, each response-surface family is in the minimax-regression class $S(a_\mu)$ (worst-case EMSE $\le C N^{-a_\mu}$), and the effect-imputation families are in $S(a_\tau)$. With the minimax base learner in each arm, writing $n$ for the number of treated and $m$ for the number of control,
$$\mathrm{EMSE}(\hat\tau) \le 2C\!\left[\frac{1-e_{\min}}{1-e_{\max}} + \frac{e_{\max}}{e_{\min}}\right]\!\left(n^{-a_\mu} + m^{-a_\mu}\right).$$
The derivation has four load-bearing steps. First, the algebraic split: since $\hat\tau - \tau = (\hat\mu_1 - \mu_1) - (\hat\mu_0 - \mu_0)$, the inequality $(a-b)^2 \le 2a^2 + 2b^2$ gives $(\hat\tau(X) - \tau(X))^2 \le 2(\hat\mu_1(X) - \mu_1(X))^2 + 2(\hat\mu_0(X) - \mu_0(X))^2$, so defining $A$ and $B$ as the treated- and control-surface EMSEs, $\mathrm{EMSE}(\hat\tau) \le 2A + 2B$. Second, recovering independence: to state a rate in terms of fixed $n$ and $m$ one wants to condition on the treated count, but conditioning on $\sum_i W_i = n$ couples the rows; conditioning instead on the entire assignment vector $W = w$ fixes which rows are treated, and then the treated tuples are i.i.d. from $P_1 = \mathrm{law}(X,Y \mid W{=}1)$ — independence follows because the original rows are mutually independent so the conditional probability factors across treated indices, and identical distribution follows because conditioning on all of $W$ reduces to conditioning on $W_i{=}1$ for the single row, the same law for every treated index. Third, $\hat\mu_1$ depends only on those $n$ i.i.d. treated rows, so its EMSE under $P_1$'s covariate law is $O(n^{-a_\mu})$. Fourth, a change of measure converts that error from $P_1$'s tilted covariate law to the target population marginal, and this is exactly where overlap pays off: since $\mathbb{E}[g(X)\mid W{=}1] = \mathbb{E}[g(X)e(X)]/\mathbb{E}[W]$ with both $e(X)$ and $\mathbb{E}[W]=\mathbb{E}[e(X)]$ pinned between $e_{\min}$ and $e_{\max}$,
$$\frac{e_{\min}}{e_{\max}}\,\mathbb{E}[g(X)] \le \mathbb{E}[g(X)\mid W{=}1] \le \frac{e_{\max}}{e_{\min}}\,\mathbb{E}[g(X)],$$
with the $1-e$ analogue for the control arm. Applying this with $g = (\hat\mu_1 - \mu_1)^2$ gives $A \le (e_{\max}/e_{\min}) C n^{-a_\mu}$ and symmetrically $B \le ((1-e_{\min})/(1-e_{\max})) C m^{-a_\mu}$, which assemble into the displayed bound. The overlap constants blow up as $e \to 0$ or $1$ — the price of confounding, paid precisely as the covariate tilt becomes extreme. As a concrete instance, for $L$-Lipschitz surfaces on $\mathrm{Unif}([0,1]^d)$ one has $a_\mu = 2/(2+d)$ via the $k$-NN bias-variance split $\mathbb{E}\|\hat\mu - \mu\|^2 \le \sigma^2/k + cL^2(k/N)^{2/d}$ minimized at $k \sim N^{2/(2+d)}$; choosing $k$-NN base learners with $k_1 \sim n^{2/(2+d)}$ and $k_0 \sim m^{2/(2+d)}$ makes the T-learner attain the minimax-optimal $O(n^{-2/(2+d)} + m^{-2/(2+d)})$. The ceiling is the honest part of the guarantee: the rate is the response-surface rate $a_\mu$, never the possibly faster effect rate $a_\tau$. When both $\mu_0$ and $\mu_1$ are wildly complicated but differ by a constant or a linear function, $\tau$ is simple yet the T-learner still estimates each surface at the slow rate and differences them, with no general mechanism to notice the difference is simpler than the parts — because the two arms never talk, that blindness is structural, and it is most visible when one arm is much larger than the other.

```python
import numpy as np
from sklearn.base import clone
from sklearn.ensemble import GradientBoostingRegressor


def make_base_regressor(seed):
    return GradientBoostingRegressor(random_state=seed)


class CATEEstimator:
    """T-learner: two outcome models, CATE = mu1_hat - mu0_hat."""

    def __init__(self, base_regressor=None):
        self._base_regressor = (
            make_base_regressor(seed=42) if base_regressor is None else base_regressor
        )
        self._models = {}

    def fit(self, X, T, Y):
        X, T, Y = np.asarray(X), np.asarray(T), np.asarray(Y)
        mask0 = T == 0                      # control rows -> mu0
        mask1 = T == 1                      # treated rows -> mu1
        self._models[0] = clone(self._base_regressor)
        self._models[1] = clone(self._base_regressor)
        self._models[0].fit(X[mask0], Y[mask0])
        self._models[1].fit(X[mask1], Y[mask1])
        return self

    def predict(self, X):
        X = np.asarray(X)
        return self._models[1].predict(X) - self._models[0].predict(X)
```
