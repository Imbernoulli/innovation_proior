Convolutional neural networks are powerful because they combine three properties: filters that are localized to a small spatial support, weight sharing that makes the parameter count independent of input size, and compositional pooling that builds multi-scale features. These properties are easy to realize on regular grids like images or audio, where shifting a filter by one pixel is well defined. But many important datasets are not grids. Social networks, regulatory networks, molecular graphs, and k-nearest-neighbor graphs of words or embeddings all have irregular structure: vertices have different degrees, and there is no canonical way to order the neighbors of one vertex to match those of another. The core challenge is therefore to define a convolutional filter on an arbitrary graph that remains localized, cheap to learn, and cheap to evaluate, without assuming a grid.

The natural starting point is spectral graph theory. The normalized graph Laplacian L = I − D^{−1/2} W D^{−1/2} is real, symmetric, and positive semidefinite, so it diagonalizes as L = U Λ U^T. The columns of U can be viewed as Fourier modes and the eigenvalues Λ as frequencies, generalizing the discrete Fourier transform. A filter can then be defined in the Fourier domain as y = U g_θ(Λ) U^T x, which is equivalent to y = g_θ(L) x. The most flexible choice, g_θ(Λ) = diag(θ) with θ ∈ R^n, is unsatisfying: it requires an O(n³) eigendecomposition and O(n²) dense multiplications by U, it has O(n) parameters that grow with the graph, and a generic spectral multiplier is not localized in the vertex domain. What is needed is a restricted family of spectral filters that are provably localized, have a fixed small number of parameters, and can be applied without ever forming U.

The method is ChebNet. It parametrizes the filter as a truncated Chebyshev expansion in the rescaled Laplacian. Because L^k has zero entries beyond k-hop neighborhoods, any degree-(K−1) polynomial of L is exactly (K−1)-hop localized and has only K coefficients. ChebNet writes that polynomial in the Chebyshev basis, which is well conditioned and generated by a stable three-term recurrence. The eigenvalues of the normalized Laplacian are bounded by 2, so rescaling λ ↦ 2λ/λ_max − 1 with λ_max = 2 gives L̃ = L − I, whose spectrum lies in [−1, 1]. The recurrence x̄_0 = x, x̄_1 = L̃ x, x̄_k = 2 L̃ x̄_{k−1} − x̄_{k−2} builds the K basis vectors at a cost of O(K|E|) sparse matrix–vector products, with no eigendecomposition. The filter output is then y = Σ_{k=0}^{K−1} θ_k x̄_k. A full layer maps F_in input feature maps to F_out output maps by learning F_in × F_out coefficient vectors θ, all shared across vertices, giving weight sharing and stationarity on the graph. For pooling, the graph is coarsened by a greedy multilevel matching inspired by Graclus, roughly halving the vertex count per level, and the hierarchy is embedded into a balanced binary tree so that graph max-pooling becomes a plain contiguous 1D max-pool. Stacking Chebyshev convolution, bias, ReLU, and graph pooling yields a genuine convolutional network on arbitrary graphs.

```python
import numpy as np
import scipy.sparse
import torch
import torch.nn as nn
import torch.nn.functional as F


def normalized_laplacian(W):
    """W: scipy.sparse weighted adjacency (symmetric). Returns L = I - D^{-1/2} W D^{-1/2}."""
    d = np.array(W.sum(axis=1)).squeeze()
    d_inv_sqrt = 1.0 / np.sqrt(d + 1e-12)
    D_inv_sqrt = scipy.sparse.diags(d_inv_sqrt)
    I = scipy.sparse.eye(W.shape[0], dtype=W.dtype)
    return I - D_inv_sqrt @ W @ D_inv_sqrt


def rescale_laplacian(L, lmax=2.0):
    """Map spectrum of L into [-1, 1]: L_tilde = (2 / lmax) * L - I."""
    return L * (2.0 / lmax) - scipy.sparse.eye(L.shape[0], dtype=L.dtype)


class ChebConv(nn.Module):
    """One ChebNet layer: Fin -> Fout feature maps on a fixed graph."""

    def __init__(self, in_features, out_features, K):
        super().__init__()
        self.in_features = in_features
        self.out_features = out_features
        self.K = K
        self.theta = nn.Parameter(torch.empty(K * in_features, out_features))
        self.bias = nn.Parameter(torch.zeros(out_features))
        nn.init.xavier_uniform_(self.theta)

    def forward(self, x, L_tilde):
        """
        x:         (N, M, Fin)  batch of N signals on M vertices with Fin features
        L_tilde:   scipy.sparse CSR matrix, rescaled Laplacian
        returns:   (N, M, Fout)
        """
        N, M, Fin = x.shape
        device = x.device

        # Build Chebyshev basis [xbar_0, ..., xbar_{K-1}] by sparse recurrence.
        # x is (N, M, Fin); transpose to (Fin, M, N) and treat as M x (Fin*N).
        x0 = x.permute(2, 1, 0).reshape(Fin * M, N)  # (Fin*M, N)

        # Convert sparse L_tilde to torch.sparse.
        L_tilde = L_tilde.tocoo()
        indices = torch.from_numpy(np.stack([L_tilde.row, L_tilde.col], axis=0)).long()
        values = torch.from_numpy(L_tilde.data).to(device)
        L_torch = torch.sparse_coo_tensor(indices, values, L_tilde.shape, device=device)

        basis = [x0]
        if self.K > 1:
            x1 = torch.sparse.mm(L_torch, x0)
            basis.append(x1)
        for k in range(2, self.K):
            x2 = 2.0 * torch.sparse.mm(L_torch, basis[k - 1]) - basis[k - 2]
            basis.append(x2)

        # Stack: K x (Fin*M) x N -> (N, M, Fin*K)
        stack = torch.stack(basis, dim=0)                 # (K, Fin*M, N)
        stack = stack.permute(2, 1, 0)                    # (N, Fin*M, K)
        stack = stack.reshape(N, M, Fin, self.K)
        stack = stack.reshape(N * M, Fin * self.K)

        y = stack @ self.theta                            # (N*M, Fout)
        y = y.reshape(N, M, self.out_features)
        return y + self.bias


class ChebNet(nn.Module):
    """Small ChebNet classifier for a fixed graph."""

    def __init__(self, L, K, F, p, n_classes, input_features=1):
        """
        L: list of scipy.sparse CSR Laplacians, one per scale.
        K: list of polynomial orders per conv layer.
        F: list of output feature maps per conv layer.
        p: list of pooling sizes per conv layer (powers of two).
        """
        super().__init__()
        self.L = [rescale_laplacian(laplacian(l), lmax=2.0) for l in L]
        self.convs = nn.ModuleList()
        self.pools = p
        fin = input_features
        for fout, k in zip(F, K):
            self.convs.append(ChebConv(fin, fout, k))
            fin = fout
        self.fc = nn.Linear(fin, n_classes)

    def forward(self, x):
        for conv, L_tilde, p in zip(self.convs, self.L, self.pools):
            x = F.relu(conv(x, L_tilde))
            if p > 1:
                # 1D max-pool along the vertex dimension after graph reordering.
                x = x.permute(0, 2, 1)            # (N, F, M)
                x = F.max_pool1d(x, kernel_size=p, stride=p)
                x = x.permute(0, 2, 1)            # (N, M', F)
        # Global mean pooling over vertices, then classify.
        x = x.mean(dim=1)
        return self.fc(x)
```
