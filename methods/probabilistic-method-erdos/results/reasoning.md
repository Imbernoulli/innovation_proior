I start from the finite Ramsey threshold. For each `k`, Ramsey tells me that a sufficiently large graph must contain either a clique of size `k` or an independent set of size `k`. The upper-bound side is already well served by the Erdos-Szekeres recurrence: fix a vertex, split its neighbors by color, and recurse. That proves `f(k,k) <= binom(2k-2,k-1) < 4^(k-1)`. But this tells me how soon order is forced. It does not tell me how long disorder can persist.

For the lower-bound side, I need a graph on many vertices with no `K_k` and no independent `k`-set. My first instinct is to build one. That is exactly where the obstacle sits. A construction gives me one graph and then asks me to verify all its `k`-subsets, but known explicit examples do not scale anywhere near the upper-bound range. The Erdos-Szekeres proof cannot simply be reversed, because it is designed to force a large monochromatic structure in every object, not to find a single object avoiding all of them.

So I change the unit of attention. I stop asking which graph is good and look at the whole finite universe of labeled graphs on `N` vertices. There are `2^binom(N,2)` such graphs. If the graphs spoiled by cliques and the graphs spoiled by independent sets do not cover this universe, then some unspoiled graph exists even if I cannot point to it.

I count the first bad class. Fix a particular set `S` of `k` vertices. A graph contains a clique on `S` exactly when all `binom(k,2)` edges inside `S` are present. Those edges are forced, and all other edges are free, so the number of graphs containing that fixed clique is `2^(binom(N,2)-binom(k,2))`. There are `binom(N,k)` possible sets `S`, and a graph can be counted more than once, but overcounting is harmless because I only need an upper bound. Thus the number of graphs containing some `K_k` is at most `binom(N,k)2^(binom(N,2)-binom(k,2))`, which is less than `(N^k/k!)2^(binom(N,2)-binom(k,2))`.

I want this bad class to be less than half of all graphs. After dividing by `2^binom(N,2)`, the condition becomes `N^k/(k!2^binom(k,2)) < 1/2`, or equivalently `2N^k < k!2^(k(k-1)/2)`. If `N <= 2^(k/2)`, then `N^k <= 2^(k^2/2)`. The remaining comparison is `2 < k!2^(-k/2)`, which holds for `k >= 3` and only grows easier. Therefore fewer than half of all labeled graphs contain a `K_k`.

Now I use complementation. The complement map is a bijection on the same graph universe, and an independent `k`-set in a graph is exactly a `K_k` in its complement. Therefore the number of graphs containing an independent `k`-set is the same as the number containing a clique, so it is also less than half of the universe. Two subsets, each of size less than half the universe, cannot cover the universe. Some graph lies in neither one. That graph has no `K_k` and no independent `k`-set, so `f(k,k) > 2^(k/2)` in the 1947 theorem's form.

This is the non-obvious proof move. I do not construct the witness; I prove that the bad witnesses are too sparse in aggregate to exhaust the space. In modern language I put the uniform distribution on all red/blue edge colorings of `K_n`. For a fixed `k`-set, the probability that it is monochromatic is `2^(1-binom(k,2))`: all red or all blue. By the union bound, the probability that some `k`-set is monochromatic is at most `binom(n,k)2^(1-binom(k,2))`. If this number is below `1`, then the probability of avoiding every bad event is positive, and a good coloring exists.

The probability language is not just cosmetic, even though the 1947 proof is a finite count. It crystallizes a new standard of evidence: positive probability certifies existence. The object can remain unnamed. This is why the result requires a distinctive proof mindset rather than recombination. Ramsey and Erdos-Szekeres already supplied forcing principles, but their arguments inspect an arbitrary object and drive toward an ordered substructure. Explicit extremal approaches try to build one exceptional object. The 1947 argument instead treats all objects as an ensemble, counts the global incidence of bad events, and obtains existence from leftover measure. Prior constructive or local counting methods do not naturally suggest that a random object should avoid all forbidden configurations with positive probability.

Once I have this mindset, the refinements are visible. The same union-bound condition `binom(n,k)2^(1-binom(k,2)) < 1` can be optimized more carefully than the simple `2^(k/2)` choice, giving `R(k,k) > (1/(e sqrt(2)) + o(1)) k 2^(k/2)`. The proof mechanism is the same: make the expected number of bad `k`-sets less than one.

Then I loosen the demand further. Let `X` be the number of monochromatic `k`-sets in a random coloring. Linearity of expectation gives `E[X] = binom(n,k)2^(1-binom(k,2))`. Some coloring has at most this many bad sets. If I delete one vertex from each bad set, at most `E[X]` vertices disappear, and no monochromatic `K_k` remains among the survivors. This gives `R(k,k) > n - binom(n,k)2^(1-binom(k,2))`, and optimizing the right side gives `(1/e + o(1)) k 2^(k/2)`.

The same first-moment logic also explains the earlier tournament example. In a random tournament, each of the `n!` vertex orderings becomes a directed Hamiltonian path with probability `2^-(n-1)`, so the expected number of Hamiltonian paths is `n!2^-(n-1)`. Some tournament has at least that many. This shows that averaging existed before the Ramsey lower-bound paper, but the 1947 result turns the idea into a central way of proving existence for hard combinatorial avoidance problems.

So the rebuilt answer is this: choose a natural random object, identify the bad events, show their total probability is less than one, and infer that some object avoids them all. If the random object is only nearly good, count blemishes by expectation and alter it. The Ramsey lower bound is the canonical shock because it solves the lower-bound obstruction without construction: a graph exists because random graphs cannot all be bad.
