# Synthesis — Convex relaxation of AC-OPF (SDP/SOCP, zero duality gap)

## Pain point / research question
Operate a power grid at minimum generation cost subject to the physical AC power-flow laws and engineering limits. The AC power-flow equations are quadratic in the complex bus voltages V, so the feasible set is nonconvex; OPF is a nonconvex QCQP and provably NP-hard (Lavaei–Low: reduce the {−1,1} discrete voltage Case 2 to it). Local methods (Newton, IP on KKT) only give local optima, no global certificate. DC-OPF linearizes (decouples P from V, assumes flat voltages, no losses, no reactive power) — convex (an LP) but a model error. Want: a convex problem that gives a *global* optimum of the true AC-OPF with a certificate.

## Background / load-bearing facts (→ context.md Background)
- Bus injection model (BIM): admittance matrix Y, I = YV, net complex injection s_j = V_j I_j* = V^H Y_j^H V where Y_j = e_j e_j^H Y. So Re s_j = V^H Φ_j V, Im s_j = V^H Ψ_j V with Φ_j = (Y_j^H + Y_j)/2, Ψ_j = (Y_j^H − Y_j)/(2i). |V_j|^2 = V^H J_j V, J_j = e_j e_j^H. All injections/voltage magnitudes are QUADRATIC FORMS in V.
- OPF is therefore a standard QCQP: min V^H C V s.t. quadratic inequality constraints. Nonconvex because the Φ_j,Ψ_j,C are indefinite.
- NP-hardness (Lavaei–Low App B): Case 1 fix |V_j|=1 ∀j → min V^H Y V over the unit-modulus torus (nonconvex, encircles origin); Case 2 add Im{Y}=0, Q=0, V_j ∈ {−1,1} → a {±1} quadratic = MAXCUT-type / number-partitioning-hard. Hence OPF NP-hard.
- DC-OPF: standard linearization (|V|≈1, sin θ≈θ, neglect resistance/losses) → economic dispatch LP. Convex but lossless & approximate; the motivation to do better while staying convex.
- Jabr 2006: for a RADIAL (tree) distribution network, the load-flow / branch-flow equations can be cast as a conic program (SOCP). Branch variables u_i = |V_i|^2/√2, R_ij = Re(V_i V_j*)·√2-ish, I_ij; the bilinear identity |V_i|^2|V_j|^2 = Re(V_iV_j*)^2 + Im(V_iV_j*)^2 becomes a rotated second-order cone 2 u_i u_j ≥ R_ij^2 + I_ij^2. Linear power-flow in (u,R,I); recover angles by traversing the tree. Limitation: arctan/angle-consistency constraints around cycles fail for MESHED networks → SOCP not tight for meshes.
- Schur complement: turns the apparent-power |S_lm|^2 ≤ (S^max)^2 (a sum of squares of linear-in-W terms) and the quadratic generation cost f_k(P) ≤ α_k into LMIs — so the lifted problem is a genuine SDP.

## The derivation chain (→ reasoning.md heart)
1. min Σ f_k(P_Gk) s.t. AC power balance (P_inj, Q_inj quadratic in V), |V| limits, line limits.
2. Everything is V^H M V for Hermitian M (Lemma 1: P_k,inj=Tr(Y_k W), Q=Tr(Ȳ_k W), |V_k|^2=Tr(M_k W), |S_lm|^2 = Tr(Y_lm W)^2 + Tr(Ȳ_lm W)^2, with W=XX^T, X=[Re V; Im V]). So it's a QCQP.
3. Lift: W := V V^H (or XX^T in real form). Then every quadratic form V^H M V = Tr(M W) is LINEAR in W. The objective and all constraints become linear in W. The only nonconvexity left: W ⪰ 0 AND rank(W)=1 (W=VV^H ⟺ W⪰0 and rank 1).
4. W⪰0 is convex; rank(W)=1 is the sole nonconvex constraint. DROP it → SDP relaxation (Optimization 3): min Tr(CW) s.t. linear constraints, W⪰0. Convex, polynomial-time.
5. Recovery / exactness: if the SDP optimum W* is rank 1, factor W*=V V^H to get the global AC-OPF optimum — relaxation EXACT. The relaxation lower-bounds OPF; exact ⟺ rank-1 ⟺ zero duality gap (Theorem 2(i): duality gap of Opt 1 zero ⟺ Opt 3 has a rank-1 solution).
6. Lavaei–Low refine via the DUAL: build the Dual OPF (Optimization 4, an SDP) by Lagrangian; its matrix variable A(x,r)⪰0. Sufficient condition (Theorem 2(ii)): if A(x*,r*) has a zero eigenvalue of multiplicity EXACTLY 2 then duality gap zero, and V* is recovered from the 2-dim null space: [X1;X2] in null(A) ⇒ V* = (ζ1+ζ2 i)(X1+X2 i). (Multiplicity 2 not 1 because of the real-form symmetry: V and e^{iφ}V give the same W, so the null space is inherently ≥2-dim.) Lemma 2: a rank-1 Opt-3 solution always implies infinitely many rank-2 ones (same symmetry) — so numerically you often see rank-2, but rank-1 is recoverable via Corollary 1(ii): if W* has rank 2 with eigenpairs (ρ1,E1),(ρ2,E2), then (ρ1+ρ2)E E^T (E=unit eigvec of ρ1, appropriately phased) is a rank-1 solution.
7. WHY zero duality gap holds in practice (Theorems 3–6, Perron–Frobenius): For the modified OPF (allow load over-satisfaction: P_L ≥ P_D instead of =), the dual gets λ_k ≥ 0. Then the off-diagonal entries of T(x*,r*) are non-positive (because line/transformer admittances y_lm have Re ≥ 0 (resistance nonneg) and the off-diag of Re{Y} is −Re{y_lm} ≤ 0). A symmetric matrix with non-positive off-diagonals that is irreducible (strongly-connected graph) is (after PSD shift) governed by Perron–Frobenius ⇒ its smallest eigenvalue is SIMPLE ⇒ via the block structure A=[[T,0],[0,T]], the zero eigenvalue of A has multiplicity exactly 2 ⇒ Theorem 2(ii) ⇒ zero gap. The transformer resistance must be > 0 (else the resistive graph disconnects → multiplicity 4 → condition fails); adding 1e-5 pu resistance to lossless transformers reconnects Re{Y}, restoring multiplicity 2. This is exactly why IEEE 14/30/57/118/300 all satisfy it after the tiny perturbation.
8. Radial special case: for trees, the SDP can be replaced by the much cheaper SOCP — only the 2×2 principal submatrices W(j,k)=[[W_jj, W_jk],[W_kj, W_kk]] need to be PSD (drop rank). For a tree (chordal, all cliques are edges) the per-edge 2×2 PSD set is equivalent to the full W⪰0 with a consistent rank-1 completion (Jabr). For meshes the 2×2-PSD SOCP is a strictly weaker relaxation than the full SDP (cycle angle consistency lost).

## Design decisions → why (table)
- Lift to W=VV^H not keep V: only way to convexify — quadratics become linear; rank-1 isolates ALL the nonconvexity into one constraint you can drop. Alternative (DC linearization) loses losses/reactive/voltage; lifting keeps the exact AC physics.
- Drop rank(W)=1 (not e.g. penalize it): rank is the sole nonconvex constraint; dropping gives an SDP with a provable lower bound and an exactness certificate (rank of the solution). Penalizing rank (nuclear-norm etc.) reintroduces nonconvexity/heuristics with no global guarantee.
- Solve the DUAL (Opt 4) rather than primal SDP (Opt 3): (a) Opt 4 has O(|N|+|L|) variables vs O(n^2) dense W in Opt 3 — cheaper and sparse; (b) the load profile and limits appear only in the objective, the topology Y only in the LMIs → clean decomposition; (c) the zero-eigenvalue-multiplicity certificate is read directly off A(x*,r*). 
- Multiplicity-2 (not 1) certificate: the lifting has a global phase symmetry (V ↦ e^{iφ}V), so the real-form null space is inherently even-dimensional; "exactly 2" is the rank-1-equivalent condition.
- Modified OPF (P_L ≥ P_D): turns equality load constraints into inequalities so λ_k ≥ 0, which is what makes T's off-diagonals signed and Perron–Frobenius applicable. Justified physically: over-delivering power is never cheaper (positive nodal price), so optimum delivers exactly P_D ⇒ same solution as OPF in normal operation (Theorem 4 / Corollary 2).
- Add 1e-5 pu resistance to lossless transformers: makes Re{Y} strongly connected so the multiplicity is 2 not >2. Tiny so it doesn't change the solution materially; pure conditioning of the certificate.
- SOCP for radial: trees are chordal with edge-cliques only, so 2×2 PSD per edge ≡ full SDP and is far cheaper. Don't use it for meshes — strictly weaker.
