I start with the ordinary risk minimization problem. I have samples `z_i = (x_i, y_i)` from an unknown distribution, a loss `L(y, f(x, alpha))`, and a true risk `R(alpha) = integral L(y, f(x, alpha)) dP(x,y)`. I cannot compute this risk because I do not know `P`, so I replace it with the empirical risk `R_emp(alpha) = (1/l) sum_i L(y_i, f(x_i, alpha))`.

At first it is tempting to say that I should simply minimize `R_emp`. That cannot be the whole answer. A concentration bound for one fixed function does not cover the function I choose after looking at the sample. Once the sample drives the choice, I need a statement that holds simultaneously over all functions the learning rule might select. This turns the problem from pointwise convergence into uniform convergence: `Prob{sup_alpha |R(alpha) - R_emp(alpha)| > epsilon} -> 0`.

The VC ancestor gives me the right capacity language. I look at how many dichotomies or event intersections a class can induce on finite samples. If the class can shatter every finite sample, the growth function behaves like `2^l`; if it cannot, the growth function becomes polynomially bounded. That polynomial control is what lets the empirical frequencies converge uniformly to their true probabilities.

Now I return to empirical risk. For binary classification, Vapnik gives a finite-sample VC bound of the form `Prob{sup_w |P(w) - v(w)| > epsilon} < (2le/h)^h exp(-epsilon^2 l)`, where `h` is the VC dimension, `P(w)` is true error, and `v(w)` is training error. Inverting the bound gives `P(w) < v(w) + C0(l/h, eta)`, with `C0(l/h, eta) = sqrt((h(ln(2l/h)+1) - ln eta)/l)`. I now see the key ratio: a class is useful only when the sample size is large relative to its capacity.

This changes how I choose models. If I take one huge class, empirical risk can be tiny, but the confidence interval can be too large to say anything. If I take one tiny class, the confidence interval is small, but the best function inside it may fit badly. I therefore organize the search into a structure `S1 subset S2 subset ... subset Sn`, with capacities `h1 < h2 < ... < hn`. Inside each `Sk`, I do ordinary empirical risk minimization and get a candidate `alpha_k`.

The final choice is not the candidate with the lowest training error. It is the candidate with the lowest guaranteed risk bound, `R_emp(alpha_k) + Omega(h_k, l, eta_k)`. As `k` increases, the empirical risk usually falls because the class is richer, while the capacity penalty rises because the class can fit more samples in more ways. I choose the minimum of this upper envelope. This is structural risk minimization.

The relative bound in Vapnik's 1991 paper reinforces the same point. When empirical error is zero, the confidence contribution can shrink faster inside a controlled class; in Vapnik's notation the zero-error case becomes `2C0^2`. Zero training error is therefore meaningful only after I know the class capacity. A zero-error separator in a small-capacity set is a different object from a zero-error fit in an unconstrained set.

The author retrospective widens the construction. An admissible structure has nested sets whose union is dense enough to approximate the target while each member remains capacity-controlled. If the selected level grows slowly with sample size, the procedure keeps consistency while gradually allowing richer functions. The method therefore balances approximation error from choosing a smaller class against estimation error from choosing a larger one.

I can also see why the method is not just a penalty on parameter count. VC dimension and related capacity measures depend on the function class's ability to realize labelings, not merely on how many parameters are written down. A margin-constrained hyperplane class makes this concrete: for data in a ball of radius `R` and margin `Delta`, the VC dimension is bounded by `min(R^2/Delta^2, n) + 1`. Maximizing margin reduces capacity even when the ambient feature representation is high-dimensional.

This gives the route to support vector machines. I can define the structure by margin levels, find separators that fit the data, and prefer the one with the largest margin because it lies in the smallest capacity-controlled set compatible with the sample. In dual form and with kernels, the same principle remains: I am not merely fitting the training set; I am choosing among capacity-indexed hypothesis spaces by empirical risk plus a confidence term.
