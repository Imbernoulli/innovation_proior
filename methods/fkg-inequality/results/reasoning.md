So, assembling: ‚ü®f; Œº, Œì''_a‚ü© ‚â§ ‚ü®f; Œº, Œì_a‚ü© ‚â§ ‚ü®f; Œº, Œì'_a‚ü©, hence ‚ü®f; Œº, Œì''_a‚ü© ‚â§ ‚ü®f; Œº, Œì'_a‚ü©, and the same for g, so both factors in Œ© ‚â• (Œ£'Œº Œ£''Œº)‚Åª¬π(‚Ä¶)(‚Ä¶) are ‚â• 0, so Œ© ‚â• 0, i.e. ‚ü®fg‚ü© ‚â• ‚ü®f‚ü©‚ü®g‚ü© on Œì. Induction closes. The proposition is proved: on a finite distributive lattice with a measure satisfying Œº(x‚àßy)Œº(x‚à®y) ‚â• Œº(x)Œº(y), any two increasing (or two decreasing) functions are positively correlated.

Let me step back and feel the architecture, because it's not an accident. The chain case was the engine ‚Äî the termwise Chebyshev fact. The induction peels off one atom at a time, and at each peel the obstruction (the incomparable cross terms) is reduced to a single monotonicity statement ‚Äî an increasing function averages higher above the atom than off it ‚Äî and *that* statement is itself an instance of the same inequality on a shorter lattice, with one of the two functions being a decreasing indicator (the semi-ideal œá) or a monotone density ratio (ŒºÃÉ/Œº). The condition (A) enters in exactly one place: it's what forces ŒºÃÉ/Œº to be monotone, i.e. it's what guarantees the reweighting between "off the atom" and "above the atom" doesn't fight the order. Without (A) that ratio could go the wrong way and the cross terms would win. Now the special case I started from. Take a product measure, Œº(x) = ‚àè_i Œº_i(x_i). I checked it satisfies (A) with equality, so the proposition applies immediately: increasing functions of independent coordinates are positively correlated, ‚ü®fg‚ü© ‚â• ‚ü®f‚ü©‚ü®g‚ü©, and with f = 1_A, g = 1_B for increasing events, P(A‚à©B) ‚â• P(A)P(B). That's Harris's percolation inequality ‚Äî the independent-edge measure is a product measure, increasing connection events are increasing functions, done. It really was the same fact all along.

And in the product case there's an even more transparent proof that doesn't need the lattice machinery, just induction on the number of coordinates by conditioning ‚Äî worth writing out because it shows the mechanism nakedly. Let x = (x_1, ‚Ä¶, x_n) have independent coordinates, f, g increasing. For n = 1, x and an independent copy y live on a single chain, so they're comparable and (f(x) ‚àí f(y))(g(x) ‚àí g(y)) ‚â• 0 always; taking expectations, 0 ‚â§ E[(f(x) ‚àí f(y))(g(x) ‚àí g(y))] = 2(E[fg] ‚àí E[f]E[g]), so E[fg] ‚â• E[f]E[g] ‚Äî the same Chebyshev base case. Also f_1 and g_1 are increasing functions of the single variable x_1 ‚Äî increasing because f, g are increasing in x_1 and expectation preserves it. Therefore

  E[fg] = E[h_1] ‚â• E[f_1 g_1] ‚â• E[f_1] E[g_1] = E[f] E[g],

where the first ‚â• is the conditional inequality, the second is the n = 1 case applied to f_1, g_1 on the single coordinate x_1, and the ends are the tower property. A couple of immediate corollaries fall out for free. If A, B are both *decreasing* events, their complements are increasing, and P(A‚à©B) ‚â• P(A)P(B) follows the same way. Good ‚Äî "good events help each other" is now a single theorem with two hypotheses.

I can now write the final form without any model-specific language. Let Œì be a finite distributive lattice and Œº a positive measure on Œì with Œº(x‚àßy) Œº(x‚à®y) ‚â• Œº(x) Œº(y) for all x, y. The proof: reduce to strictly positive Œº; induct on the length of Œì; at each step peel off an atom a, split the lattice into the part above a and the part not above a, drop the two same-part blocks by the inductive hypothesis, factor the cross block, and reduce its sign to "an increasing function averages higher above a than off a," which is itself an instance of the inequality on shorter lattices ‚Äî using the semi-ideal indicator's monotonicity and the increasing density ratio ŒºÃÉ/Œº that (A) supplies. The base case is the chain, where positive correlation is the termwise Chebyshev sum inequality. The product measure satisfies (A) with equality, giving Harris's inequality E[fg] ‚â• E[f]E[g] for increasing functions of independent variables ‚Äî provable on its own by conditioning one coordinate at a time down to the same scalar Chebyshev fact.

```python
from itertools import combinations
import math

GROUND  = ("a", "b", "c")
LATTICE = [frozenset(s) for k in range(len(GROUND) + 1)
           for s in combinations(GROUND, k)]            # the distributive lattice P(X)

def meet(x, y): return x & y                            # greatest lower bound
def join(x, y): return x | y                            # least upper bound

def increasing(v):                                      # monotone under inclusion
    return all(not (x <= y) or v[x] <= v[y] + 1e-12
               for x in LATTICE for y in LATTICE)

def average(mu, h):
    Z = sum(mu[x] for x in LATTICE)
    return sum(mu[x] * h[x] for x in LATTICE) / Z

def covariance(mu, f, g):                               # <fg> - <f><g>
    return average(mu, {x: f[x]*g[x] for x in LATTICE}) - average(mu, f)*average(mu, g)

def condition_A(mu):                                    # mu(meet)*mu(join) >= mu[x]*mu[y]
    return all(mu[meet(x,y)]*mu[join(x,y)] >= mu[x]*mu[y] - 1e-12
               for x in LATTICE for y in LATTICE)

# two increasing indicators of up-sets
f = {x: float("a" in x) for x in LATTICE}
g = {x: float("b" in x) for x in LATTICE}
assert increasing(f) and increasing(g)

# product (independent) measure: satisfies (A) with equality -> covariance >= 0
w = {"a": 2.0, "b": 3.0, "c": 1.5}
mu_prod = {x: 1.0 for x in LATTICE}
for x in LATTICE:
    for e in GROUND:
        mu_prod[x] *= w[e] if e in x else 1.0
assert all(abs(mu_prod[meet(x,y)]*mu_prod[join(x,y)] - mu_prod[x]*mu_prod[y]) < 1e-12
           for x in LATTICE for y in LATTICE)
assert condition_A(mu_prod) and covariance(mu_prod, f, g) >= -1e-12

# log-supermodular measure (strict for some pairs): covariance >= 0
mu_super = {x: math.exp(sum(w[e] for e in x) + 0.7*max(0, len(x)-1)) for x in LATTICE}
assert condition_A(mu_super) and covariance(mu_super, f, g) >= -1e-12

# a measure that VIOLATES (A): the same increasing f, g become NEGATIVELY correlated
mu_bad = {frozenset(): 0.135, frozenset({"a"}): 0.848, frozenset({"b"}): 0.765,
          frozenset({"c"}): 0.256, frozenset({"a","b"}): 0.496, frozenset({"a","c"}): 0.450,
          frozenset({"b","c"}): 0.653, frozenset({"a","b","c"}): 0.790}
assert not condition_A(mu_bad) and covariance(mu_bad, f, g) < 0   # condition (A) is doing real work
```
