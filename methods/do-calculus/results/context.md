## Research Situation

I observe a population generated by stable mechanisms, but the target question asks what would happen under an intervention. The data show how variables vary together when left alone. The causal question asks for the distribution of an outcome after one mechanism is externally fixed while the rest of the system continues to operate.

## Graphical Commitments

A directed graph can record which variables are direct parents of which other variables. Missing directed arrows stand for exclusion assumptions, and hidden common causes can be represented by explicit latent nodes or special confounding marks. Earlier graphical-model work already supplies a way to read conditional independence from such a graph through path blocking.

## Existing Estimation Moves

Randomized experiments make intervention effects easy to estimate because treatment assignment is severed from its usual causes. In observational studies, analysts instead try regression, stratification, or matching. These moves work only when the chosen covariates block the right paths without opening new ones, and they give little guidance when the causal route runs through variables affected by treatment.

## Main Obstacle

Conditioning on a variable is not the same as setting it. Conditioning selects units whose value already arose from the system; setting a value replaces part of the system. A valid method must keep track of that difference while also using the graph to decide which comparisons survive hidden confounding, mediation, and selection by colliders.

## Desired Output

The needed object is an identification criterion: a disciplined way to decide whether passive data plus explicit causal assumptions determine the interventional quantity. When the assumptions leave two causal models observationally indistinguishable but interventionally different, the method should expose that failure rather than returning a misleading adjustment formula.
