We are handed only unlabelled observations $x_1, \dots, x_l$ drawn i.i.d. from an unknown distribution $P$ on a high-dimensional input space, and we want a simple region $S$ that holds most of the probability mass, so that a future point landing outside $S$ can be flagged as novel, faulty, or anomalous. We also want one operational knob: a direct control on the fraction of training points we are willing to leave outside the learned region, because condition-monitoring and novelty settings usually arrive with a prior belief about contamination. The reflexive solution is to estimate the density $p$ and threshold its level sets $\{x : p(x) \ge t\}$, but that solves a strictly larger problem than the one posed. Estimating $p$ in high dimension is data-hungry, and the density may not even exist if $P$ has singular components, whereas the support or high-mass region is still meaningful. There is a precise statistical statement of the right target — fixing a set class $\mathcal C$ and a size functional $\lambda$, the minimum-volume set is $U(\alpha) = \inf\{\lambda(C) : P(C) \ge \alpha,\, C \in \mathcal C\}$, the support at $\alpha = 1$ — but the word "volume" is the obstacle: directly minimizing input-space volume over a rich high-dimensional class is not a convex kernel computation we can run. The supervised SVM offers a different kind of size control, since its $\tfrac12\|w\|^2$ term controls the flatness of a feature-space decision function rather than literal volume, and the kernel trick turns flat feature-space functions into nonlinear input-space boundaries. That machinery — a convex QP, a support-vector expansion, a kernel decision function — is exactly what we want to reuse, but it is structurally a two-class separator, and here there is only one class with no negative sample to push against.

I propose the One-Class SVM. The single idea that unlocks the one-class setting is to use the one canonical feature-space point that comes for free, the origin, as the reference to separate the data from. We seek a hyperplane that puts the mapped data on one side and the origin on the other with maximum margin, and call the data side the accepted region. This is well posed precisely because of Gaussian-kernel geometry: for $k(x,y)=\exp(-\|x-y\|^2/c)$ every diagonal entry $k(x,x)=1$ so all mapped points have unit norm, and every pairwise inner product is positive so all pairwise angles are acute — the images sit together on a patch of the unit sphere, cleanly separable from the origin, and the maximum-margin hyperplane cuts off a spherical cap containing the data. Concretely we want $\langle w, \Phi(x_i)\rangle$ to reach a margin value $\rho$ for most points, with slacks $\xi_i$ for the failures, while pushing $\rho$ upward and keeping $\|w\|$ small:
$$\min_{w,\xi,\rho}\ \tfrac12\|w\|^2 + \frac{1}{\nu l}\sum_i \xi_i - \rho \qquad \text{s.t.}\quad \langle w, \Phi(x_i)\rangle \ge \rho - \xi_i,\ \ \xi_i \ge 0.$$
The $-\rho$ term is what rewards pushing the separating hyperplane away from the origin; without it $\rho$ would collapse to zero and the cap would be empty. The decision is the signed expansion $f(x) = \operatorname{sgn}(\langle w, \Phi(x)\rangle - \rho)$, positive for inliers and negative for outliers. The strange-looking coefficient $1/(\nu l)$ on the slacks is not a guess — it is chosen so that the KKT conditions later hand $\nu$ a clean counting meaning.

Introducing multipliers $\alpha_i \ge 0$ on the margin constraints and $\beta_i \ge 0$ on $\xi_i \ge 0$ and writing the Lagrangian, stationarity gives the entire dual. From $\partial L/\partial w = 0$ we get the kernel expansion $w = \sum_i \alpha_i \Phi(x_i)$; from $\partial L/\partial \xi_i = 0$ we get $\alpha_i = 1/(\nu l) - \beta_i$, hence the box $0 \le \alpha_i \le 1/(\nu l)$; and from $\partial L/\partial \rho = 0$ we get the equality $\sum_i \alpha_i = 1$. Substituting $w$ back, the quadratic terms collapse to $\tfrac12\sum_{ij}\alpha_i\alpha_j k(x_i,x_j) - \sum_{ij}\alpha_i\alpha_j k(x_i,x_j)$, the $\rho$ terms cancel because the multipliers sum to one, and the slack terms vanish by stationarity, leaving the box-plus-equality QP
$$\min_\alpha\ \tfrac12\sum_{ij}\alpha_i\alpha_j k(x_i,x_j) \qquad \text{s.t.}\quad 0 \le \alpha_i \le \frac{1}{\nu l},\ \ \sum_i \alpha_i = 1,$$
with kernel decision $f(x) = \operatorname{sgn}\!\big(\sum_i \alpha_i k(x_i,x) - \rho\big)$. The offset $\rho$ is recovered from any non-bound support vector: if $0 < \alpha_i < 1/(\nu l)$ then $\beta_i > 0$, so $\beta_i \xi_i = 0$ forces $\xi_i = 0$ and the margin constraint is tight, giving $\rho = \sum_j \alpha_j k(x_j, x_i)$.

The same KKT equations are what give $\nu$ its meaning, and this is the payoff of the $1/(\nu l)$ scaling. Write $U = 1/(\nu l)$. Any training point that is actually outside the region has positive slack $\xi_i$, so complementary slackness on $\beta_i\xi_i$ forces $\beta_i = 0$, hence $\alpha_i = U$: every outlier is pinned at the top of the box. Since the $\alpha_i$ are nonnegative and sum to one, at most $\nu l$ of them can sit at that ceiling, so the training outlier fraction is at most $\nu$. The count runs the other way for support vectors: each nonzero $\alpha_i$ contributes at most $U$, so reaching total mass one needs at least $\nu l$ nonzero coefficients, putting the support-vector fraction at least $\nu$. Under an analytic nonconstant kernel and a distribution without discrete components the margin-only fraction (points exactly on the margin) vanishes asymptotically, so both fractions converge to $\nu$ — the penalty parameter has become a count parameter. The endpoints confirm the design: as $\nu \to 0$ the ceiling $U \to \infty$ and the slack penalty becomes infinite, giving a hard-margin support estimator, which stays feasible only because $\rho$ is left unconstrained and free to go negative (had we imposed $\rho \ge 0$, the $\rho$ stationarity would degrade to an inequality $\sum_i\alpha_i \ge 1$ and the multipliers could diverge instead of giving the clean equality); at $\nu = 1$ the box $0 \le \alpha_i \le 1/l$ with the equality forces every $\alpha_i = 1/l$, so the expansion becomes $(1/l)\sum_i k(x_i,x) - \rho$, a thresholded Parzen-window estimate. There is also a bounded-influence property absent from density estimation: once a point has positive slack its $\alpha_i$ is pinned at $U$, so moving that outlying feature-space point parallel to $w$ without erasing its slack leaves the separating hyperplane unchanged.

This is the same description as the soft feature-space ball, or support-vector data description, that finds a center $a$ and radius $R$ via $\min_{R,a,\xi} R^2 + \frac{1}{\nu l}\sum_i\xi_i$ subject to $\|\Phi(x_i)-a\|^2 \le R^2 + \xi_i$. Its dual carries an extra linear term $\sum_i\alpha_i k(x_i,x_i)$, but whenever $k(x,x)$ is constant — as for the Gaussian and any translation-invariant kernel — that term is constant under $\sum_i\alpha_i=1$ and drops out, so the ball dual and the OCSVM dual share the same optimizer up to the irrelevant factor $\tfrac12$, and their decision functions coincide. Geometrically, constant diagonal puts all mapped points on a sphere, and the smallest enclosing segment of that sphere is exactly what the maximum-margin hyperplane cuts; we keep the hyperplane form because one signed kernel expansion and one offset are cleaner than an explicit center and radius.

To solve the dual at scale we exploit that it has a single equality constraint, so the smallest feasible move adjusts a pair of multipliers at once — sequential minimal optimization. Freezing all but $\alpha_1,\alpha_2$, set $\Delta = 1 - \sum_{i\ge 3}\alpha_i$ so that $\alpha_1 + \alpha_2 = \Delta$, write $K_{ij}=k(x_i,x_j)$ and $C_i = \sum_{j\ge 3}\alpha_j K_{ij}$. Eliminating $\alpha_1 = \Delta - \alpha_2$ and differentiating the pair's objective gives $\alpha_2 = [\Delta(K_{11}-K_{12}) + C_1 - C_2]/(K_{11}+K_{22}-2K_{12})$, which in terms of the current expansion outputs $O_i = K_{1i}\alpha_1^* + K_{2i}\alpha_2^* + C_i$ reads as the Newton step $\alpha_2 = \alpha_2^* + (O_1 - O_2)/(K_{11}+K_{22}-2K_{12})$ along the equality-preserving direction. We then clip $\alpha_2$ to $\max(0,\Delta-U) \le \alpha_2 \le \min(U,\Delta)$ with $U = 1/(\nu l)$, set $\alpha_1 = \Delta - \alpha_2$, recompute $\rho$, and iterate by scanning for a KKT violator and pairing it with the non-bound support vector that maximizes $|O_i - O_j|$. With an RBF kernel $\exp(-\gamma\|x-y\|^2)$, small $\gamma$ gives a smoother looser region and large $\gamma$ a tighter more variable boundary, and because those distances live in the exponent, feature scaling matters before fitting. Finally, since the signed value $\sum_i\alpha_i k(x_i,x)-\rho$ is positive for inliers but an anomaly detector wants larger to mean more unusual, we return its negative.

```python
from sklearn.svm import OneClassSVM


class SupportRegionDetector:
    """PyOD-style wrapper around sklearn's libsvm-backed one-class solver."""

    def __init__(self, nu=0.5, gamma="auto", kernel="rbf"):
        self.nu = nu
        self.gamma = gamma
        self.kernel = kernel
        self.model = OneClassSVM(kernel=kernel, nu=nu, gamma=gamma)

    def fit(self, X):
        # X is unlabelled feature data; scale features before fitting an RBF kernel.
        self.model.fit(X)
        return self

    def decision_function(self, X):
        # sklearn returns sum_i alpha_i k(x_i, x) - rho, positive for inliers.
        # PyOD inverts that ordering so larger values are more anomalous.
        return -self.model.decision_function(X)
```
