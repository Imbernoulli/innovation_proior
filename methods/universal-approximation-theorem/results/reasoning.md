I start with the narrowness of the dictionary. Every hidden unit is only one scalar curve seen through an affine functional, and the output only forms a finite linear combination of those views. If I stare at an arbitrary continuous function on a cube, the dictionary looks too small. It has no explicit products, no coordinate grid, and no target-specific basis functions. The training story is a distraction here, because the question is not whether gradient descent finds parameters. The question is whether the closed linear span is already all of `C([0,1]^n)`.

The older exact-superposition result helps me frame the difficulty but does not solve it. It says multivariate continuous functions can be built from one-variable continuous functions and addition, but the one-variable functions are not forced to be repeated copies of one fixed activation. Fourier approximation also helps only as an analogy: a repeated family can be complete, but trigonometric functions bring their own algebraic structure. A hidden layer gives ridge functions, each constant along hyperplanes. I need to understand whether such ridge responses have any continuous blind spot.

I change the problem from construction to separation. If the span is not dense, then Hahn-Banach supplies a nonzero bounded linear functional that vanishes on it. On a compact cube, Riesz represents that functional as integration against a finite signed regular Borel measure. So failure of approximation would mean that some nonzero measure `mu` satisfies `int sigma(w . x + theta) dmu(x) = 0` for every direction and threshold. Now the activation is not just a nonlinear curve; its affine copies are tests applied to signed measures. The right property is that these tests discriminate every nonzero measure.

A sigmoidal activation is promising because scaling turns it into a half-space probe. I first subtract the lower limiting value and divide by the jump, which is harmless because constant hidden units are available by setting the direction to zero. Now the limiting values are `0` and `1`, and I look at `sigma(lambda(w . x + theta) + phi)` as `lambda` grows. Away from the hyperplane it tends to a half-space indicator; on the hyperplane it keeps the value `sigma(phi)`. Since the annihilation assumption holds for every affine copy, bounded convergence gives an identity involving the measure of the open half-space and the measure of the separating hyperplane. Varying `phi` separates those two terms, so both must be zero.

The next step needs care. It is tempting to wave at simple functions, but the safer route is to push the measure forward. Fix a direction `w` and send `x` to `w . x`; call the resulting signed measure on the line `nu_w`. The half-space and hyperplane conclusions say that every half-line and every point has zero `nu_w`-mass. Intervals are built from these sets, and intervals generate the Borel sets, so the push-forward measure is zero. Therefore every one-dimensional Fourier integral `int exp(i s w . x) dmu(x)` is zero. Since `s w` ranges over all frequencies, the Fourier transform of `mu` vanishes everywhere. A finite compactly supported measure is determined by its Fourier transform, so `mu` is zero. That contradicts the separating functional and closes the density proof.

This argument explains why the theorem is existential. It never tells me how to select the hidden units. It proves that no nonzero continuous functional can separate a target from the closed span. The hidden units become probes against measures first, not constructive tiles. A step-and-bump picture can give intuition for why large weights carve local features, but the formal single-hidden-layer theorem is the measure-duality argument.

The classification statement now has the right limitation. A continuous network cannot uniformly reproduce a discontinuous decision indicator. Instead, a measurable decision function can be replaced by a continuous function on a large-measure subset, and that continuous replacement can be uniformly approximated. For a closed region, a continuous distance ramp gives correct classification inside the set and outside a chosen boundary band. The bad set is not magic; it is the price of continuity.

I then ask whether sigmoidality is the true boundary. It is sufficient, but it is too special. A polynomial activation clearly fails, because affine substitution into a degree `d` polynomial still produces a multivariate polynomial of degree at most `d`, and finite sums stay in that finite-degree space. The closure cannot contain all continuous functions. So non-polynomiality is at least necessary for the standard thresholded span.

The sharp frontier turns that necessary condition into an equivalence under mild regularity. The multivariate problem reduces to ridge functions of the form `f_i(a_i . x)`, and the remaining burden is one-dimensional: can shifted and scaled copies of the activation approximate any continuous function on an interval? In the smooth non-polynomial case, difference quotients and derivatives bring monomials into the closed span; Weierstrass then brings in every continuous univariate target. For nonsmooth activations, convolution with smooth test functions moves the problem to the smooth case. If every convolution were polynomial with bounded degree, distribution theory would force the original activation to be polynomial almost everywhere, which contradicts the hypothesis.

The threshold is essential, not a cosmetic bias term. Without shifts, `sin(w x)` spans odd functions on a symmetric interval and cannot approximate an even function such as `cos(x)`. Adding a threshold gives translations and escapes that symmetry trap.

So the insight is that shallow networks are dense because their affine activation copies can separate measures, and the activation frontier says the thresholded span fails exactly when the activation is polynomial almost everywhere. The theorem removes representability as the basic obstruction, but it leaves width, conditioning, optimization, data, and inductive bias untouched.
