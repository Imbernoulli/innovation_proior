I want to decide, for a given sequence of reals $\alpha_1, \alpha_2, \alpha_3, \dots$, whether its fractional parts $\{\alpha_n\} = \alpha_n - \lfloor \alpha_n \rfloor$ spread evenly around the circle of circumference 1: whether for every subinterval $a = [b,c) \subseteq [0,1)$ the count $n_a$ of the first $n$ points landing in $a$ satisfies $n_a/n \to |a| = c-b$. A sequence with this property is *uniformly distributed (equidistributed) mod 1*. The trouble is that the definition is a brutal object to attack directly: it quantifies over *all* subintervals at once, and an interval indicator $\chi_a$ is a rigid, discontinuous step function ‚Äî nothing one can differentiate, expand, or sum in closed form. For the one family I genuinely understand, the linear sequence $\{n\xi\}$ with $\xi$ irrational, the known proofs (Bohl, Sierpi≈Ñski, Weyl around 1909‚Äì1910) are elementary interval-chasing: take two equal arcs, show their counts differ by a bounded amount, conclude each gets its fair share. That works only because successive points $n\xi$ and $(n{+}1)\xi$ differ by the constant step $\xi$ ‚Äî the whole argument lives off the additive structure of the arithmetic progression. The moment the points are $\{n^2\xi\}$ the increments themselves change with $n$ and the two-interval comparison has nothing to grip; Hardy and Littlewood's 1912/1914 quadratic proof rests on a Cauchy integral built around degree 2 and offers no route to higher degree; and Kronecker's 1884 theorem delivers only *density* on the torus ‚Äî that the orbit $(n\xi_1,\dots,n\xi_p)$ comes arbitrarily close to every point ‚Äî never the *frequency* with which a region is visited. What is missing is not another family-specific trick but one checkable condition that *is* equidistribution, applicable to any sequence by a single mechanism.

The first move is to loosen the definition, since indicators are the obstruction. Counting points in arcs is exactly averaging indicator functions, $n_a/n = \frac1n \sum_{h\le n} \chi_a(\alpha_h)$, with target $|a| = \int_0^1 \chi_a$. So "$n_a/n \to |a|$ for every arc" says "$\frac1n \sum_h f(\alpha_h) \to \int_0^1 f$ for every interval indicator $f$" ‚Äî and this widens, with no loss, to *every* bounded 1-periodic Riemann-integrable $f$. The equivalence is a sandwich: any Riemann-integrable $f$ is, by the very definition of the integral, trapped between two step functions $f_1 \le f \le f_2$ with $\int f_2 - \int f_1 < \varepsilon$; a step function is a finite combination of interval indicators, so the averaging law holds for $f_1, f_2$ by the arc statement, and monotone averaging squeezes it onto $f$ within $\varepsilon$. This buys freedom: I am no longer chained to indicators, I may test against *any* convenient family of functions, provided the family is rich enough that controlling it controls every Riemann-integrable $f$ by approximation.

Which family? I want functions that behave well under the dynamics ‚Äî which here is "add a real number mod 1" ‚Äî and that are dense enough to approximate everything. The functions that diagonalize translation are the exponentials. Write $e(x) := e^{2\pi i x}$; then $e(k(x+t)) = e(kx)\,e(kt)$, so a translation merely multiplies by a constant. These characters are the genuine analytic invariants of a residue class mod 1, their integrals are the cleanest possible ‚Äî $\int_0^1 e(mx)\,dx = 1$ if $m=0$ and $0$ if $m \ne 0$ ‚Äî and by Weierstrass/Fej√©r every continuous periodic function is a uniform limit of finite trigonometric polynomials, i.e. of finite combinations of the $e(kx)$, so they are Fourier-complete. Running the averaging law against $f = e(mx)$ for $m \ne 0$ has target $\int_0^1 e(mx)\,dx = 0$, so it reads $\frac1N \sum_{n=1}^N e(m\alpha_n) \to 0$. Assuming exactly this for every $m \ne 0$ recovers the full law: by linearity it holds for every trig polynomial (the $j=0$ term gives $\int f$, every $j \ne 0$ term vanishes), by uniform approximation and squeezing it holds for every continuous periodic $f$, and by trapping jumps between steep continuous ramps it holds for every Riemann-integrable $f$, in particular for arc indicators ‚Äî which is equidistribution.

The method I propose is **Weyl's equidistribution theorem**, built on **Weyl's criterion** together with the technique I call **Weyl differencing**. The criterion states that for a sequence $(\alpha_n)$ the following are equivalent: (1) $(\alpha_n)$ is uniformly distributed mod 1; (2) $\frac1N \sum_{n=1}^N f(\alpha_n) \to \int_0^1 f$ for every Riemann-integrable 1-periodic $f$; and (3) for every integer $m \ne 0$,
$$\frac1N \sum_{n=1}^N e(m\alpha_n) \to 0.$$
This is the whole game changed: a geometric condition on infinitely many arcs has become an analytic condition on a single kind of exponential sum $S_N(m) = \sum_{n=1}^N e(m\alpha_n)$ ‚Äî an object I can attack with closed forms, with Cauchy‚ÄìSchwarz, with algebra.

On the linear case the criterion is immediate and not vacuous. Fix $m \ne 0$ and set $\eta = m\xi$, which is non-integer because $\xi$ is irrational; the sum is literally geometric,
$$\sum_{n=1}^N e(n\eta) = \frac{e((N{+}1)\eta) - e(\eta)}{e(\eta) - 1},$$
whose numerator has modulus at most 2, so $|\sum_{n=1}^N e(n\eta)| \le 2/|e(\eta)-1| = 1/|\sin \pi\eta|$ (using $|e^{2\pi i \eta} - 1| = 2|\sin \pi\eta|$). This is a *constant independent of $N$* ‚Äî not merely $o(N)$ but $O(1)$, the partial sums never grow ‚Äî so criterion (3) holds and $\{n\xi\}$ is equidistributed, with no recourse to additive geometry. The same characters live on the torus: the criterion becomes $\sum_{n=1}^N e(m \cdot \alpha(n)) = o(N)$ for every nonzero integer vector $m$, and for $\alpha(n) = (n\xi_1,\dots,n\xi_p)$ with no nontrivial integer relation $\sum l_i \xi_i \in \mathbb{Z}$ the inner product $m\cdot\alpha(n) = n\eta$ with $\eta = \sum m_i\xi_i \notin \mathbb{Z}$, so the same geometric series gives $O(1)$. This is strictly stronger than Kronecker: equidistribution forces density, and adds the frequency of visits Kronecker never saw. (If $\xi = a/b$ is rational, take $m=b$: then $\eta \in \mathbb{Z}$, the denominator vanishes, the sum equals $N$, and indeed $\{n\xi\}$ cycles through finitely many residues ‚Äî the criterion sees exactly the right dividing line.)

The real wall is the polynomial case: $\alpha_h = \varphi(h)$ with $\varphi(z) = \alpha_q z^q + \dots + \alpha_1 z + \alpha_0$ and some non-constant coefficient irrational. The criterion demands $\sigma_N := \sum_{h=0}^N e(\varphi(h)) = o(N)$, but the geometric move fails: $e(\varphi(h{+}1))/e(\varphi(h)) = e(\varphi(h{+}1) - \varphi(h))$ is not a constant ratio, because $\varphi(h{+}1) - \varphi(h)$ is itself a polynomial in $h$ of degree $q-1$. There is no antiderivative trick as in the continuous case. What I *can* do to a sum I cannot close is square its modulus and watch the phase: $|\sigma_N|^2 = \sum_{h,k} e(\varphi(h) - \varphi(k))$, and putting $h = k+r$ gives $\varphi(k{+}r) - \varphi(k) = r\,\varphi(r,k)$ with $\varphi(r,k)$ of degree $q-1$ in $k$, leading term $q\alpha_q k^{q-1}$ (the constant $\alpha_0$ cancels in the difference). Differencing dropped the degree by exactly one. That is the lever ‚Äî and Cauchy‚ÄìSchwarz lets me iterate it, isolating the inner sum over $k$ while the new outer variable $r$ comes along:
$$|\sigma_N|^4 \le n_1 \sum_r \Big| \sum_k e(r\varphi(r,k)) \Big|^2 = n_1 \sum_{r,s}\sum_l e(rs\,\varphi(r,s,l)),$$
where $k = l+s$ and $\varphi(r,s,l)$ has degree $q-2$. Each round costs one Cauchy‚ÄìSchwarz (a counting factor $n_i$) and doubles the exponent on $|\sigma_N|$ ‚Äî $2, 4, 8, \dots$ ‚Äî but lowers the inner degree by one, so after $q-1$ rounds the inner phase in the running index $h$ is linear,
$$\varphi(r_1,\dots,r_{q-1}, h) = q!\,\alpha_q \cdot h + (\beta_0 + \beta_1 r_1 + \dots + \beta_{q-1} r_{q-1}).$$
The leading coefficient is $q!\,\alpha_q$ because each step multiplied it by the descending factor $q, q{-}1, \dots, 2$; *and because $\alpha_q$ is irrational, $q!\,\alpha_q$ is irrational*, so the inner sum over $h$ is precisely the linear case already cracked. Writing $Q = 2^{q-1}$, $\xi = q!\,\alpha_q$, $R = r_1\cdots r_{q-1}$, $\varrho = R(\beta_0 + \sum \beta_i r_i)$, and accumulating the counting factors into $N \sim \kappa\, n^{Q-q}$, the inequality is $|\sigma_N|^Q \le N \sum_{r' : |r'|\le n} |\,e(\varrho)\sum_h e(R\xi h)\,|$, with $r'$ ranging over the octahedron $|r'| = |r_1| + \dots + |r_{q-1}| \le n$.

The genuinely hard estimate is not the differencing ‚Äî that is bookkeeping ‚Äî but controlling *how many* $r'$ are bad, i.e. make the multiplicatively-coupled frequency $R\xi = r_1\cdots r_{q-1}\,q!\alpha_q$ land near an integer, where $1/|\sin \pi(R\xi)|$ blows up. I isolate this as a lattice-point counting lemma: for $\xi$ irrational, $\sum_{|r|\le n} e(r_1 r_2 \cdots r_q\,\xi) = o(n_q)$ where $n_q \sim (2n)^q/q!$, and in particular the number of $r'$ in the $(q{-}1)$-dimensional octahedron with $r_1\cdots r_{q-1}\xi \bmod 1 \in (-\varepsilon,\varepsilon)$ is $< 3\varepsilon\, n_{q-1}$ for large $n$. The lemma is proved by induction on $q$ using the same square-bounded geometric sum, peeling off the last variable and splitting the rest into good and bad $r'$, the bad count being supplied by the induction hypothesis one dimension down. Feeding the corollary back in ‚Äî good $r'$ (at most $n_{q-1}$) contribute $\le 1/|\sin \pi\varepsilon|$ each, bad $r'$ (count $< 3\varepsilon\, n_{q-1}$) get the crude $\le n+1$ ‚Äî gives $|\sigma_N|^Q \le N\cdot n_{q-1}\{3\varepsilon(n{+}1) + 1/|\sin\pi\varepsilon|\}$, and dividing by $n^Q$ yields $|\sigma_N/N|^Q \le 3\varepsilon(\kappa\cdot 2^{q-1}/(q{-}1)! + 1)$ for $N$ large; $\varepsilon$ arbitrary forces $\sigma_N = o(N)$. One last case: if the *leading* coefficient is rational but $\alpha_l$ irrational lower down, I manufacture an irrational leading coefficient by splitting the index into residue classes $Gh + r$ modulo a common denominator $G$ of $\alpha_q,\dots,\alpha_{l+1}$ ‚Äî the cleared top coefficients contribute integers and vanish mod 1, leaving each of the $G$ inner sums a degree-$l$ polynomial with irrational leading coefficient $\alpha_l G^l$, hence $o(n)$. So $\varphi$ is equidistributed whenever *any* non-constant coefficient is irrational. By Abel summation the bound extends to weighted sums: $\sum a_h e(\varphi(h)) = o(\sum a_h)$ for positive decreasing $a_h$ with divergent sum, e.g. $\sum_{h\le n} e(\varphi(h))/h = o(\log n)$ ‚Äî the kind of sum that controls $\zeta$ on the line $\mathrm{Re}\,s = 1$, the original analytic motive.

The proofs are the artifact. The code below only exhibits the analytic facts they establish ‚Äî that linear sums stay bounded by $1/|\sin \pi m\xi|$ while polynomial sums decay ‚Äî and is not itself the proof.

```python
import cmath, math

def e(x):
    return cmath.exp(2j * math.pi * x)

def char_partial_sum(seq, m, N):
    "S_N(m) = sum_{n=1}^N e(m * seq(n))"
    return sum(e(m * seq(n)) for n in range(1, N + 1))

def star_discrepancy(seq, N, grid=1000):
    pts = sorted((seq(n) % 1.0) for n in range(1, N + 1))
    d, j = 0.0, 0
    for i in range(grid + 1):
        c = i / grid
        while j < len(pts) and pts[j] < c:
            j += 1
        d = max(d, abs(j / N - c))
    return d

alpha = math.sqrt(2)

# Linear: |S_N(m)| stays <= 1/|sin(pi m alpha)|, uniformly in N (stronger than o(N)).
for m in (1, 2, 3):
    bound = 1.0 / abs(math.sin(math.pi * m * alpha))
    sup_N = max(abs(char_partial_sum(lambda n: n * alpha, m, N)) for N in range(1, 4000))
    print(f"linear m={m}: sup_N |S_N| = {sup_N:.3f},  1/|sin(pi m a)| = {bound:.3f}")

# Quadratic n^2 alpha: no geometric series, but |S_N|/N -> 0 and discrepancy -> 0.
for N in (1000, 4000, 16000):
    s = char_partial_sum(lambda n: (n * n) * alpha, 1, N)
    print(f"quadratic N={N}: |S_N|/N = {abs(s)/N:.5f},  D*_N = "
          f"{star_discrepancy(lambda n: (n*n)*alpha, N):.5f}")
```

Running it, the linear suprema match $1/|\sin \pi m\alpha|$ exactly ‚Äî the sums never grow ‚Äî while for $n^2\alpha$ both $|S_N|/N$ and the star-discrepancy shrink toward 0 as $N$ grows, the quantitative shadow of the equidistribution proved above.
