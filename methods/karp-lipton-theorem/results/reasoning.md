Karp-Lipton starts from a deliberately weak-looking hypothesis: `NP subseteq P/poly`. This does not say that a polynomial-time algorithm can build circuits for `SAT`; it only says that for each input length there exists a small circuit that gets all instances of that length right. In advice language, the algorithm may receive a polynomial-length string depending on the length, not on the particular input. That is why the assumption is nonuniform.

The theorem's force comes from showing that even this nonuniform assumption has uniform consequences. A circuit family for `SAT` can be used as more than a yes/no oracle. Because `SAT` is self-reducible, a decision procedure for satisfiability can be used to find a satisfying assignment: set one variable to 0 or 1, keep the branch that remains satisfiable, and repeat. Thus a small decision circuit can be packaged into a small circuit that outputs witnesses whenever witnesses exist.

This matters for the polynomial hierarchy because the second level contains statements like `forall x exists y R(z, x, y)`. The existential `y` may depend on the universal `x`, so at first glance a `Sigma_2` simulation cannot simply put all witnesses at the front. Karp-Lipton observes that a single guessed circuit can stand in for the whole function mapping each `x` to a valid `y`. The existential quantifier guesses the circuit; the universal quantifier checks it against every `x`.

The soundness is the key point. If the original `Pi_2` statement is true, then under `NP subseteq P/poly` there is a correct small witness-producing circuit for all relevant SAT instances, so the existential guess can choose it. If the statement is false, then there is some `x` for which no `y` satisfies `R(z, x, y)`. No guessed circuit can pass the universal check on that `x`, no matter how it is wired. This converts the `forall exists` pattern into an `exists forall` pattern.

Once `Pi_2^p` is contained in `Sigma_2^p`, the second level closes under complement in the relevant way and the usual hierarchy-collapse argument gives `PH = Sigma_2^p` in the standard modern statement. The same proof idea is often described as guessing the advice and using the remaining quantifier power to verify that the guessed advice behaves well enough for the simulation.

This is not a circuit lower bound. It does not prove `SAT notin P/poly`, and it does not directly separate `P` from `NP`. Its contribution is conditional structure: if small circuits for `NP` existed, they would be so powerful that they would compress not only individual computations but also whole families of existential witnesses across universal choices. Since such a polynomial-hierarchy collapse is considered unlikely, the theorem gives a precise reason to view `NP subseteq P/poly` as an implausibly strong nonuniform assumption.

The distinctive conceptual lesson is that nonuniformity is not harmless preprocessing. Advice and circuits can be guessed inside higher-level uniform computations, and self-reducibility lets those circuits supply witnesses rather than just decisions. Karp-Lipton exposes the pressure this puts on alternation: small circuits for `NP` would make the hierarchy lose depth.
