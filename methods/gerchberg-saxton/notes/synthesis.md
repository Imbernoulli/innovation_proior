## The pain point / research question (in-frame)
Detectors record intensity, not phase. Given the image-plane amplitude |f| and the diffraction(Fourier)-plane amplitude |F| of the SAME wave (an electron-microscopy pair), recover the lost phase so the full complex wave — hence the specimen — can be reconstructed. Equivalently (synthesis direction): given a fixed illumination amplitude on a phase-only modulator (constant |A| per pixel — a kinoform/SLM) and a desired Fourier-plane intensity (a target image or an array of focal spots), find the per-pixel phase φ_j whose Fourier transform has the target amplitude. The hard constraint is **phase-only**: the SLM cannot touch amplitude, only phase, so you can't just write the inverse FFT of the target.

## Discovery-order skeleton for reasoning.md
1. Phase problem: |f| and |F| measured, phase lost. F = FT(f). If I knew the phase I'd be done. Brute force over phases is hopeless.
2. Reframe as feasibility: I know two facts — modulus in object domain (=|f|) and modulus in Fourier domain (=|F|), tied by an FFT. Define two sets: M_o = {g : |g|=|f|}, M_F = {g : |FT g|=|F|}. Want a point in M_o ∩ M_F.
3. Each set is a "fix-the-modulus, free-the-phase" set. Nearest point on such a set to a current estimate: keep the phase, reset modulus → P(x)=|target|·x/|x|. (Derive: minimize |x−ae^{iθ}|² over a=|target| fixed ⇒ θ=arg x.)
4. Alternating projection drops out: FFT, project onto |F| (keep phase ψ, set mod |F|), IFFT, project onto |f| (keep phase φ, set mod |f|), repeat. These are exactly Fienup Eqs.(6)–(9) / Wikipedia B,C,D,A.
5. Prove error non-increasing (the Thimons/Fienup proof, worked fully): Parseval makes the Fourier-domain residual equal an object-domain distance; the modulus-reset is the nearest feasible point, so each projection can't increase the distance ⇒ E_{k+1} ≤ e_k ≤ E_k. Worked via cos(θ)≤1 expansion.
6. Wall: monotone ↓ but the sets are non-convex (fixed-modulus circles), so the iteration converges to a fixed point that need not be a true intersection — stagnation / plateaus (Fienup Fig.2, the stripe artifact; Thimons centrosymmetric failure). Also the phase-only synthesis target may not be exactly reachable at all.
7. Synthesis specialization: object domain = SLM, |f| → fixed illumination amplitude (uniform or Gaussian beam); Fourier domain target = desired image / array of M spots. For a spot array the natural target amplitude is uniform over the spots. Run GS.
8. Second wall (the spot-array-specific one): naive GS with equal target amplitudes does NOT give equal delivered spot intensities — phase-only can't split power evenly; some spots come out bright, some dim; uniformity poor, ghost traps. (Bianchi/Di Leonardo: SR gives u of a few %.)
9. The weighting fix (derive from the goal of uniform efficiency): I don't actually care that the *target amplitudes* are equal; I care that the *delivered* |Vm| are equal. So treat the target amplitudes as free weights wm and adapt them: where a spot comes out too dim (|Vm| < ⟨|V|⟩) push its weight up, too bright push it down. Multiplicative feedback wm ← wm·⟨|Vm|⟩/|Vm| is the fixed-point-preserving choice (at convergence |Vm|=⟨|V|⟩ ⇒ wm unchanged). Equivalent intensity form wm ← wm·√(Ĩ/Im). Keep the GS phase θm=arg Vm each step.
10. Land on code: numpy FFT GS loop (kurokuman_gs) then the one-line weight update making it WGS (kurokuman_wgs); note the spot-array/Fourier-optics direct-sum version (ppozzi) and metrics e,u.

## Design decisions → why (the table)
- **Keep computed phase, reset only modulus** at each constraint: because that is the *orthogonal projection* / nearest feasible point on the fixed-modulus set; any other amplitude assignment moves farther, breaking the monotone-error guarantee. Derived from min |x−ae^{iθ}|².
- **Why two FFTs per iteration / alternate domains**: the two constraints live in Fourier-conjugate domains; an FFT is the only coupling, so you must hop domains to apply both.
- **Phase-only / constant illumination amplitude** (|A| fixed): it's a *hardware* constraint of a kinoform/SLM (modulates phase, not amplitude) — this is what makes the problem non-trivial; if you could set amplitude you'd just inverse-FFT the target.
- **Random (or constant) initial phase**: you must start somewhere; random avoids the centrosymmetric-input degeneracy where constant phase makes DFT/IDFT never change the estimate (Thimons §7). Diversity also lets you restart to escape a bad local minimum (Fienup §VII).
- **Error metric = Σ(|computed|−|target|)²** (×√N in Fourier domain for Parseval-comparability): it's exactly the squared distance to the constraint set, the thing the projections provably shrink.
- **Why GS stagnates**: non-convex modulus sets ⇒ alternating projection has spurious fixed points; ER = steepest descent with a fixed (double-length) step, no momentum/feedback, so it sits in shallow local minima (Fienup §III, §V); HIO adds feedback to escape (context only).
- **Why uniform target → non-uniform spots**: phase-only modulation has limited degrees of freedom; forcing equal target amplitudes over a structured array drives interference that the phase mask can't perfectly satisfy, so delivered intensities scatter and ghost orders appear (Bianchi/Di Leonardo).
- **WGS weight update form wm·⟨|Vm|⟩/|Vm|** (multiplicative, ratio to the mean): chosen so the *uniform* state |Vm|=⟨|V|⟩ is a fixed point (weights stop moving exactly when uniform); multiplicative so weights stay positive; ratio-to-mean so total power (efficiency) is roughly preserved while only the *spread* is corrected. sqrt appears iff you write it in intensity Im=|Vm|² (√(Ĩ/Im)); amplitude form has no sqrt.
- **Efficiency e and uniformity u**: e=Σ Im fraction of power into wanted spots (want high); u=1−(Imax−Imin)/(Imax+Imin) → 1 when all equal. Standard yardsticks.

## Exact formulas to use (verified, with signs)
- Projection onto |target| set: P(x) = |target| · x/|x| = |target|·exp(i·arg x).
- GS iteration (Fienup 6–9 / Thimons 4–7): x_k=|f|e^{iφ_{k-1}}; X_k=FFT(x_k); Y_k=|F|·X_k/|X_k|; y_k=IFFT(Y_k); φ_k=arg y_k.
- Convergence: E_{k+1} ≤ e_k ≤ E_k (Thimons Thm 1; Fienup Eq.22 E_F,k+1 ≤ E_O,k ≤ E_F,k).
- Field at trap m: Vm=(1/N)Σ_j exp(i(φ_j−Δ_j^m)); Im=|Vm|².
- WGS weight update: wm^(k)=wm^(k−1)·⟨|Vm|⟩_m / |Vm|  (≡ wm·√(Ĩ/Im) for uniform target). φ_j=arg Σ_m wm·exp(i(Δ_j^m+θm)), θm=arg Vm.
- Metrics: e=Σ_m Im; u=1−(max Im−min Im)/(max Im+min Im).
