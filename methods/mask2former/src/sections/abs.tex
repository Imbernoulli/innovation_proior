\begin{abstract}

Image segmentation groups %
pixels with different semantics, \eg, category or instance membership. %
Each choice of semantics defines a task. %
While only the semantics of each task differ,
current research focuses on designing specialized architectures for each task.
We present \modelnamelong (\modelname), a new architecture capable of addressing any image segmentation task (panoptic, instance or semantic).
Its key components include \emph{masked attention}, which extracts localized features by constraining cross-attention within predicted mask regions. %
In addition to reducing the research effort by at least three times, it outperforms the best specialized architectures by a significant margin on four popular datasets.
Most notably, \modelname sets a new state-of-the-art for panoptic segmentation (57.8 PQ on COCO), instance segmentation (50.1 AP on COCO) and semantic segmentation (57.7 mIoU on ADE20K).
\end{abstract}
