\begin{abstract}


We present \drqs{}, a model-free reinforcement learning (RL) algorithm for visual continuous control. \drqs{} builds on DrQ, an off-policy actor-critic approach that uses data augmentation to learn directly from pixels. We introduce several improvements that yield state-of-the-art results on the DeepMind Control Suite. Notably, \drqs{} is able to solve complex humanoid locomotion tasks directly from pixel observations, previously unattained by model-free RL. \drqs{} is conceptually simple, easy to implement, and provides significantly better computational footprint compared to prior work, with the majority of tasks taking just $8$ hours to train on a single GPU. Finally, we publicly release~\drqs{}'s implementation to  provide RL practitioners with a strong and computationally efficient baseline.
\end{abstract}
