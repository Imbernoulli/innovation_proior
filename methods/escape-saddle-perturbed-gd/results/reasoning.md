Let me start from the thing that actually bothers me. Gradient descent is the workhorse precisely because of one fact: for an $\ell$-smooth function it reaches a point with $\|\nabla f(\mathbf{x})\|\le\epsilon$ in about $\ell(f(\mathbf{x}_0)-f^\star)/\epsilon^2$ steps, and that count has no $d$ in it. Dimension-free. That is the whole reason it scales to the enormous problems people throw at it. I can rederive it in one line: $\ell$-smoothness gives $f(\mathbf{x}_{t+1})\le f(\mathbf{x}_t)+\nabla f(\mathbf{x}_t)^\top(\mathbf{x}_{t+1}-\mathbf{x}_t)+\tfrac{\ell}{2}\|\mathbf{x}_{t+1}-\mathbf{x}_t\|^2$, and with $\mathbf{x}_{t+1}-\mathbf{x}_t=-\eta\nabla f$ that is $f(\mathbf{x}_t)-\eta\|\nabla f\|^2+\tfrac{\eta^2\ell}{2}\|\nabla f\|^2$. Take $\eta\le 1/\ell$ so $\tfrac{\eta^2\ell}{2}\le\tfrac{\eta}{2}$, and the decrease per step is at least $\tfrac{\eta}{2}\|\nabla f\|^2$. As long as the gradient is bigger than $\epsilon$ I am draining function value at a fixed rate, and $f$ can only fall by $f(\mathbf{x}_0)-f^\star$ total, so I terminate in $O(\ell(f_0-f^\star)/\epsilon^2)$ steps. Clean, and nowhere did $d$ appear.

Here is the catch. In a nonconvex landscape, a point with zero gradient is not a victory. It can be a local minimum, but it can equally be a saddle or a local maximum. For the problems I care about — tensor decomposition, dictionary learning, phase retrieval, matrix sensing and completion, even some neural nets — there is a striking structural fact: every local minimum is global. So I do not actually need the global minimum; I need *a* local minimum. The thing standing between me and a local minimum is therefore not bad local minima at all. It is saddle points. And the diagnostic picture from the high-dimensional landscape literature (Dauphin and colleagues, 2014) sharpens this: in high dimensions the critical points proliferate, and they are overwhelmingly saddles, not poor minima. Saddles are the bottleneck. So the real target is: get past saddles, using only gradients, and do it without paying in dimension.

Why does plain gradient descent struggle at a saddle? Because at a saddle $\nabla f=0$ exactly, so the update $\mathbf{x}-\eta\nabla f$ does nothing. The first-order method is blind there — its only sensor reads zero, and zero is also what it reads at a minimum. It cannot tell the two apart. So I need to upgrade what I am aiming for. A first-order stationary point is too weak a target. Let me ask for a *second-order* stationary point: $\nabla f(\mathbf{x})=0$ and $\lambda_{\min}(\nabla^2 f(\mathbf{x}))\ge0$. If I additionally know every saddle is *strict* — meaning at every saddle $\lambda_{\min}(\nabla^2 f)<0$, there is an honest direction of negative curvature — then a second-order stationary point cannot be a saddle, because the saddle's negative eigenvalue violates $\lambda_{\min}\ge0$. Let me make sure that's not vacuous on a concrete case. Take the canonical two-dimensional saddle $f(\mathbf{x})=\tfrac12 x_1^2-\tfrac12 x_2^2$: at the origin $\nabla f=0$, so it passes the first-order test, but $\nabla^2 f=\mathrm{diag}(1,-1)$ has $\lambda_{\min}=-1<0$, so it fails the second-order test exactly along $x_2$ — and that is the descent direction a minimizer should follow. Good: the second-order condition really does reject this saddle while a local minimum like $f=\tfrac12(x_1^2+x_2^2)$ (Hessian $\mathrm{diag}(1,1)$, $\lambda_{\min}=1>0$) passes. So under the strict-saddle property, the only points surviving both tests are local minima, and "find a second-order stationary point" *is* "find a local minimum." That is the right target.

I need an approximate version, because I will never hit it exactly. The gradient part is obvious: $\|\nabla f(\mathbf{x})\|\le\epsilon$. The curvature part needs a tolerance too: $\lambda_{\min}(\nabla^2 f(\mathbf{x}))\ge-\,?$. What is the right scale for that question mark? Let me think in units. If the Hessian is $\rho$-Lipschitz, then curvature has units of $\rho\times$length, and the gradient has units of curvature$\times$length $=\rho\times$length$^2$. So if my gradient tolerance is $\epsilon$, the length scale it implies is $\sqrt{\epsilon/\rho}$, and the curvature at that scale is $\rho\cdot\sqrt{\epsilon/\rho}=\sqrt{\rho\epsilon}$. That is the only curvature tolerance with the right units to pair with $\epsilon$. So I will call $\mathbf{x}$ an $\epsilon$-second-order stationary point when $\|\nabla f(\mathbf{x})\|\le\epsilon$ and $\lambda_{\min}(\nabla^2 f(\mathbf{x}))\ge-\sqrt{\rho\epsilon}$. This is the Nesterov–Polyak pairing, and I like it for an extra reason: the gradient requirement is *literally identical* to the first-order target, so any comparison of my eventual rate against the dimension-free first-order rate is apples to apples.

Now, who can already do this, and at what cost? Let me lay out the tools on the table honestly, because the gap each one leaves is what tells me where to push.

The classical answer is to use curvature directly. Cubic-regularized Newton (Nesterov–Polyak) minimizes $f(\mathbf{x}_t)+\nabla f^\top(\mathbf{x}-\mathbf{x}_t)+\tfrac12(\mathbf{x}-\mathbf{x}_t)^\top\nabla^2 f\,(\mathbf{x}-\mathbf{x}_t)+\tfrac{\rho}{6}\|\mathbf{x}-\mathbf{x}_t\|^3$ each step. The cubic term penalizes long steps just enough that the subproblem "knows" about negative curvature and walks downhill along it; it reaches an $\epsilon$-second-order stationary point in $O(1/\epsilon^{1.5})$ iterations, which is genuinely fast in $\epsilon$. Trust-region methods get the same. But each iteration needs the full Hessian and an eigen/inverse subproblem — quadratic-to-cubic per-step cost in $d$, the opposite of what makes gradient descent usable. The Hessian-vector-product line (Agarwal and colleagues; Carmon and colleagues, around 2016) softens this: with only the oracle $\mathbf{u}\mapsto\nabla^2 f(\mathbf{x})\mathbf{u}$ they get $O(\log d/\epsilon^{7/4})$, or $O(\log d/\epsilon^2)$ by solving the cubic subproblem with gradient descent. Better, but still second-order access; the curvature is still doing the escaping. I want the bare gradient.

So the gradient-only line is where I should be looking, and it has three entries, each instructive.

Ge, Huang, Jin and Yuan (2015) is the closest ancestor and the one I will lean on. Their move: inject isotropic noise at *every* step, $\mathbf{w}_{t+1}=\mathbf{w}_t-\eta(\widehat\nabla f(\mathbf{w}_t)+\mathbf{n})$ with $\mathbf{n}$ uniform on the sphere. The reasoning is that near a saddle the only thing stopping escape is the iterate having no component along the negative-curvature direction; constant noise guarantees nonnegligible variance in *every* direction at *every* step, so the iterate always has something to amplify. Under the robust strict-saddle property they prove polynomial convergence to a local minimum. But here is the wound: to make their analysis go through they need the noise ball and the step size to be tiny — scaling like $1/d$ — so that over the whole noise ball the function looks like its quadratic model. Tiny steps means many steps. Their iteration count is $\mathrm{poly}(d/\epsilon)$ with the polynomial of order at least four; at least $\Omega(d^4)$. Compared to dimension-free, $d^4$ is a disaster. Levy (2016) trims this to $O(d^3\cdot\mathrm{poly}(1/\epsilon))$ with normalized gradients — still a power of $d$.

And then there is Lee, Simchowitz, Jordan and Recht (2016), which is the cleanest and also, in a way, the most frustrating. No added noise at all: just gradient descent with constant step $\alpha<1/\ell$ from a random start. They show, via the Stable Manifold Theorem, that it avoids every strict saddle *almost surely*. The argument is beautiful — the gradient map $g(\mathbf{x})=\mathbf{x}-\alpha\nabla f(\mathbf{x})$ is a diffeomorphism for $\alpha<1/\ell$ (it is injective because $\|g(\mathbf{x})-g(\mathbf{y})\|\ge\|\mathbf{x}-\mathbf{y}\|-\alpha\|\nabla f(\mathbf{x})-\nabla f(\mathbf{y})\|\ge(1-\alpha\ell)\|\mathbf{x}-\mathbf{y}\|$, and surjective by inverting via the proximal map of $-f$), the local stable manifold of a strict saddle has positive codimension hence measure zero, and a diffeomorphism's iterated preimages of a measure-zero set are measure zero — so the set of starts that converge to the saddle is null. Lovely. But it is purely asymptotic: it says the *limit* is not a saddle, with probability one. It bounds *no number of steps*. You can string together a chain of saddles so a single random start crawls past them as slowly as you like. There is no rate.

Let me extract the one mechanical picture they all share, because it is the engine. Near a strict saddle, write the Hessian there as $\mathbf{H}$ and look at the gradient step on the local quadratic: $\mathbf{x}_{t+1}=(\mathbf{I}-\eta\mathbf{H})\mathbf{x}_t$, so in the eigenbasis $\langle\mathbf{e}_i,\mathbf{x}_{t+1}\rangle=(1-\eta\lambda_i)\langle\mathbf{e}_i,\mathbf{x}_t\rangle$. With $\eta<1/\ell$, every positive-curvature direction has $|1-\eta\lambda_i|<1$ and *contracts*; every negative-curvature direction has $1-\eta\lambda_i>1$ and *expands*. Gradient descent near a saddle is power iteration on $\mathbf{I}-\eta\mathbf{H}$: it geometrically blows up the most-negative-eigenvalue direction — as long as the iterate has a nonzero component there. The whole problem is reduced to: get a nonzero component along the escape direction.

Before I build a whole algorithm on this picture, I want to actually watch it happen, because "one nonzero component, then deterministic descent does the rest" is the load-bearing assumption and it would be embarrassing to find it needs help. Let me take a concrete strict saddle that is bounded below so there is a real minimum to reach: $f(\mathbf{x})=-\tfrac12\gamma x_1^2+\tfrac14 x_1^4+\tfrac12\sum_{i\ge2}\lambda_i x_i^2$ with $\gamma=1$ and the $\lambda_i$ positive. At the origin $\nabla f=0$ and $\nabla^2 f=\mathrm{diag}(-1,\lambda_2,\dots)$, a strict saddle with escape direction $\mathbf{e}_1$; the true minima sit at $x_1=\pm\sqrt\gamma=\pm1$, all other coordinates zero, with $f=-\tfrac14\gamma^2=-0.25$. Start gradient descent *exactly at the origin*: the gradient is identically zero, so plain GD never moves — I confirmed it stays put, which is the failure I expect. Now run the perturb-once-then-descend loop (the code below) from the origin. In $d=2$ it leaves with $f$ dropping from $0$ to $-0.2500$ and lands at $x_1=+1.0000$; in $d=10$ it lands at $x_1=-1.0000$, same $f=-0.2500$ — and the *sign flips between runs*, which is the tell that the escape direction was set by the random kick, not by anything deterministic in the dynamics. A single perturbation, of size $r\ll1$, was enough; the geometric amplification of the $\mathbf{e}_1$ component carried it the rest of the way to a genuine minimum. So the engine runs as advertised, and it really does only need one component planted along the escape direction. That is what the noise in Ge and the random start in Lee are both buying — and it is what tells me I might get away with buying it just *once* per saddle.

Stare at the tension between Ge and Lee for a moment, because the answer is hiding in the gap between them. Ge adds noise *constantly* and pays $d^4$ because constant noise forces tiny steps. Lee adds noise *once* (the random start) and keeps the natural step $\Theta(1/\ell)$ but gets no rate, because once is not enough when there is a whole chain of saddles to clear. What I want is the best of both: keep the natural large step $\eta=\Theta(1/\ell)$ — that is non-negotiable, it is the only way to stay dimension-free, since shrinking $\eta$ to $1/d$ is exactly how $d$ creeps back in — and add randomness, but *only when and where I need it*. I do not need variance in every direction at every step. I need a fresh kick along the escape direction each time I find myself stuck at a new saddle. So: run ordinary gradient descent; whenever the gradient gets small (the signal that I might be at a saddle), add *one* perturbation; then let deterministic gradient descent's own power-iteration amplification do the escaping; and don't perturb again until enough steps have passed to either escape or confirm I'm at a genuine minimum.

Let me try to turn that into a concrete loop. Run $\mathbf{x}_{t+1}=\mathbf{x}_t-\eta\nabla f(\mathbf{x}_t)$. When $\|\nabla f(\mathbf{x}_t)\|\le g_{\mathrm{thres}}$ and I haven't perturbed recently, snapshot the current point as $\tilde{\mathbf{x}}_t$, add $\xi_t$ drawn uniformly from a ball of radius $r$, and keep running plain gradient descent for $t_{\mathrm{thres}}$ steps. After those steps, check: did the function value fall by at least $f_{\mathrm{thres}}$? If yes, I escaped — carry on. If no, I'll *want* to conclude I was at an honest second-order stationary point and output $\tilde{\mathbf{x}}_t$ — but I should be careful here, because that conclusion is exactly the thing I have not earned yet: it only holds if "perturb-and-descend fails to drop $f$" really implies "no negative curvature." That is a claim about the escape succeeding *whenever* there is negative curvature to exploit, and I owe a proof of it before I can trust the stopping rule. So let me treat it as a conjecture pinned to the algorithm and come back to discharge it. The large-gradient regime is handled by the descent lemma I already have; perturbing there would only add noise to a step that's already making progress, so I gate the perturbation on the gradient being small. And capping it to once every $t_{\mathrm{thres}}$ steps keeps the dynamics *mostly* ordinary gradient descent, so all my dimension-free first-order machinery still applies between perturbations.

Now I owe two things. First, the easy half: when the gradient is large I make progress. That is just the descent lemma above — with $\eta\le1/\ell$, $f(\mathbf{x}_{t+1})\le f(\mathbf{x}_t)-\tfrac{\eta}{2}\|\nabla f(\mathbf{x}_t)\|^2$, so a gradient of size $\ge g_{\mathrm{thres}}$ buys a decrease of $\ge\tfrac{\eta}{2}g_{\mathrm{thres}}^2$ in one step. Done.

Second, the hard half, the one that matters: I am at a point $\tilde{\mathbf{x}}$ with $\|\nabla f(\tilde{\mathbf{x}})\|\le g_{\mathrm{thres}}$ but $\lambda_{\min}(\nabla^2 f(\tilde{\mathbf{x}}))\le-\sqrt{\rho\epsilon}$ — small gradient, genuine negative curvature. I add the perturbation and run gradient descent $t_{\mathrm{thres}}$ steps. I need: with high probability the function value drops by $f_{\mathrm{thres}}$. After perturbing, the start $\mathbf{x}_0=\tilde{\mathbf{x}}+\xi$ is uniform over the ball $\mathbb{B}_{\tilde{\mathbf{x}}}(r)$ — call it the perturbation ball. Some starts in this ball escape (function falls a lot); call those the escaping region. The rest get stuck; call that the stuck region $\mathcal{X}_{\mathrm{stuck}}$. I want to show the stuck region is a tiny fraction of the ball's volume, so a uniform draw almost surely lands in the escaping region.

Let me first see why Ge's approach forced tiny steps, so I know what trap to avoid. Suppose the Hessian were exactly constant over the ball — a pure quadratic. Then the stuck region is just the set of starts whose component along the escape direction $\mathbf{e}_1$ (the most-negative eigenvector) is too small to amplify in time: a flat slab, perpendicular to $\mathbf{e}_1$, of some small thickness. A flat disk. Its volume is easy. But the Hessian is *not* constant; it is only $\rho$-Lipschitz. If I keep $\eta$ and $r$ at $\Theta(1)$ scale (no $1/d$), then across a ball of radius $r$ the Hessian changes appreciably, the gradient flow curves, and the stuck region is no longer a flat slab. It bends. In two dimensions it is a curved "narrow band" hugging the gradient flow lines; in three it is a "thin pancake," warped. Ge avoids this entirely by shrinking $r$ to $1/d$ so the quadratic model is accurate over the whole ball and the slab stays flat — and pays $d^4$ for the privilege. If I want the large step I have to confront the curved pancake directly. And approximating a curved pancake by a flat disk loses polynomial factors in $d$ — that is precisely the suboptimality I'm trying to kill, so I cannot cheat that way.

Here is the thing I keep tripping over: I have *no analytic description* of this pancake. I don't know its shape; the function near the saddle is only known up to Lipschitz control of its Hessian. Computing the volume of a set I can't even write down feels hopeless. So let me stop trying to know its shape. What do I actually need? I need its *volume* to be small. And a region can have small volume for a reason much weaker than knowing its shape: if it is *thin* in one direction, no matter how it curves, its volume is bounded by (thickness)$\times$(cross-section). The cross-section is at most that of the ball. So everything reduces to: how *thick* is the stuck region along $\mathbf{e}_1$?

Let me make "thin" precise by a coupling. Take a starting point $\mathbf{u}_0$ in the perturbation ball and a same-line point $\mathbf{w}_0=\mathbf{u}_0+\mu r\,\mathbf{e}_1$ with $\mu\le1$; if $\mathbf{w}_0$ falls outside the ball it contributes nothing to the stuck volume, and if it falls inside the ball it is the paired start I care about. I want to argue that if the two points are far enough apart along $\mathbf{e}_1$, they *cannot both* be stuck — at least one of the two gradient-descent trajectories must escape. If I can show that, then on any line parallel to $\mathbf{e}_1$ the stuck set is confined to an interval no longer than that separation. That bounds the thickness, regardless of how the pancake curves, and I never needed its shape.

So the claim I'll chase: there is a threshold separation, on the order of $\delta r/\sqrt{d}$, such that any two same-$\mathbf{e}_1$-line points farther apart than that cannot both be stuck. Let me see what that buys before I prove it. If true, then for any line in the $\mathbf{e}_1$ direction, the stuck set on it is an interval of length at most $\delta r/\sqrt{d}$. Integrate: the volume of the stuck region is at most the $(d-1)$-dimensional cross-section of the ball times $\delta r/\sqrt{d}$. Concretely, splitting a point as $(x^{(1)},\mathbf{x}^{(-1)})$ with $x^{(1)}$ the $\mathbf{e}_1$ component,
$$\mathrm{Vol}(\mathcal{X}_{\mathrm{stuck}})=\int_{\mathbb{B}^{(d)}_{\tilde{\mathbf{x}}}(r)}\!\mathbf{1}_{\mathcal{X}_{\mathrm{stuck}}}(\mathbf{x})\,d\mathbf{x}=\int_{\mathbb{B}^{(d-1)}(r)}\!d\mathbf{x}^{(-1)}\!\int\! dx^{(1)}\,\mathbf{1}_{\mathcal{X}_{\mathrm{stuck}}}\le\int_{\mathbb{B}^{(d-1)}(r)}\!d\mathbf{x}^{(-1)}\cdot\frac{\delta r}{\sqrt{d}}=\mathrm{Vol}\big(\mathbb{B}^{(d-1)}(r)\big)\cdot\frac{\delta r}{\sqrt{d}},$$
because on each line the indicator is supported on an interval of length at most $2\cdot\tfrac{\delta}{2\sqrt d}r=\tfrac{\delta r}{\sqrt d}$. Now the ratio to the full ball:
$$\frac{\mathrm{Vol}(\mathcal{X}_{\mathrm{stuck}})}{\mathrm{Vol}(\mathbb{B}^{(d)}(r))}\le\frac{\delta r/\sqrt{d}\cdot\mathrm{Vol}(\mathbb{B}^{(d-1)}(r))}{\mathrm{Vol}(\mathbb{B}^{(d)}(r))}.$$
The ball-volume ratio $\mathrm{Vol}(\mathbb{B}^{(d-1)}(r))/\mathrm{Vol}(\mathbb{B}^{(d)}(r))$ is $\tfrac{1}{r\sqrt\pi}\,\tfrac{\Gamma(d/2+1)}{\Gamma(d/2+1/2)}$, so the whole thing is $\tfrac{\delta}{\sqrt{\pi d}}\,\tfrac{\Gamma(d/2+1)}{\Gamma(d/2+1/2)}$. I want this to be $\le\delta$, i.e. I want $\tfrac{1}{\sqrt{\pi d}}\tfrac{\Gamma(d/2+1)}{\Gamma(d/2+1/2)}\le1$. The Gamma ratio grows like $\sqrt{d/2}$, so dividing by $\sqrt{d}$ should leave a bounded number — let me pin it down rather than wave at it. There is a clean fact, $\tfrac{\Gamma(x+1)}{\Gamma(x+1/2)}<\sqrt{x+\tfrac12}$ for $x\ge0$ (I checked it numerically across $x=0,\tfrac12,1,2,5,10,50$ and it holds with a little room each time — e.g. at $x=10$, $3.202<3.240$), giving $\tfrac{\delta}{\sqrt{\pi d}}\sqrt{\tfrac{d}{2}+\tfrac12}\le\delta$. And just to be sure the constant isn't secretly above $1$, I evaluated the whole prefactor $\tfrac{1}{\sqrt{\pi d}}\tfrac{\Gamma(d/2+1)}{\Gamma(d/2+1/2)}$ directly: $0.450$ at $d=2$, $0.409$ at $d=10$, $0.400$ at $d=100$, $0.3990$ at $d=10^3$, settling toward $\tfrac{1}{\sqrt{2\pi}}\approx0.3989$ as $d\to\infty$. Always under $1$, with margin, and decreasing in $d$ — so the bound $\le\delta$ holds for every $d$, not just asymptotically.

So the $\sqrt{d}$ in the thickness threshold and the $\sqrt{d}$ from the ball's cross-section cancel, and the stuck-region fraction is $\le\delta$ with no surviving power of $d$. A uniform perturbation misses the pancake with probability $\ge1-\delta$, and the only place $d$ can re-enter is through how small I have to make $\delta$, which feeds a logarithm. That is where the polylog-$d$ comes from, and it came from the geometry: thin-in-one-direction plus the way ball cross-sections scale. The reason this works for *me* and not for Ge is the $1/\sqrt d$ threshold: I am allowed a stuck region that is genuinely curved, as long as it is thin, and the coupling controls thinness directly without ever flattening anything. Of course this all rests on the thickness threshold $\delta r/\sqrt d$ actually being correct — which I have assumed and not yet shown — so that coupling claim is now the whole ballgame.

Now I have to actually prove the coupling claim: $\mathbf{u}_0$ and $\mathbf{w}_0=\mathbf{u}_0+\mu r\mathbf{e}_1$ with $\mu\ge\delta/(2\sqrt d)$ can't both be stuck. I'll do it in two pieces. Piece one: if a trajectory is stuck (doesn't decrease the function much), it stays trapped in a small ball around the saddle. Piece two: if $\mathbf{u}$ stays trapped, then the $\mathbf{e}_1$-separation between $\mathbf{w}$ and $\mathbf{u}$ grows geometrically until $\mathbf{w}$ is flung out of that small ball — so $\mathbf{w}$ cannot also be trapped. Together: at most one of them is stuck.

Before the pieces, let me set the bookkeeping, because the proof is a careful balancing of four quantities — function value, gradient, distance, and time — and I want their natural units so the constants don't drown me. Let $\gamma=-\lambda_{\min}(\nabla^2 f(\tilde{\mathbf{x}}))\ge\sqrt{\rho\epsilon}$ be the negative-curvature magnitude, $\kappa=\ell/\gamma$ the condition number, and abbreviate the log factor $\mathscr{L}=\hat c\log(\tfrac{d\kappa}{\delta})$. The only combinations of $\gamma$ and $\rho$ with the right units are: a distance unit $\mathscr{S}\sim\tfrac{\gamma}{\rho}$, a gradient unit $\mathscr{G}\sim\tfrac{\gamma^2}{\rho}$, a function-value unit $\mathscr{F}\sim\tfrac{\gamma^3}{\rho^2}$, and a time unit $\mathscr{T}\sim\tfrac{1}{\eta\gamma}$. Carrying the logs explicitly,
$$\mathscr{S}=\sqrt{\eta\ell}\,\tfrac{\gamma}{\rho}\log^{-1}(\tfrac{d\kappa}{\delta}),\quad\mathscr{G}=\sqrt{\eta\ell}\,\tfrac{\gamma^2}{\rho}\log^{-2}(\tfrac{d\kappa}{\delta}),\quad\mathscr{F}=\eta\ell\,\tfrac{\gamma^3}{\rho^2}\log^{-3}(\tfrac{d\kappa}{\delta}),\quad\mathscr{T}=\tfrac{\log(\tfrac{d\kappa}{\delta})}{\eta\gamma},$$
and these interlock: $\mathscr{G}\cdot\tfrac{\log(d\kappa/\delta)}{\gamma}=\mathscr{S}$ and $\sqrt{\mathscr{F}\cdot\tfrac{\log(d\kappa/\delta)}{\gamma}}=\mathscr{S}$, which I can read off directly from the powers. (Sanity: $\mathscr G\cdot\log/\gamma=\sqrt{\eta\ell}\tfrac{\gamma^2}{\rho}\log^{-2}\cdot\log/\gamma=\sqrt{\eta\ell}\tfrac{\gamma}{\rho}\log^{-1}=\mathscr S$; and $\mathscr F\cdot\log/\gamma=\eta\ell\tfrac{\gamma^3}{\rho^2}\log^{-3}\cdot\log/\gamma=\eta\ell\tfrac{\gamma^2}{\rho^2}\log^{-2}$, whose square root is $\sqrt{\eta\ell}\tfrac{\gamma}{\rho}\log^{-1}=\mathscr S$.) Good, the units are consistent, so a perturbation radius of order $\mathscr{S}/(\kappa\log)$, the natural choice, will make everything dimensionally tidy.

Piece one. Set $\tilde{\mathbf{x}}=0$, $\mathbf{u}_0=0$, $\mathbf{H}=\nabla^2 f(0)$. I'll track the trajectory against the *quadratic model* $\tilde f_{\mathbf{y}}(\mathbf{x})=f(\mathbf{y})+\nabla f(\mathbf{y})^\top(\mathbf{x}-\mathbf{y})+\tfrac12(\mathbf{x}-\mathbf{y})^\top\mathbf{H}(\mathbf{x}-\mathbf{y})$, and "stuck" will mean this model value hasn't dropped by $3\mathscr{F}$. The gradient-descent update, written exactly with a Taylor remainder, is
$$\mathbf{u}_{t+1}=\mathbf{u}_t-\eta\nabla f(\mathbf{u}_t)=\mathbf{u}_t-\eta\nabla f(0)-\eta\Big(\int_0^1\nabla^2 f(\theta\mathbf{u}_t)\,d\theta\Big)\mathbf{u}_t=(\mathbf{I}-\eta\mathbf{H}-\eta\Delta_t)\mathbf{u}_t-\eta\nabla f(0),$$
where $\Delta_t=\int_0^1\nabla^2 f(\theta\mathbf{u}_t)\,d\theta-\mathbf{H}$ is the deviation of the average Hessian along the segment from the saddle's Hessian. By Hessian-Lipschitzness $\|\Delta_t\|\le\rho(\|\mathbf{u}_t\|+\|\tilde{\mathbf{x}}\|)$, and by smoothness $\|\nabla f(0)\|\le\|\nabla f(\tilde{\mathbf{x}})\|+\ell\|\tilde{\mathbf{x}}\|\le\mathscr{G}+\ell\cdot2\tfrac{\mathscr S}{\kappa\log}\le3\mathscr{G}$ (using $\ell\mathscr S/(\kappa\log)=\sqrt{\eta\ell}\,\gamma/\log^2\cdot\ldots$ which is of order $\mathscr G$). So the dynamics are power-iteration on $\mathbf{I}-\eta\mathbf{H}$ plus two controllable error terms.

Split the space. Let $\mathcal{S}$ be the span of eigenvectors of $\mathbf{H}$ with eigenvalue below $-\gamma/\mathscr{L}$ — the strongly-negative directions, where power iteration genuinely amplifies over my time budget — and $\mathcal{S}^c$ the rest. Write $\bm\alpha_t=\mathrm{proj}_{\mathcal S}\mathbf{u}_t$, $\bm\beta_t=\mathrm{proj}_{\mathcal S^c}\mathbf{u}_t$. The "not stuck yet" condition says, for all $t<T$, $\tilde f_0(\mathbf{u}_t)-f(0)>-3\mathscr{F}$. Expand the left side: $\tilde f_0(\mathbf{u}_t)-f(0)=\nabla f(0)^\top\mathbf{u}_t-\tfrac12\mathbf{u}_t^\top\mathbf{H}\mathbf{u}_t$. Wait — sign: $\tilde f_0(\mathbf u_t)-f(0)=\nabla f(0)^\top\mathbf u_t+\tfrac12\mathbf u_t^\top\mathbf H\mathbf u_t$, but on $\mathcal S$ the Hessian is at most $-\gamma/\mathscr L$, so $\tfrac12\bm\alpha_t^\top\mathbf H\bm\alpha_t\le-\tfrac{\gamma}{2\mathscr L}\|\bm\alpha_t\|^2$, hence $\tilde f_0(\mathbf u_t)-f(0)\le\nabla f(0)^\top\mathbf u_t-\tfrac{\gamma}{2\mathscr L}\|\bm\alpha_t\|^2+\tfrac12\bm\beta_t^\top\mathbf H\bm\beta_t$. Since this exceeds $-3\mathscr F$, I can solve for $\|\bm\alpha_t\|^2$ and then, via $\|\mathbf u_t\|^2=\|\bm\alpha_t\|^2+\|\bm\beta_t\|^2$,
$$\|\mathbf{u}_t\|^2\le\tfrac{2\mathscr{L}}{\gamma}\big(3\mathscr{F}+\nabla f(0)^\top\mathbf{u}_t+\tfrac12\bm\beta_t^\top\mathbf{H}\bm\beta_t\big)+\|\bm\beta_t\|^2\le14\max\Big\{\tfrac{\mathscr{G}\mathscr{L}}{\gamma}\|\mathbf u_t\|,\ \tfrac{\mathscr{F}\mathscr{L}}{\gamma},\ \tfrac{\bm\beta_t^\top\mathbf H\bm\beta_t\mathscr L}{\gamma},\ \|\bm\beta_t\|^2\Big\},$$
using $\|\nabla f(0)\|\le3\mathscr G$. Solving the resulting quadratic-in-$\|\mathbf u_t\|$,
$$\|\mathbf{u}_t\|\le14\max\Big\{\tfrac{\mathscr{G}\mathscr{L}}{\gamma},\ \sqrt{\tfrac{\mathscr{F}\mathscr{L}}{\gamma}},\ \sqrt{\tfrac{\bm\beta_t^\top\mathbf H\bm\beta_t\mathscr L}{\gamma}},\ \|\bm\beta_t\|\Big\}.$$
By the unit relations the first two terms are both $\Theta(\mathscr S)$. So everything hinges on the last two — I must show the $\mathcal S^c$ component $\bm\beta_t$ stays small. That's where the split pays off: the directions outside $\mathcal S$ either contract or grow only by the mild factor $1+\eta\gamma/\mathscr L$ over the time window.

The $\bm\beta$ update from the decomposed dynamics is $\bm\beta_{t+1}=(\mathbf I-\eta\mathbf H)\bm\beta_t-\eta\,\mathrm{proj}_{\mathcal S^c}(\Delta_t\mathbf u_t+\nabla f(0))$; lump the perturbation as $\bm\delta_t$ with $\|\bm\delta_t\|\le\|\Delta_t\|\|\mathbf u_t\|+\|\nabla f(0)\|$. Inductively assuming $\|\mathbf u_t\|\le100\mathscr S\hat c$, Hessian-Lipschitz gives $\|\Delta_t\|\le\rho(100\hat c\,\mathscr S+\ldots)$, so $\|\bm\delta_t\|\le\rho\cdot100\hat c(100\hat c+2)\mathscr S^2+\mathscr G=[100\hat c(100\hat c+2)\sqrt{\eta\ell}+1]\mathscr G\le2\mathscr G$ once $\eta\le c_{\max}/\ell$ with $c_{\max}$ small. Now on $\mathcal S^c$ the eigenvalues exceed $-\gamma/\mathscr L$, so $\|\mathbf I-\eta\mathbf H\|\le1+\eta\gamma/\mathscr L$ there, and recursing from $\bm\beta_0=0$:
$$\|\bm\beta_{t+1}\|\le\sum_{\tau=0}^{t}(1+\tfrac{\eta\gamma}{\mathscr L})^\tau\cdot2\eta\mathscr G\le2\cdot3\cdot T\eta\mathscr G\le6\mathscr S\hat c,$$
where $(1+\eta\gamma/\mathscr L)^T\le3$ because $T\le\hat c\mathscr T=\hat c\log(d\kappa/\delta)/(\eta\gamma)$ makes the exponent $\le\hat c\cdot\eta\gamma/\mathscr L\cdot\mathscr T=\hat c$ which, with $\mathscr L=\hat c\log$, is $O(1)$. So $\|\bm\beta_{t+1}\|=O(\mathscr S)$. Good.

The other term, $\bm\beta_{t+1}^\top\mathbf H\bm\beta_{t+1}$, is more delicate because $\mathbf H$ on $\mathcal S^c$ has a range of eigenvalues up to $\ell$, and I need its positive part not to blow up. Unroll $\bm\beta_t=\sum_{\tau=0}^{t-1}(\mathbf I-\eta\mathbf H)^\tau\bm\delta_\tau$. In the eigenbasis of $\mathbf H$, the negative eigenvalues only reduce $\bm\beta^\top\mathbf H\bm\beta$, so for an upper bound I keep only $\lambda_i\ge0$:
$$\bm\beta_{t+1}^\top\mathbf H\bm\beta_{t+1}\le4\eta^2\mathscr G^2\sum_{\tau_1,\tau_2}\max_{\lambda\in[0,\ell]}\lambda(1-\eta\lambda)^{\tau_1+\tau_2}.$$
Define $g_t(\lambda)=\lambda(1-\eta\lambda)^t$ on $\lambda\ge0$; its derivative $(1-\eta\lambda)^t-t\eta\lambda(1-\eta\lambda)^{t-1}$ vanishes at $\lambda_t^\star=\tfrac{1}{(1+t)\eta}$, the unique maximizer, and $g_t$ is increasing up to there. So $\max_{\lambda\in[0,\ell]}\lambda(1-\eta\lambda)^{\tau_1+\tau_2}\le\tfrac{1}{(1+\tau_1+\tau_2)\eta}$. Hence
$$\bm\beta_{t+1}^\top\mathbf H\bm\beta_{t+1}\le4\eta\mathscr G^2\sum_{\tau_1,\tau_2}\tfrac{1}{1+\tau_1+\tau_2}\le8\eta T\mathscr G^2\le8\mathscr S^2\gamma\hat c\log^{-1}(\tfrac{d\kappa}{\delta}),$$
where the double sum collapses by counting diagonals: $\sum_{\tau=0}^{2t}\min\{1+\tau,2t+1-\tau\}\tfrac{1}{1+\tau}\le2t+1<2T$. Plug both bounds back into the max for $\|\mathbf u_{t+1}\|$: every branch is $O(\mathscr S\hat c)$, and the constant works out to $\|\mathbf u_{t+1}\|\le100\mathscr S\hat c$. The induction closes. So a stuck trajectory stays within $O(\mathscr S\hat c)$ of the saddle for the whole budget $T$ — it is *confined*. Intuitively exactly what I expected: not decreasing the function means it never got far along the strongly negative directions, and the remaining directions cannot accumulate enough growth to pull it away.

Piece two. Now $\mathbf{u}$ is confined; I show $\mathbf{w}=\mathbf{u}+\mathbf{v}$, separated along $\mathbf{e}_1$, must escape. Track $\mathbf{v}_t=\mathbf{w}_t-\mathbf{u}_t$. Subtracting the two updates,
$$\mathbf{v}_{t+1}=(\mathbf{I}-\eta\mathbf{H}-\eta\Delta'_t)\mathbf{v}_t,\qquad\Delta'_t=\int_0^1\nabla^2 f(\mathbf{u}_t+\theta\mathbf{v}_t)\,d\theta-\mathbf{H},$$
with $\|\Delta'_t\|\le\rho(\|\mathbf u_t\|+\|\mathbf v_t\|+\|\tilde{\mathbf x}\|)$. While confined, $\|\mathbf v_t\|\le\|\mathbf u_t\|+\|\mathbf w_t\|\le200\mathscr S\hat c$; the same confinement lemma applies to $\mathbf w$ from the enlarged $2r$ starting radius allowed in that lemma. So $\|\Delta'_t\|\le\rho\mathscr S(300\hat c+1)$. Let $\psi_t$ be the magnitude of $\mathbf v_t$ along $\mathbf e_1$ and $\varphi_t$ its magnitude on the orthogonal complement. Since $\mathbf e_1$ is the most-negative eigenvector, $\langle\mathbf e_1,(\mathbf I-\eta\mathbf H)\mathbf v\rangle=(1+\eta\gamma)\psi$ — it *grows* — while the perturbation $\Delta'_t$ couples in at most $\mu\sqrt{\psi^2+\varphi^2}$ with $\mu=\eta\rho\mathscr S(300\hat c+1)$:
$$\psi_{t+1}\ge(1+\eta\gamma)\psi_t-\mu\sqrt{\psi_t^2+\varphi_t^2},\qquad\varphi_{t+1}\le(1+\eta\gamma)\varphi_t+\mu\sqrt{\psi_t^2+\varphi_t^2}.$$
I want to show $\mathbf v_t$ stays essentially aligned with $\mathbf e_1$, so the $\mathbf e_1$ component grows cleanly. Claim by induction: $\varphi_t\le4\mu t\,\psi_t$. Base case $\varphi_0=0$ holds since $\mathbf v_0\parallel\mathbf e_1$. For the step, from the two recursions I need $(1+4\mu(t+1))\sqrt{\psi_t^2+\varphi_t^2}\le4(1+\eta\gamma)\psi_t$, and choosing $\sqrt{c_{\max}}$ small enough that $4\mu T\le4\sqrt{\eta\ell}(300\hat c+1)\hat c\le1$, this reduces to $2\sqrt{2}\,\psi_t\le4\psi_t$ when $\varphi_t\le\psi_t$, which holds. So $\varphi_t\le\psi_t$ throughout, the off-axis leakage never dominates, and therefore
$$\psi_{t+1}\ge(1+\eta\gamma)\psi_t-\sqrt2\,\mu\psi_t\ge\big(1+\tfrac{\eta\gamma}{2}\big)\psi_t,$$
since $\mu=\eta\rho\mathscr S(300\hat c+1)\le\sqrt{c_{\max}}(300\hat c+1)\gamma\eta\log^{-1}<\tfrac{\eta\gamma}{2\sqrt2}$ for small $c_{\max}$. The $\mathbf e_1$-gap grows geometrically at rate $1+\tfrac{\eta\gamma}{2}$. Now combine with confinement: while both are trapped, $200\mathscr S\hat c\ge\|\mathbf v_t\|\ge\psi_t\ge(1+\tfrac{\eta\gamma}{2})^t\psi_0$, and $\psi_0\ge\tfrac{\delta}{2\sqrt d}\cdot\tfrac{\mathscr S}{\kappa}\log^{-1}$ from the separation. Solving for when the geometric growth overruns the confinement ceiling,
$$T<\frac{\log\!\big(400\tfrac{\kappa\sqrt d}{\delta}\mathscr L\big)}{2\log(1+\tfrac{\eta\gamma}{2})}\le\frac{\log\!\big(400\tfrac{\kappa\sqrt d}{\delta}\mathscr L\big)}{\eta\gamma}\le(2+\log(400\hat c))\,\mathscr T.$$
Choose $\hat c$ large enough that $2+\log(400\hat c)\le\hat c$, and this says $T<\hat c\mathscr T$ — i.e. *before* the time budget is exhausted, the gap has blown past the confinement radius. But $\mathbf u$ is confined for the whole budget. So $\mathbf w$ cannot also be confined; $\mathbf w$ left the small ball, which (running the confinement argument in reverse) means its model function value dropped by $3\mathscr F$. Therefore at most one of $\mathbf u_0,\mathbf w_0$ is stuck. The coupling claim holds.

I should convert "model dropped $3\mathscr F$" into "true function dropped." When $\mathbf w$ first leaves the ball at some step $\Ts$, $\|\mathbf w_{\Ts}\|=O(\mathscr S)$, and by Hessian-Lipschitz the gap between the true decrease and the model decrease is at most $\tfrac{\rho}{6}\|\cdot\|^3+\tfrac{\rho}{2}\|\cdot\|\|\cdot\|^2=O(\rho\mathscr S^3)=O(\sqrt{\eta\ell}\,\mathscr F)$, which is a small fraction of $\mathscr F$. So a $3\mathscr F$ model drop is at least a $2.5\mathscr F$ true drop, and once gradient descent (with $\eta\le1/\ell$) has dropped the function it never gives it back, so the drop persists to time $T$.

Assemble the saddle lemma. After perturbing, $\mathbf x_0$ is uniform on the ball, the stuck fraction is $\le\delta$, so with probability $\ge1-\delta$ the start is in the escaping region and $f(\mathbf x_T)-f(\mathbf x_0)\le-2.5\mathscr F$. The perturbation itself can only have raised the function by $\nabla f(\tilde{\mathbf x})^\top\xi+\tfrac{\ell}{2}\|\xi\|^2\le\mathscr G\cdot\tfrac{\mathscr S}{\kappa\log}+\tfrac12\ell(\tfrac{\mathscr S}{\kappa\log})^2\le\tfrac32\mathscr F$, so net $f(\mathbf x_T)-f(\tilde{\mathbf x})\le-2.5\mathscr F+1.5\mathscr F\le-\mathscr F$. In the original units (substitute $\eta=c/\ell$, $\gamma=\sqrt{\rho\epsilon}$, $\delta=\tfrac{d\ell}{\sqrt{\rho\epsilon}}e^{-\chi}$) this is: a perturbation plus $t_{\mathrm{thres}}$ steps drops the function by $f_{\mathrm{thres}}$ with probability $\ge1-\tfrac{d\ell}{\sqrt{\rho\epsilon}}e^{-\chi}$.

Now I can set the algorithm's constants and finish the global count. Let $\chi=3\max\{\log(\tfrac{d\ell\Delta_f}{c\epsilon^2\delta}),4\}$ — a single log knob that all the thresholds share — and take $\eta=\tfrac c\ell$, $r=\tfrac{\sqrt c}{\chi^2}\tfrac{\epsilon}{\ell}$, $g_{\mathrm{thres}}=\tfrac{\sqrt c}{\chi^2}\epsilon$, $f_{\mathrm{thres}}=\tfrac{c}{\chi^3}\sqrt{\tfrac{\epsilon^3}{\rho}}$, $t_{\mathrm{thres}}=\tfrac{\chi}{c^2}\tfrac{\ell}{\sqrt{\rho\epsilon}}$. Let me compute the per-step decrease in each regime and see which one is the bottleneck. Large gradient drops $\ge\tfrac{\eta}{2}g_{\mathrm{thres}}^2=\tfrac{c}{\ell}\cdot\tfrac12\cdot\tfrac{c}{\chi^4}\epsilon^2=\tfrac{c^2}{2\chi^4}\tfrac{\epsilon^2}{\ell}$ in one step; the saddle case drops $f_{\mathrm{thres}}$ amortized over $t_{\mathrm{thres}}$ steps, i.e. $\tfrac{f_{\mathrm{thres}}}{t_{\mathrm{thres}}}=\tfrac{(c/\chi^3)\sqrt{\epsilon^3/\rho}}{(\chi/c^2)\,\ell/\sqrt{\rho\epsilon}}=\tfrac{c^3}{\chi^4}\cdot\tfrac{\sqrt{\epsilon^3/\rho}\cdot\sqrt{\rho\epsilon}}{\ell}=\tfrac{c^3}{\chi^4}\tfrac{\epsilon^2}{\ell}$ per step (the $\rho$ and the half-integer powers of $\epsilon$ cancel cleanly — a good sign I matched the units of $f_{\mathrm{thres}}$ and $t_{\mathrm{thres}}$ to $\mathscr F$ and $\mathscr T$ correctly). I plugged concrete numbers in to be sure I didn't drop a factor: with $c=0.05,\chi=12,\epsilon=0.05,\ell\approx7.75,\rho=9$ I get large $=1.94\times10^{-11}$ and saddle $=1.94\times10^{-12}$, ratio exactly $10=1/(2c)$. So the two are *not* equal — the large-gradient step drains faster by $1/(2c)$, and since $c\le c_{\max}\ll1$ the slower, binding regime is the saddle one, at $\tfrac{c^3}{\chi^4}\tfrac{\epsilon^2}{\ell}$ per step. That is fine for the count: both are $\Theta(\tfrac{\epsilon^2}{\ell})$ up to the shared $\chi^4$ and a constant, so the worst-case per-step decrease is $\Theta(\tfrac{c^3}{\chi^4}\tfrac{\epsilon^2}{\ell})$ and that is what I divide into the budget. (I had briefly hoped to tune $f_{\mathrm{thres}},t_{\mathrm{thres}}$ so the two rates were literally equal; they're not, but it doesn't matter — only the minimum of the two controls the count, and it has the right order.) Function value can fall by at most $f(\mathbf x_0)-f^\star$ total, so the algorithm runs at most
$$\frac{f(\mathbf x_0)-f^\star}{\tfrac{c^3}{\chi^4}\tfrac{\epsilon^2}{\ell}}=\frac{\chi^4}{c^3}\cdot\frac{\ell(f(\mathbf x_0)-f^\star)}{\epsilon^2}=O\!\left(\frac{\ell(f(\mathbf x_0)-f^\star)}{\epsilon^2}\log^4\!\Big(\frac{d\ell\Delta_f}{\epsilon^2\delta}\Big)\right)$$
iterations, because $\chi\sim\log$ and $\chi^4\sim\log^4$. So the count comes out to the dimension-free first-order rate $\ell(f_0-f^\star)/\epsilon^2$ multiplied by $\log^4(d\ell\Delta_f/(\epsilon^2\delta))$ — the same $\epsilon$-dependence as plain gradient descent to a first-order point, with $d$ entering only through that fourth power of a logarithm. The extra cost of handling every strict saddle along the way is just those log factors.

Correctness — does it output a genuine second-order stationary point? This is the conjecture I deferred when I first wrote down the loop: that a failed drop really certifies no negative curvature. Now I can discharge it. The algorithm only stops when, after perturbing at a small-gradient point, the function *failed* to drop by $f_{\mathrm{thres}}$. But the saddle lemma I just proved says: if $\lambda_{\min}(\nabla^2 f(\tilde{\mathbf x}))\le-\sqrt{\rho\epsilon}$, the drop *would* happen with probability $\ge1-\tfrac{d\ell}{\sqrt{\rho\epsilon}}e^{-\chi}$. So a failure to drop means, with high probability, $\lambda_{\min}>-\sqrt{\rho\epsilon}$ — exactly the curvature half of the target — and the gradient half is the trigger condition $\|\nabla f\|\le g_{\mathrm{thres}}\le\epsilon$. The number of perturbations over a run is at most $\tfrac{1}{t_{\mathrm{thres}}}\cdot\tfrac{\chi^4}{c^3}\tfrac{\ell(f_0-f^\star)}{\epsilon^2}=\tfrac{\chi^3}{c}\tfrac{\sqrt{\rho\epsilon}(f_0-f^\star)}{\epsilon^2}$, and union-bounding the per-perturbation failure $\tfrac{d\ell}{\sqrt{\rho\epsilon}}e^{-\chi}$ over them gives total failure $\tfrac{\chi^3e^{-\chi}}{c}\cdot\tfrac{d\ell(f_0-f^\star)}{\epsilon^2}$. I want this $\le\delta$, which needs $\chi^3e^{-\chi}$ small; with $\chi\ge12$ I claim $\chi^3e^{-\chi}\le e^{-\chi/3}$, i.e. $\chi^3\le e^{2\chi/3}$. Let me check the boundary, since the bound is only as good as where it first holds: at $\chi=12$, $\chi^3e^{-\chi}=1.06\times10^{-2}$ while $e^{-\chi/3}=1.83\times10^{-2}$ — true, and only barely, so $\chi\ge12$ is the honest threshold (at $\chi=20$ the gap is already two orders of magnitude, so it stays true above). Hence the total failure is $\le e^{-\chi/3}\tfrac{d\ell(f_0-f^\star)}{c\epsilon^2}\le\delta$ by the choice of $\chi=3\max\{\log(\tfrac{d\ell\Delta_f}{c\epsilon^2\delta}),4\}$ (the $\max\{\cdot,4\}$ is exactly what forces $\chi\ge12$). So with probability $1-\delta$ every escape that should happen does, and the output is an $\epsilon$-second-order stationary point — the deferred conjecture holds, and the stopping rule is sound.

One restriction is for free: I assumed $\epsilon\le\ell^2/\rho$. If instead $\epsilon>\ell^2/\rho$, then $\sqrt{\rho\epsilon}>\ell\ge-\lambda_{\min}$ always, so the curvature condition $\lambda_{\min}\ge-\sqrt{\rho\epsilon}$ is automatic and plain gradient descent to a first-order stationary point already suffices — no saddle handling needed. So the interesting regime is exactly $\epsilon\le\ell^2/\rho$.

Why was a *single* perturbation enough, when Ge needed noise at every step? Because I separated the two jobs noise was doing. Ge's constant noise both (a) breaks the symmetry to put a component on $\mathbf e_1$ and (b) keeps doing so to fight the tiny steps that the constant-noise analysis itself forced. I only need (a), once per saddle: deterministic gradient descent's own power-iteration amplification does the rest of the escaping for me — that's piece two. And I avoid (b) entirely by keeping $\eta=\Theta(1/\ell)$, which is licensed because the coupling argument tolerates a curved stuck region. Lee's random-init-only fails because *one* perturbation at the very start cannot handle a *chain* of saddles encountered later; my fix is to re-perturb at each new stuck point, gated on the gradient being small. So: perturb when stuck, once, then let the dynamics work.

Now strict-saddle and local speedup, almost as corollaries. If $f$ is robustly $(\theta,\gamma,\zeta)$-strict-saddle — every point has $\|\nabla f\|\ge\theta$, or $\lambda_{\min}\le-\gamma$, or is $\zeta$-close to a local minimum — then set $\tilde\epsilon=\min(\theta,\gamma^2/\rho)$. An $\tilde\epsilon$-second-order stationary point has $\|\nabla f\|\le\tilde\epsilon\le\theta$ (fails region 1) and $\lambda_{\min}\ge-\sqrt{\rho\tilde\epsilon}\ge-\gamma$ (fails region 2), so it must be in region 3: $\zeta$-close to a local minimum. So the main theorem with $\epsilon\to\tilde\epsilon$ lands me near a local minimum, in $O(\tfrac{\ell(f_0-f^\star)}{\tilde\epsilon^2}\log^4(\cdot))$ steps.

And once I'm near a local minimum, if there is local structure I can switch to fast linear convergence. Suppose in a $\zeta$-neighborhood the function obeys an $(\alpha,\beta)$-regularity condition, $\langle\nabla f(\mathbf x),\mathbf x-\mathrm{proj}_{\mathcal X^\star}(\mathbf x)\rangle\ge\tfrac{\alpha}{2}\|\mathbf x-\mathrm{proj}\|^2+\tfrac{1}{2\beta}\|\nabla f\|^2$ — which strong-convexity-plus-smoothness implies, but which is weaker and holds even when the local minima form a connected set. Run plain gradient descent with step $\tfrac1\beta$. Then
$$\|\mathbf x_{t+1}-\mathrm{proj}(\mathbf x_{t+1})\|^2\le\|\mathbf x_{t+1}-\mathrm{proj}(\mathbf x_t)\|^2=\|\mathbf x_t-\mathrm{proj}(\mathbf x_t)\|^2-2\eta\langle\nabla f,\mathbf x_t-\mathrm{proj}\rangle+\eta^2\|\nabla f\|^2,$$
and substituting the regularity inequality with $\eta=\tfrac1\beta$ (so $\eta^2-\tfrac{\eta}{\beta}=0$ kills the gradient term) gives $\le(1-\tfrac{\alpha}{\beta})\|\mathbf x_t-\mathrm{proj}(\mathbf x_t)\|^2$. Two things fall out: the distance is monotone, so the iterate never leaves the neighborhood; and it contracts at rate $1-\tfrac\alpha\beta$, giving $\epsilon$-closeness in $O(\tfrac\beta\alpha\log\tfrac\zeta\epsilon)$ steps. So the full run is two phases — perturbed gradient descent to reach the basin, then plain gradient descent for linear convergence inside it: $O\big(\tfrac{\ell(f_0-f^\star)}{\tilde\epsilon^2}\log^4(\tfrac{d\ell\Delta_f}{\tilde\epsilon^2\delta})+\tfrac\beta\alpha\log\tfrac\zeta\epsilon\big)$.

Let me trace the whole causal chain once, now that every link has been checked. The obstacle in nonconvex first-order optimization is saddles, not local minima, so the right target is a second-order stationary point, which under strict-saddle equals a local minimum (I verified the second-order test really does reject the canonical saddle and accept a minimum). Plain gradient descent stalls at saddles because its only sensor is the gradient and that is zero there — I watched it sit motionless at the origin of a concrete strict saddle. Near a saddle, gradient descent is power iteration on $\mathbf I-\eta\mathbf H$, so it escapes automatically *if* the iterate has a component along the negative-curvature direction — which is the one thing a perturbation can supply, and one perturbation sufficed in the numerical run, with the escape sign set by the random kick. To stay dimension-free I must keep the natural large step, which makes the stuck region a curved pancake I can't describe; but I don't need its shape, only that it's thin, and a two-point coupling along $\mathbf e_1$ shows its width is $O(\delta r/\sqrt d)$ because if two same-line points were both stuck the geometric growth of their $\mathbf e_1$-gap would fling one out of the confining ball. The $\sqrt d$ in that width cancels the $\sqrt d$ in the ball's cross-section — a cancellation I confirmed leaves a prefactor under $1/\sqrt{2\pi}$ for every $d$ — so the stuck fraction is $\le\delta$ with no power of $d$. Perturb only when the gradient is small; the binding per-step decrease is the saddle one, $\tfrac{c^3}{\chi^4}\tfrac{\epsilon^2}{\ell}$ (the large-gradient steps drain faster), so the function drains at a fixed rate until a genuine second-order stationary point, in $O(\tfrac{\ell(f_0-f^\star)}{\epsilon^2}\log^4 d)$ steps — the first-order rate, with $d$ confined to log factors, escaping every strict saddle at the cost of only those logs.

```python
import numpy as np

# Perturbed Gradient Descent: finds an eps-second-order stationary point
# (||grad f|| <= eps and lambda_min(Hess f) >= -sqrt(rho*eps)) for an
# ell-smooth, rho-Hessian-Lipschitz f, in O( ell*(f0 - f*)/eps^2 * log^4 d ).

def perturbed_gradient_descent(grad, f, x0, ell, rho, eps, c, delta, Delta_f):
    x = np.array(x0, dtype=float)
    d = x.size
    # one shared log knob chi ~ log(d ell Delta_f / (c eps^2 delta)); >= 12
    chi   = 3.0 * max(np.log(d * ell * Delta_f / (c * eps**2 * delta)), 4.0)
    eta   = c / ell                         # natural large step, Theta(1/ell): keeps it dimension-free
    r     = (np.sqrt(c) / chi**2) * eps / ell        # perturbation radius: worst-case f-rise < escape drop
    g_thr = (np.sqrt(c) / chi**2) * eps              # "gradient small" => maybe at a saddle
    f_thr = (c / chi**3) * np.sqrt(eps**3 / rho)     # required drop to certify an escape
    t_thr = int(np.ceil((chi / c**2) * ell / np.sqrt(rho * eps)))  # escape time budget

    t_noise = -t_thr - 1                    # last perturbation time (allow immediate perturb)
    x_tilde, f_tilde = None, None
    t = 0
    while True:
        g = grad(x)
        # small gradient and not perturbed recently -> kick once along all directions,
        # then let deterministic GD's power-iteration amplify the e_1 component.
        if np.linalg.norm(g) <= g_thr and t - t_noise > t_thr:
            x_tilde, f_tilde, t_noise = x.copy(), f(x), t
            # uniform on the d-ball: U^{1/d} * Y/||Y||, scaled by r
            y = np.random.randn(d); u = np.random.rand()**(1.0 / d)
            x = x + r * u * y / np.linalg.norm(y)
        # t_thr steps after a perturbation: if f didn't fall by f_thr, x_tilde is an eps-SOSP.
        if x_tilde is not None and t - t_noise == t_thr:
            if f(x) - f_tilde > -f_thr:
                return x_tilde
        x = x - eta * grad(x)               # plain gradient step (descent lemma drains f when grad large)
        t += 1
```
