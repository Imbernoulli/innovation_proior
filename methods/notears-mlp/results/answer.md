# NOTEARS-MLP, distilled

NOTEARS-MLP learns a nonlinear DAG from observational data by turning score-based structure
learning into a single smooth, sparsity-penalized continuous optimization problem. Each variable
is modeled by its own multilayer perceptron; the dependence of a child on a candidate parent is
defined nonparametrically by the `L2` norm of the partial derivative, then represented in the MLP
case by the norm of the first-layer weight column for that input. Acyclicity is enforced by the
trace-exponential constraint `h = tr(e^{W∘W}) - d = 0` applied to that derived adjacency matrix,
and the whole program is solved by augmented Lagrangian with an L-BFGS-B inner solver, followed
by a hard threshold that cuts weak residual edges.

## Problem it solves

Given `n` i.i.d. observations of `d` variables generated by a nonlinear structural equation model
`X_j = f_j(X_{pa(j)}) + z_j`, recover the directed acyclic graph. The acyclicity constraint makes
the feasible set the superexponential, NP-hard space of DAGs; existing nonlinear methods enforce
it with combinatorial search (greedy edge edits, order search) and commit to one model class.
NOTEARS-MLP avoids both: one continuous program, generic solver, all edges updated jointly.
Nonlinear additive-noise identifiability (Hoyer et al. 2009; Peters et al. 2014, Cor. 31) is what
makes the direction recoverable from observational data — the Gaussian-linear case is not
identifiable.

## Key idea

1. **Smooth acyclicity (Zheng et al. 2018).** `(B^k)_{ii}` counts length-`k` closed walks, so a
   graph is acyclic iff `tr(B^k) = 0` for all `k`. The matrix exponential collapses all powers
   into one number, `tr(e^B) - d = ∑_{k≥1} tr(B^k)/k!`, convergent for every matrix and
   numerically stable (the `1/k!` damps the explosive high powers, unlike `tr((I-B)^{-1})` which
   needs spectral radius `< 1` and is ill-conditioned). For real weights, Hadamard-square so all
   edge weights are nonnegative:
   ```
   h(W) = tr( e^{W∘W} ) - d = 0   iff   G(W) is a DAG,     ∇h(W) = (e^{W∘W})^T ∘ 2W.
   ```
2. **A nonparametric surrogate for `W`.** A general nonlinear SEM has no coefficient matrix, but
   `h` only needs a nonnegative "does `f_j` use `X_k`" scalar per ordered pair. A smooth `f_j` is
   independent of `X_k` iff `∂_k f_j ≡ 0`, i.e. `||∂_k f_j||_{L2} = 0`. Define the surrogate
   ```
   [W(f)]_{kj} := || ∂_k f_j ||_{L2},
   ```
   feed it to the same `h`. When the `f_j` are linear this reduces to the linear weighted
   adjacency — a genuine generalization.
3. **MLP instantiation + the column-norm proposition.** Model `f_j` by
   `MLP(u; A^{(1)},...,A^{(h)}) = σ(A^{(h)} σ(··· σ(A^{(1)} u)))`. The MLPs independent of `u_k`
   are *exactly* those whose `k`-th column of `A^{(1)}` is zero (proof below) — no expressivity is
   lost by zeroing a column. Hence the dependence summary is the squared column norm
   `[W(θ)]_{kj}^2 = ∑_b (A_j^{(1)})_{bk}^2 = [W(θ)∘W(θ)]_{kj}`, which `h` consumes directly, and
   acyclicity is **independent of network depth**.
4. **Continuous program.**
   ```
   min_θ  (1/n) ∑_j ℓ( x_j, MLP(X; θ_j) ) + λ ∑_j ||A_j^{(1)}||_{1,1}    s.t.   h(W(θ)) = 0.
   ```
   Least-squares `ℓ` for additive-noise; `ℓ_1` on the first layer drops parents; solved by
   augmented Lagrangian + L-BFGS-B, then hard-threshold `|W| < ω` to cut weak residual edges.

## The key proposition (full proof)

Let `F` = MLPs independent of `u_k`, `F_0` = MLPs with the `k`-th column of `A^{(1)}` all zero.
Then `F = F_0`.

- `F_0 ⊆ F`: if the `k`-th column of `A^{(1)}` is zero, `A^{(1)}u` ignores `u_k`, so does the
  whole network.
- `F ⊆ F_0`: take `f ∈ F`. Let `ũ` be `u` with coordinate `k` set to 0; since `f` ignores `u_k`,
  `f(u) = f(ũ)`. Build `Ã^{(1)}` from `A^{(1)}` by zeroing column `k`. For each hidden unit `s`,
  `(Ã^{(1)} u)_s = ∑_{k'≠k} A^{(1)}_{sk'} u_{k'} = ∑_{k'} A^{(1)}_{sk'} ũ_{k'} = (A^{(1)} ũ)_s`,
  so `Ã^{(1)} u = A^{(1)} ũ`. Then
  `f(u) = f(ũ) = σ(A^{(h)} σ(··· σ(A^{(1)} ũ))) = σ(A^{(h)} σ(··· σ(Ã^{(1)} u))) = MLP(u; Ã^{(1)}, A^{(2)},...,A^{(h)}) ∈ F_0`.
  (Biases ride along unchanged.) ∎

This is why dependence is fully controlled by the first-layer columns and the constraint does not
grow with depth.

## Optimization

Augmented Lagrangian for `min_θ L(θ)` s.t. `h(W(θ)) = 0`:
```
L^ρ(θ, α) = L(θ) + (ρ/2) |h(W(θ))|^2 + α h(W(θ)) + λ1 ||·||_1 + (λ2/2)R2(θ).
```
Outer loop (dual ascent): solve the inner problem for `θ`; set `α ← α + ρ h`; if `h` did not
shrink to `≤ 0.25` of its previous value, set `ρ ← 10ρ` and resolve. Stop when `h ≤ h_tol` (`1e-8`)
or `ρ ≥ ρ_max` (`1e16`). The multiplier `α` is what lets a *finite* `ρ` recover the constrained
solution (a pure quadratic penalty would need `ρ → ∞` and become ill-conditioned).

Inner solver: L-BFGS-B (bound-constrained quasi-Newton). The nonsmooth `ℓ_1` is handled by
**variable splitting** — `A^{(1)} = A^{(1)}_+ - A^{(1)}_-` with both parts bounded `≥ 0`, so
the split objective uses `∑ (A^{(1)}_+ + A^{(1)}_-)`, which is linear under nonnegativity and equals
the `ℓ_1` value at the optimum. The same box bounds pin the diagonal to `(0,0)` to forbid self-loops.

Final **thresholding** `|W| < ω` (`ω = 0.3`): `h` is only `≤ 1e-8` at termination, not symbolically
0, so hard-thresholding removes weak residual edges; deleting edges cannot create cycles.

## Defaults and why

- **One hidden layer, ~10 sigmoid units.** Zero units = linear (misses the nonlinear cause/effect
  asymmetry); a sharp recovery jump appears with a few units; too many units overfit at small `n`.
  Sigmoid is smooth, so partials and the objective are differentiable.
- **`λ1` (`ℓ_1`)** ≈ 0.01–0.03 (larger for fewer samples), drives first-layer columns to zero;
  **`λ2`** is the optional squared-weight penalty used by the canonical implementation.
- **`ρ` schedule (×10 when `h` stalls), threshold 0.25, `h_tol = 1e-8`, `ρ_max = 1e16`,
  `ω = 0.3`.**
- **Squared-column-norm surrogate** computed directly (no `sqrt`) so the constraint stays smooth
  near zero; `sqrt` is taken only to report `W` for thresholding.
- Complexity per L-BFGS-B iteration `O(n d^2 m + d^2 m + d^3)` for `m` hidden units; the `d^3` is
  the matrix exponential.

## Working code

Faithful to the canonical implementation: per-node MLPs (`fc1` split into nonnegative parts for
`ℓ_1`, deeper layers as one local-linear map per node), the squared-column-norm adjacency fed to a
trace-exponential autograd function, augmented Lagrangian with an L-BFGS-B inner solver, and a
final hard threshold.

```python
import math
import numpy as np
import scipy.linalg as slin
import scipy.optimize as sopt
import torch
import torch.nn as nn


class TraceExpm(torch.autograd.Function):
    """f = tr(exp(A)).  d tr(exp(A)) = tr(exp(A) dA)  =>  grad is exp(A)^T."""
    @staticmethod
    def forward(ctx, input):
        E = slin.expm(input.detach().numpy())
        f = np.trace(E)
        E = torch.from_numpy(E)
        ctx.save_for_backward(E)
        return torch.as_tensor(f, dtype=input.dtype)

    @staticmethod
    def backward(ctx, grad_output):
        (E,) = ctx.saved_tensors
        return grad_output * E.t()

trace_expm = TraceExpm.apply


class LocallyConnected(nn.Module):
    """One independent linear map per node: [n, d, m1] -> [n, d, m2]."""
    def __init__(self, num_linear, in_features, out_features, bias=True):
        super().__init__()
        self.weight = nn.Parameter(torch.Tensor(num_linear, in_features, out_features))
        self.bias = nn.Parameter(torch.Tensor(num_linear, out_features)) if bias else None
        k = 1.0 / in_features
        nn.init.uniform_(self.weight, -math.sqrt(k), math.sqrt(k))
        if self.bias is not None:
            nn.init.uniform_(self.bias, -math.sqrt(k), math.sqrt(k))

    def forward(self, x):
        out = torch.matmul(x.unsqueeze(2), self.weight.unsqueeze(0)).squeeze(2)
        if self.bias is not None:
            out = out + self.bias
        return out


class NotearsMLP(nn.Module):
    def __init__(self, dims, bias=True):                 # dims = [d, m1, ..., 1]
        super().__init__()
        assert len(dims) >= 2 and dims[-1] == 1
        d = dims[0]
        self.dims = dims
        self.fc1_pos = nn.Linear(d, d * dims[1], bias=bias)   # variable splitting for L1
        self.fc1_neg = nn.Linear(d, d * dims[1], bias=bias)
        self.fc1_pos.weight.bounds = self._bounds()
        self.fc1_neg.weight.bounds = self._bounds()
        self.fc2 = nn.ModuleList(
            [LocallyConnected(d, dims[l + 1], dims[l + 2], bias=bias)
             for l in range(len(dims) - 2)])

    def _bounds(self):
        d = self.dims[0]
        bounds = []
        for j in range(d):
            for _ in range(self.dims[1]):
                for i in range(d):
                    bounds.append((0, 0) if i == j else (0, None))  # no self-loops
        return bounds

    def forward(self, x):                                # [n, d] -> [n, d]
        x = self.fc1_pos(x) - self.fc1_neg(x)            # effective first layer
        x = x.view(-1, self.dims[0], self.dims[1])       # [n, d, m1]
        for fc in self.fc2:
            x = torch.sigmoid(x)
            x = fc(x)
        return x.squeeze(2)

    def _sq_colnorms(self):
        # [W o W]_{kj} = sum_b (A_j^{(1)})_{bk}^2 = squared k-th-column norm of node j's first layer
        d = self.dims[0]
        w1 = (self.fc1_pos.weight - self.fc1_neg.weight).view(d, -1, d)  # [j, m1, i]
        return torch.sum(w1 * w1, dim=1).t()             # [i, j]

    def h_func(self):
        return trace_expm(self._sq_colnorms()) - self.dims[0]   # tr(exp(W o W)) - d

    def l2_reg(self):
        reg = torch.sum((self.fc1_pos.weight - self.fc1_neg.weight) ** 2)
        for fc in self.fc2:
            reg = reg + torch.sum(fc.weight ** 2)
        return reg

    def fc1_l1_reg(self):
        return torch.sum(self.fc1_pos.weight + self.fc1_neg.weight)  # linear under >= 0 bounds

    @torch.no_grad()
    def fc1_to_adj(self):
        return torch.sqrt(self._sq_colnorms()).cpu().numpy()        # W = sqrt(W o W)


class LBFGSBScipy(torch.optim.Optimizer):
    """Wrap scipy L-BFGS-B; reads each parameter's `.bounds`."""
    def __init__(self, params):
        super().__init__(params, dict())
        self._params = self.param_groups[0]['params']
        self._numel = sum(p.numel() for p in self._params)

    def _flat_grad(self):
        return torch.cat([(p.grad.data.view(-1) if p.grad is not None
                           else p.data.new(p.numel()).zero_()) for p in self._params])

    def _flat_bounds(self):
        b = []
        for p in self._params:
            b += p.bounds if hasattr(p, 'bounds') else [(None, None)] * p.numel()
        return b

    def _flat_params(self):
        return torch.cat([p.data.view(-1) for p in self._params])

    def _set_params(self, flat):
        off = 0
        for p in self._params:
            n = p.numel()
            p.data = flat[off:off + n].view_as(p.data)
            off += n

    def step(self, closure):
        def wrapped(flat):
            self._set_params(torch.from_numpy(flat).to(torch.get_default_dtype()))
            loss = closure().item()
            return loss, self._flat_grad().cpu().detach().numpy().astype('float64')
        x0 = self._flat_params().cpu().detach().numpy()
        sol = sopt.minimize(wrapped, x0, method='L-BFGS-B', jac=True, bounds=self._flat_bounds())
        self._set_params(torch.from_numpy(sol.x).to(torch.get_default_dtype()))


def squared_loss(output, target):
    n = target.shape[0]
    return 0.5 / n * torch.sum((output - target) ** 2)


def dual_ascent_step(model, X, lambda1, lambda2, rho, alpha, h_prev, rho_max):
    optimizer = LBFGSBScipy(model.parameters())
    X_t = torch.from_numpy(X)
    h_new = None
    while rho < rho_max:
        def closure():
            optimizer.zero_grad()
            loss = squared_loss(model(X_t), X_t)
            h = model.h_func()
            penalty = 0.5 * rho * h * h + alpha * h          # (rho/2) h^2 + alpha h
            l2_reg = 0.5 * lambda2 * model.l2_reg()
            l1_reg = lambda1 * model.fc1_l1_reg()
            obj = loss + penalty + l2_reg + l1_reg
            obj.backward()
            return obj
        optimizer.step(closure)
        with torch.no_grad():
            h_new = model.h_func().item()
        if h_new > 0.25 * h_prev:
            rho *= 10
        else:
            break
    alpha += rho * h_new
    return rho, alpha, h_new


def notears_nonlinear(model, X, lambda1=0., lambda2=0., max_iter=100,
                      h_tol=1e-8, rho_max=1e16, w_threshold=0.3):
    rho, alpha, h = 1.0, 0.0, np.inf
    for _ in range(max_iter):
        rho, alpha, h = dual_ascent_step(model, X, lambda1, lambda2, rho, alpha, h, rho_max)
        if h <= h_tol or rho >= rho_max:
            break
    W = model.fc1_to_adj()
    W[np.abs(W) < w_threshold] = 0
    return W


# Usage: W[k, j] != 0  means edge X_k -> X_j.
#   model = NotearsMLP(dims=[d, 10, 1], bias=True)
#   W_est = notears_nonlinear(model, X, lambda1=0.01, lambda2=0.01)
```

## Relation to prior methods

- **Linear NOTEARS** is the special case where each `f_j` is linear: the surrogate `W(f)`
  collapses to the regression-coefficient matrix and `h` is applied to it directly.
- **CAM** assumes additive `f_j = ∑_k f_{jk}(X_k)` and runs a three-stage pipeline (neighborhood
  selection, greedy order search, edge pruning); NOTEARS-MLP is a single continuous program that
  can optionally be wrapped with the same pre/post steps.
- **Sobolev/basis variant (NOTEARS-Sob):** replace each MLP by an orthogonal-series additive
  expansion `f_j = ∑_{k≠j} ∑_r a_{jkr} φ_r(X_k)`; then
  `[W(θ)]_{kj} = (∑_r a_{jkr}^2)^{1/2}`, fed to the same `h`. Same framework, different plug-in
  estimator.
