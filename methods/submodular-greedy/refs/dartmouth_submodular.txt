Greedy Algorithms and Submodularity1
In this note we generalize the set cover (and related) problems using the abstraction of submodular
functions. Submodularity is a fundamental concept in discrete optimization, and arises in many surprising
domains. Therefore, these generalizations have many applications.
Let V be a finite universe. A set function f : 2V ‚Üí R assigns values to every subset of V . We call a
set function non-negative if its range is non-negative reals. We call a set function monotone if for any two
A ‚äÜ B, we have f (A) ‚â§ f (B). We call a set function submodular if it satifies the following

Notes
The result on submodular set cover is from the paper [3] by Wolsey. The result on constrained submodular
maximization is from the paper [2] by Fisher, Nemhauser, and Wolsey. Both these are extremely influential
papers, and started off research on submodular optimization which has really flourished in the last 20 years.
An excellent survey can be found in the paper [1] by Buchbinder and Feldman.
