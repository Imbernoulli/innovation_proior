I want to understand a commutative Banach algebra as if it were an algebra of functions, but I cannot start with a space because the algebra is all I have. In `C(X)` the space is visible first and the algebra comes afterward: a point `x` is something at which every function can be evaluated, and evaluation is multiplicative because `(fg)(x)=f(x)g(x)`. So if an abstract algebra is secretly a function algebra, a point should be a scalar-valued multiplicative probe.

Let me try the most literal version. A point should be a nonzero homomorphism `phi:A->C`, with `phi(ab)=phi(a)phi(b)` and `phi(1)=1`. Then each `a` would define a scalar function by `hat a(phi)=phi(a)`. Addition and multiplication would automatically become pointwise operations:

`hat(a+b)(phi)=phi(a+b)=phi(a)+phi(b)`,

`hat(ab)(phi)=phi(ab)=phi(a)phi(b)=hat a(phi)hat b(phi)`.

That is already the right shape. The concern is whether there are enough such probes, and whether they form a space rather than just a set.

Ideals give the same object from the opposite direction. In `C(X)`, the functions vanishing at a point form a maximal ideal. If I only know the algebra, maximal ideals are the algebraic shadows of points. If `phi` is a character, `ker phi` is maximal because the quotient maps into `C` and the image contains `1`, so the quotient is just `C`. Conversely, if `M` is maximal, I need `A/M` to be a field. It is a commutative unital Banach algebra with no nontrivial ideals. I also need it to be a Banach algebra, so `M` must be closed. If `M` were not closed, its closure would still be an ideal. It cannot contain `1`, because then some `m in M` would satisfy `||1-m||<1`, making `m` invertible by the Neumann series and forcing `M=A`. So the closure is proper, and maximality gives `M` closed. Now `A/M` is a complex Banach division algebra. The Gelfand-Mazur theorem says such an algebra is `C`. The quotient map followed by that identification is a character. Good: maximal ideals and characters are the same candidates for points.

Now the topology. A character is a bounded linear functional. Why bounded? If `|lambda|>||a||`, then `lambda 1-a` is invertible by the Neumann series. Applying a character gives `lambda-phi(a) != 0`, so `|phi(a)|<=||a||`. Thus `||phi||<=1`, and since `phi(1)=1`, `||phi||=1`. The character set sits in the weak-* compact unit ball of `A^*`. The equations defining characters are weak-* closed: `phi(1)=1`, `phi(a+b)=phi(a)+phi(b)`, `phi(alpha a)=alpha phi(a)`, and `phi(ab)=phi(a)phi(b)` are all pointwise conditions on finitely many elements of `A`. So the character set is compact in the weak-* topology; because it lies in a dual weak-* space, the inherited topology is Hausdorff. This is the missing space.

What does an element `a` become on this space? It becomes `hat a(phi)=phi(a)`, and weak-* topology is exactly the topology of pointwise evaluation on elements of `A`, so `hat a` is continuous. I have a homomorphism from `A` into `C(Delta(A))`, where `Delta(A)` is the character space. The next question is what information this homomorphism preserves.

The spectrum is the right test. In a function algebra, the spectrum of a function is its range, because `lambda 1-f` is invertible exactly when it has no zeros. I want the same statement abstractly. If `lambda 1-a` is invertible, then every character sends it to a nonzero number, so `lambda != phi(a)` for all `phi`. Conversely, if `lambda 1-a` is not invertible, the ideal generated by `lambda 1-a` is proper; otherwise it would contain `1`, which would produce an inverse. A proper ideal sits inside a maximal ideal, and that maximal ideal is the kernel of a character. This character kills `lambda 1-a`, so `lambda=phi(a)`. Therefore

`sigma(a) = { phi(a) : phi in Delta(A) }`.

So the transformed function has range equal to the spectrum of the original element. Its uniform norm is

`||hat a||_infty = sup_phi |phi(a)| = sup{|lambda| : lambda in sigma(a)} = r(a)`.

The final equality is the spectral-radius formula in a Banach algebra:

`r(a)=lim_n ||a^n||^(1/n)`.

This is a wall if I hoped to preserve the original norm for every commutative Banach algebra. The construction is not wrong; it is telling me the honest invariant. Abstract multiplication knows the spectral radius. The kernel of the transform consists of elements whose spectrum is `{0}`, the radical. So in a semisimple Banach algebra the transform is injective, but even then it need not be isometric for the given norm. The function picture is present, but the norm may contain more geometry than multiplication alone forces.

The `C^*` case has exactly the extra rigidity needed. In a commutative `C^*`-algebra every element is normal, and for normal elements the `C^*` identity makes norm and spectral radius coincide. Thus

`||a|| = r(a) = ||hat a||_infty`.

Now the transform is isometric. An isometric image of a Banach space inside `C(Delta(A))` is closed, so the range is a closed subalgebra.

I still need the involution to match complex conjugation. If `h=h^*`, then its spectrum is real. A character takes values in the spectrum, so `phi(h)` is real. For a general `a`, write `a=h+ik` with `h=(a+a^*)/2` and `k=(a-a^*)/(2i)` self-adjoint. Then

`phi(a^*)=phi(h-ik)=phi(h)-i phi(k)=overline{phi(h)+i phi(k)}=overline{phi(a)}`.

So `hat(a^*)=overline{hat a}`. The range is closed under complex conjugation.

Does the range fill all of `C(Delta(A))`? It contains constants because `1` transforms to the constant function `1`. It separates points because two distinct characters differ on some `a`, and then `hat a` has different values at those two characters. It is a closed self-adjoint subalgebra of `C(Delta(A))` containing constants and separating points. Stone-Weierstrass now forces it to be all of `C(Delta(A))`.

So the whole picture closes. The abstract points are not imported from outside; they are maximal ideals, equivalently characters. The topology is the weakest one making all evaluations `phi -> phi(a)` continuous. Each algebra element becomes a continuous function on that space. For a general commutative Banach algebra the transform records spectrum and spectral radius, with the radical as the possible kernel. For a commutative `C^*`-algebra, the `C^*` identity removes the norm loss and the involution supplies complex conjugation, so the algebra is exactly `C(Delta(A))`.
