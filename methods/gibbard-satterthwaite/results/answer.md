# Gibbard-Satterthwaite Theorem

Let `A` be a finite set of at least three alternatives and let `N` be a finite set of voters. Each voter has a strict ranking of `A`, and a deterministic voting rule is a function

```text
f: L(A)^N -> A
```

from preference profiles to a single chosen alternative. This is not a recombination theorem about how to merge rankings into a better aggregate ranking; it is an incentive theorem. If `f` is onto and strategy-proof, then `f` is dictatorial. Equivalently, every onto non-dictatorial deterministic voting rule with at least three alternatives is manipulable.

Strategy-proof means that no voter can ever obtain a strictly preferred outcome by misreporting while the other reports are fixed. Onto means every alternative is selected at some profile. Dictatorial means there is a voter `i` such that `f(R)` is always voter `i`'s top-ranked alternative.

## Proof

First derive monotonicity. Suppose `f(R) = a`. Let `R'` be obtained from `R` by changing voters' rankings while never moving `a` below an alternative that was below `a` before. Change the profile one voter at a time from `R` to `R'`. If the outcome ever changes, there is a first step at which one voter changes his report and the outcome changes from `a` to some `b`. Under one of the two reports, that voter would strictly prefer the outcome generated by the other report, because strict rankings compare `a` and `b` and the change has not made `a` worse relative to `b`. That is a profitable manipulation. Hence the outcome cannot change, so `f(R') = a`.

Next derive weak Pareto optimality. Suppose every voter ranks `a` above `b` at a profile `R`, and suppose for contradiction that `f(R) = b`. Since `f` is onto, there is a profile `Q` with `f(Q) = a`. Construct a profile `S` where every voter ranks `a` first, `b` second, and all other alternatives below `b`. From `Q` to `S`, the alternative `a` is moved only upward relative to alternatives it previously beat, so monotonicity gives `f(S) = a`. From `R` to `S`, the alternative `b` remains below `a` and above every alternative it beat at `R`, so monotonicity gives `f(S) = b`. This contradiction proves that no profile where all voters rank `a` above `b` can select `b`. Thus no Pareto-dominated alternative can be chosen.

Now prove the two-voter case. Pick distinct alternatives `a` and `b`. Consider a profile where voter 1 ranks `a` above `b` above all other alternatives, and voter 2 ranks `b` above `a` above all other alternatives. Weak Pareto excludes every alternative below both `a` and `b`, so the outcome is either `a` or `b`. This profile is pivotal: whichever side wins identifies a voter whose top report can be made decisive.

If the outcome is `a`, change voter 2's ranking so that `b` stays above the remaining alternatives and `a` is placed at the bottom. Weak Pareto still restricts the outcome to `a` or `b`. It cannot become `b`, because then voter 2 would have been able to obtain the outcome he preferred at the original profile by changing his report. Therefore `a` remains the outcome. By monotonicity, once voter 1 ranks `a` first, changes elsewhere in the profile cannot make `a` lose. Voter 1 is decisive for `a`. If the original outcome is `b`, the symmetric argument makes voter 2 decisive for `b`.

Repeat this construction over all pairs of alternatives. Let `A_1` be the set of alternatives for which voter 1 is decisive in the sense just derived, and let `A_2` be the analogous set for voter 2. For every pair `{x, y}`, the pair construction puts at least one of `x` or `y` into one of these sets. Hence at most one alternative lies outside `A_1 union A_2`.

The sets `A_1` and `A_2` cannot both be nonempty. If `x in A_1` and `y in A_2`, then at a profile where voter 1 ranks `x` first and voter 2 ranks `y` first, decisiveness of voter 1 requires `f = x`, while decisiveness of voter 2 requires `f = y`. Since the rule is single-valued, this is impossible. Thus one set is empty. Suppose `A_2` is empty and `A_1` is nonempty; the other case is symmetric.

No alternative can remain outside `A_1`. Pick `x in A_1`. If some `z` were outside `A_1`, apply the pair construction to `{x, z}` with voter 1 ranking `z` above `x` and voter 2 ranking `x` above `z`. The construction implies either `z in A_1` or `x in A_2`. The second option is impossible because `A_2` is empty, so `z in A_1`. Therefore `A_1 = A`, and voter 1 is decisive for every top-ranked alternative. With two voters, the rule is dictatorial.

For finitely many voters, use induction. Assume the result for `k` voters and consider `k + 1` voters. Form a two-agent derived rule by letting agent 1 be voter 1 and agent 2 be the block of voters `2, ..., k + 1`, all reporting the same ranking. The derived rule inherits strategy-proofness from `f`; a profitable manipulation by either agent would give a profitable manipulation to voter 1 or to some voter in the block along a one-voter-at-a-time path. It is also onto under the preceding weak Pareto consequence.

By the two-voter result, the derived rule has a dictator. If voter 1 is that dictator, monotonicity extends his decisive power from block profiles to arbitrary profiles, so voter 1 is a dictator for `f`. If the block is the derived dictator, fix voter 1's report and restrict attention to voters `2, ..., k + 1`. This gives a `k`-voter rule that is strategy-proof and onto, so by the induction hypothesis some voter in the block is its dictator. Since the fixed report of voter 1 was arbitrary, monotonicity carries that voter's decisive power across all full profiles. Therefore `f` is dictatorial.

The theorem follows: incentive compatibility plus nonimposition leaves dictatorship as the only possibility. Hence any onto non-dictatorial voting rule with at least three alternatives admits a strategic manipulation.
