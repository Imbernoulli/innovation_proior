# Answer: Proto-Chamic → Phan Rang Cham & Tsat

Regular sound change carried Proto-Chamic into both daughters. In each, the voiced obstruents stopped being recoverable as plain voicing — but before that contrast vanished they had depressed the pitch of nearby vowels, and that pitch survived as **tone** (tonogenesis). Phan Rang Cham stayed mostly disyllabic and grew a single low tone; Tsat reduced most words to the final syllable, merged final stops to ʔ, deleted final *h, and grew a register-by-coda tone system. The lost voicing and some lost codas survive in Tsat only as tone.

Key: V = vowel, C = consonant, N = nasal (m, n, ŋ), L = liquid (l, r), D = voiced depressor obstruent (b, d, j in the data).

## Phan Rang Cham sound laws

| change | example |
|---|---|
| voiced obstruents devoice: *b→p, *d→t, *j→c (and *g→k where present) | *dikiʔ → takiʔ; *jalaːn → calàn |
| *a → ɨ after a nasal | *lima → limɨ; *maray → mɨray |
| *aː → a (long vowels shorten) | *ʔikaːn → ikan |
| penult *ʔV → V | *ʔiduŋ → itùng; *ʔikat → ikaʔ |
| penult *lV → li | *lanah → linɨh; *labaːt → lipàʔ |
| penult *bV(h)L → piL | *bara → pirà; *bahrow → piròw |
| otherwise penult *CV(V)- → Ca- | *hitam → hatam; *dikiʔ → takiʔ |
| *s → th | *basah → pathah |
| final *-p/-t/-k → ʔ | *labaːt → lipàʔ |
| final *-h kept | *dilah → talàh |

**Tone (tonogenesis):** the root-final vowel takes **low tone** iff the consonant before it was a voiced obstruent *D, **or** that consonant is a liquid *L whose own preceding consonant was *D. In compounds, this applies component by component. (*paday → patày; *jalaːn → calàn; *dua lapan → tàlipan.) Otherwise no tone mark.

## Tsat sound laws

The penultimate syllable is **deleted**, leaving a monosyllable — **except** for initial *{b,p,g}(V)L, where the penult survives as **{ph,p,kh}i** (the liquid drops): *bara → phia, *paley → piai. Then for the surviving final syllable:

| change | example |
|---|---|
| surviving voiced stop onsets *b, *d, *g → ph, th, kh | *paday → thaːiʔ; *labaːt → phaːʔ |
| *r → z | *ʔurat → zaʔ; *rək → zəʔ |
| *s kept as s (not th) | *basah → sa⁵⁵ |
| final *-p/-t/-k → ʔ | *batuk → tuʔ |
| final *-h → ∅ (→ tone 55) | *dilah → la⁵⁵ |
| *-ow → ə, *-ey → ai | *bulow → phiə; *bəsey → sai |
| *aː kept long | *ʔikaːn → kaːn; *jalaːn → laːn |

**Tone (Tsat = high 55 / falling 42 / mid 33 / rising 24 / low 11):**

Special rhymes override everything:
- final *-h → tone **55**
- final short *-aN (a + nasal; **not** long *-aːn) → -aːnʔ **42**
- final *-ay → -aːiʔ **42**

Otherwise a register-by-coda grid (register = was there a voiced obstruent *D anywhere in the ancestor, even in the deleted penult):

| | stop-final (…ʔ) | open / sonorant-final |
|---|---|---|
| **+D** (voiced register) | **42** | **11** |
| **−D** (voiceless register) | **24** | **33** |

(*dikiʔ → kiʔ⁴²; *rataːk → taːʔ²⁴; *bara → phia¹¹; *mata → ta³³. *jalaːn → laːn¹¹ shows the depressor *j surviving only as tone after its whole syllable was deleted.)
