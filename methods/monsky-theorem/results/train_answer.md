I present Monsky's theorem, the statement that a square cannot be dissected into an odd number of triangles all having the same area. The pieces are ordinary triangles with disjoint interiors, and they may meet edge-to-edge or with T-junctions and irrationally placed vertices; the only constraint is that every piece has identical Euclidean area. After translating, rotating, and scaling the square to the unit square S = [0,1]^2, which multiplies all areas by one common nonzero factor and leaves the number of pieces unchanged, the problem becomes: if S is cut into n triangles of equal area, then n must be even, because each piece would have area 1/n. This is a parity claim hidden inside continuous geometry, and the key is to read coordinates through the 2-adic valuation.

The method I use is Monsky's argument. It combines a 2-adic valuation with a three-colouring of the plane, an arithmetic area lemma, and a Sperner-type parity count. On the rational numbers, the additive 2-adic valuation v is defined by writing a nonzero rational x uniquely as 2^k (a/b) with a and b odd and setting v(x) = k, while v(0) is infinity. This valuation satisfies v(xy) = v(x) + v(y) and v(x+y) >= min{v(x), v(y)}, with the crucial equality case that v(x+y) = v(x) whenever v(x) < v(y), so a strictly dominant term survives a sum. The arithmetic fact we need is that v(n) = 0 exactly when the integer n is odd, and therefore v(1/n) = 0 whenever n is odd, equivalently |1/n|_2 = 1 for the associated 2-adic absolute value |x|_2 = 2^{-v(x)}.

One difficulty is that a geometric dissection may have irrational coordinates, while v as defined lives only on Q. So I extend v from Q to the field generated by all vertex coordinates and intersection endpoints, or even to all of R, using valuation theory. The valuation ring O = {x : v(x) >= 0} for the 2-adic valuation on Q is Z_(2), the rationals with odd denominator, a local ring whose maximal ideal is generated by 2. Chevalley's extension theorem says that for any field K, subring R, and prime ideal p of R, there is a valuation ring of K containing R whose maximal ideal contracts to p. Applying this with K = R and R = Z_(2) over Q gives a valuation on R whose restriction to Q is the original v, preserving v(2) = 1 and v(1/2) = -1. The value group can enlarge, for example v(sqrt(2)) must be 1/2, but we only compare valuations and reuse the fixed rational values, so an ordered abelian value group is enough.

With v extended, I colour every relevant point (x,y) by comparing v(x), v(y), and v(1) = 0. A point is colour A if v(x) > 0 and v(y) > 0. It is colour B if v(x) <= 0 and v(x) <= v(y). Otherwise it is colour C, meaning v(y) <= 0 and v(y) < v(x). The weak inequality in B and the strict inequality in C make A, B, and C a genuine partition of the plane. The four corners of the unit square are then (0,0) mapped to A, (1,0) mapped to B, (0,1) mapped to C, and (1,1) mapped to B.

This colouring is invariant under translation by an A-point. Adding an A-point (a,b) to a B-point (x,y) leaves the x-coordinate valuation unchanged because v(x) <= 0 < v(a) forces v(x+a) = v(x), and the y-coordinate inequality v(y+b) >= v(x+a) persists. Adding an A-point to a C-point leaves the y-coordinate valuation unchanged and keeps the x-coordinate valuation strictly larger. Because A-points are closed under negation, any triangle can be translated by the negative of its A-vertex without changing the colours of the other two vertices.

The area lemma now connects colour to the determinant. Take a triangle whose three vertices are one of each colour, a so-called rainbow triangle. Translate the A-vertex to the origin, so the vertices are (0,0), (x_B,y_B) in B, and (x_C,y_C) in C. The area is T = (1/2)(x_B y_C - x_C y_B). The B conditions give v(x_B) <= v(y_B) and v(x_B) <= 0, while the C conditions give v(y_C) < v(x_C) and v(y_C) <= 0. Adding the cross-inequalities yields v(x_B) + v(y_C) < v(y_B) + v(x_C), so v(x_B y_C) < v(x_C y_B). By the equality case of the ultrametric inequality, the lower valuation survives the subtraction, giving v(x_B y_C - x_C y_B) = v(x_B) + v(y_C) <= 0. Dividing by 2 lowers the additive valuation by 1, so v(T) = v(1/2) + v(x_B y_C - x_C y_B) <= -1. Equivalently |T|_2 >= 2 > 1. This is where the prime 2 enters the proof.

The same computation also shows that a straight line cannot contain points of all three colours. If three differently coloured points were collinear, the triangle they span would have area 0, which has valuation infinity, contradicting v(T) <= -1. Therefore every line carries at most two colours, and each side of every triangle is a two-colour object.

To force a rainbow triangle in any dissection, I build a finite graph from all triangle corners, square corners, and endpoints of intersections between sides, then subdivide each side into basic segments. I count A/B basic segments, meaning segments whose endpoints are coloured A and B. Because each side lies on a line with at most two colours, a side whose original endpoints are A and B has an odd number of A/B switches along it, while any other side has an even number. A triangle with corner colours A, B, and C has exactly one A/B side; any non-rainbow corner colouring has zero or two A/B sides. So if no original triangle were rainbow, every triangle would contain an even number of A/B basic segments. Summing over all triangles modulo 2, each interior basic segment is counted twice and cancels, so the square boundary would contain an even number of A/B segments.

But the boundary count is forced to be odd. The bottom edge y = 0 has v(y) = infinity, so it carries no C-points, only A and B, starting at A at (0,0) and ending at B at (1,0); hence it has an odd number of A/B switches. The left edge x = 0 carries no B-point, and the right and top edges carry no A-point because v(1) = 0, so they contribute none. Thus the whole boundary has an odd number of A/B basic segments. This contradiction proves that some original triangle is rainbow.

That rainbow triangle has area T with v(T) <= -1. If the dissection had n equal-area triangles with n odd, every piece would have area 1/n with v(1/n) = 0. The same real number cannot have valuation both <= -1 and 0. Therefore n cannot be odd, and the square has no equidissection of odd cardinality. The canonical name of the method is Monsky's theorem.

The following Python snippet illustrates the pieces of the argument on exact rational coordinates. It defines the 2-adic valuation, applies the A/B/C colouring, checks that random rational rainbow triangles have area valuation at most -1, and shows simple even equidissections of the square.

```python
from fractions import Fraction
import random

def v2_frac(frac):
    """Additive 2-adic valuation of a Fraction; v(0) = infinity."""
    if frac == 0:
        return float('inf')
    n, d = abs(frac.numerator), frac.denominator
    k = 0
    while n % 2 == 0:
        n //= 2
        k += 1
    while d % 2 == 0:
        d //= 2
        k -= 1
    return k

def color_point(p):
    x, y = p
    vx = v2_frac(x)
    vy = v2_frac(y)
    if vx > 0 and vy > 0:
        return 'A'
    if vx <= 0 and vx <= vy:
        return 'B'
    return 'C'

def area_triangle(tri):
    (x1, y1), (x2, y2), (x3, y3) = tri
    return abs(Fraction(1, 2) * (x1*(y2-y3) + x2*(y3-y1) + x3*(y1-y2)))

# Corners of the unit square
corners = [(Fraction(0), Fraction(0)),
           (Fraction(1), Fraction(0)),
           (Fraction(0), Fraction(1)),
           (Fraction(1), Fraction(1))]
print("Corner colours:")
for p in corners:
    print(p, "->", color_point(p))

# Random rational rainbow triangles satisfy v(area) <= -1
random.seed(1)
print("\nRainbow triangle area valuations:")
found = 0
attempts = 0
while found < 10 and attempts < 1000:
    attempts += 1
    pts = []
    for _ in range(3):
        x = Fraction(random.randint(-5, 5), random.randint(1, 7))
        y = Fraction(random.randint(-5, 5), random.randint(1, 7))
        pts.append((x, y))
    cols = [color_point(p) for p in pts]
    if set(cols) == {'A', 'B', 'C'}:
        a = area_triangle(pts)
        if a == 0:
            continue
        found += 1
        print(pts, "colours", cols, "area", a, "v2(area)", v2_frac(a))

# Even equidissections are easy to construct
print("\nEven dissections are possible:")
half = area_triangle([corners[0], corners[1], corners[3]])
print("one half-triangle area =", half, "v2 =", v2_frac(half))
center = (Fraction(1, 2), Fraction(1, 2))
four = [[corners[0], corners[1], center],
        [corners[1], corners[3], center],
        [corners[3], corners[2], center],
        [corners[2], corners[0], center]]
print("four centre triangles have areas:", [area_triangle(t) for t in four])
```
