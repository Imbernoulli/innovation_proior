Let me start with the obstruction I need to produce. I have a square cut into finitely many genuine triangles with disjoint interiors, and every one of those triangles has the same ordinary Euclidean area. I can choose coordinates, translate and rotate the square, and scale it to [0,1]^2; this multiplies every triangle area by the same factor, so equal areas stay equal and the number of pieces does not change. If there are n pieces after that normalization, each area is 1/n. The claim is not about shape; the triangles can be thin, irrationally placed, and non-edge-to-edge. The claim is that n cannot be odd. So I need to pull a parity statement out of coordinates and areas, and ordinary area by itself does not remember parity.

Counting vertices and edges is tempting because it produces integers, but it does not see the equal-area condition. Euler's formula can tell me incidences; it cannot tell me that each triangle area is the same determinant divided by 2. Area sees coordinates but not parity. I need something that reads coordinates and reports divisibility by 2, and that means I should bring in the 2-adic valuation.

On the rationals this is clean. For nonzero x in Q, write x = 2^k(a/b) with a and b odd, and set v(x) = k; also set v(0) = infinity. Then v(xy) = v(x) + v(y), and v(x + y) >= min(v(x), v(y)), with equality when the two valuations are different. In this additive convention v(2) = 1, v(1/2) = -1, and an integer n is odd exactly when v(n) = 0. If I switch to the associated 2-adic absolute value, it is |x|_2 = 2^{-v(x)}, so |2|_2 = 1/2 and |1/2|_2 = 2. I have to keep that conversion fixed: small additive valuation means large 2-adic size.

The first wall is that a geometric dissection does not promise rational coordinates. A vertex can be any real point, while my valuation currently lives only on Q. I cannot prove the rational-coordinate case and pretend it is the whole problem. What I need is an extension of the 2-adic valuation from Q to the real field, or at least to the field generated by the finitely many coordinates of the vertices and the endpoints of side intersections that will appear in the basic-segment graph. Valuation theory gives exactly this. A valuation is equivalently a valuation ring O = {x : v(x) >= 0}, a local ring whose maximal ideal is {x : v(x) > 0}. For the 2-adic valuation on Q, this ring is Z_(2), the rationals with odd denominator, and its maximal ideal is generated by 2. Chevalley's extension theorem says that if I have a field K, a subring R, and a prime ideal p of R, then there is a valuation ring of K containing R whose maximal ideal contracts to p. Applying the prolongation form to the real field over Q, starting from the valuation ring Z_(2) in Q and its maximal ideal (2), I get a valuation ring of the real numbers whose restriction to Q is the original 2-adic one. If I want to see why this is believable, I can order pairs (A,I), with A a subring and I a proper ideal extending the starting local data; Zorn gives a maximal pair, and maximality forces the ring to be a valuation ring, because if both x and x^{-1} were missing then adjoining each would make the ideal generate 1, and the two minimal-degree relations would combine to shorten one of them. The value group may enlarge, as v(sqrt(2)) has to be 1/2, but I only compare valuations and keep the old rational values, so an ordered abelian value group is enough. I fix one such extension once and for all.

Now I need to turn this arithmetic into a colouring of the coordinate field I am using. At a point (x,y), the three quantities that naturally compete are v(x), v(y), and v(1) = 0. I colour the point A if v(x) > 0 and v(y) > 0. If not, I colour it B when v(x) <= 0 and v(x) <= v(y). The remaining case is C: v(y) <= 0 and v(y) < v(x). The weak inequality in B and the strict inequality in C are not cosmetic; they make the three classes disjoint and exhaustive, with one strict comparison left over. The corners of the unit square become A at (0,0), B at (1,0), C at (0,1), and B at (1,1).

Before I use the colouring, I need one invariance. If I add an A-point (a,b), then v(a) and v(b) are both positive. A B-point (x,y) has v(x) <= 0 and v(x) <= v(y). Since v(x) < v(a), the strict case of the valuation inequality gives v(x+a) = v(x). Also v(y+b) >= min(v(y),v(b)), and both entries in that minimum are at least v(x), so v(y+b) >= v(x+a). Thus the translated point is still B. For a C-point, v(y) <= 0 and v(y) < v(x); now v(y+b) = v(y), while v(x+a) > v(y) because both x and a have valuation bigger than v(y). Thus the translated point is still C. A-points are closed under negation, so I may translate a triangle by the negative of its A-vertex without changing whether the other two vertices are B and C.

Now the determinant starts to talk to the colouring. Suppose I have a triangle with one vertex of each colour. I translate the A-vertex to the origin, so the vertices are (0,0), (x_B,y_B), and (x_C,y_C), with the latter two still B and C. Its area is

  T = (1/2)(x_B y_C - x_C y_B).

The B condition gives v(x_B) <= v(y_B) and v(x_B) <= 0. The C condition gives v(y_C) < v(x_C) and v(y_C) <= 0. Adding the cross-inequalities gives

  v(x_B) + v(y_C) < v(y_B) + v(x_C),

so v(x_B y_C) < v(x_C y_B). The two determinant terms have different valuations, so the lower one survives in the difference:

  v(x_B y_C - x_C y_B) = v(x_B y_C) = v(x_B) + v(y_C) <= 0.

Dividing by 2 lowers the additive valuation by 1, hence

  v(T) = v(1/2) + v(x_B y_C - x_C y_B) <= -1.

Equivalently |T|_2 = 2^{-v(T)} >= 2 > 1. That is the arithmetic obstruction I need: a rainbow triangle has additive valuation below 0, or 2-adic absolute value above 1. If all pieces have area 1/n and n is odd, then v(1/n) = -v(n) = 0 and |1/n|_2 = 1. So an odd equidissection is impossible as soon as I can force one triangle in the dissection to have all three colours at its vertices.

I also get the line fact for free from the same determinant. If three points of colours A, B, and C were collinear, the triangle they span would have ordinary area 0. But the computation I just made did not use nondegeneracy; after translating the A-point to the origin it shows that the determinant has finite valuation at most 0 and the area has valuation at most -1. Zero would have valuation infinity. Therefore three differently coloured points cannot lie on one straight line. Every line carries at most two of the three colours.

Now I have to be careful about the combinatorics. If the dissection were already an edge-to-edge triangulation, Sperner's parity count would be immediate: count AB-edges, get an odd boundary count, and therefore an odd number of rainbow triangles. But the problem allows T-junctions and other side-intersection points, and replacing the picture by a refined triangulation can create small triangles whose vertices are not the original triangle corners. I need the parity count to speak about the original triangles themselves.

I form the planar graph of the dissection by putting a vertex at every triangle corner, every square corner, and every endpoint of an intersection between two sides of pieces; a T-junction point is one of these endpoints, and an overlapping side segment contributes its two endpoints. Each triangle side is split into the resulting smaller edge segments. I count AB segments, meaning graph edges whose endpoints are coloured A and B. For one original triangle U, let f(U) be the number of AB segments along its boundary. Because each side of U is a straight line, the line lemma says each side omits at least one colour. I can delete the extra vertices along a side without changing f(U) modulo 2: if that side ever contains C, then it cannot contain both A and B, so it contributes no AB segments; if it contains only A and B, the parity of AB segments along the side is exactly whether the two end colours differ. After deleting all side-interior vertices in this parity sense, f(U) modulo 2 is just the number of sides of U whose original endpoints are A and B. A triangle with corner colours A, B, C has exactly one such side, while every non-rainbow corner-colouring has zero or two. Thus f(U) is odd exactly for original triangles whose three corners have all three colours.

Summing f(U) over all original triangles gives, modulo 2, the number of original rainbow triangles. The same sum can be counted by graph edges: every interior AB segment lies on the boundaries of two triangles and contributes twice, while every boundary AB segment contributes once. So the number of original rainbow triangles is congruent modulo 2 to the number of AB segments on the square boundary.

The boundary count is forced by the four sides. On the left side x = 0, no point is B, since B needs v(x) <= 0 and v(0) = infinity. On the right side x = 1 and the top side y = 1, no point is A, since v(1) = 0. On the bottom side y = 0, no point is C, and the endpoints are A at (0,0) and B at (1,0). Walking along the bottom, the colours are only A and B and they start at A and end at B, so the number of colour changes between A and B, hence the number of boundary AB segments, is odd. Therefore the whole boundary has an odd number of AB segments, and the parity count says there are an odd number of original rainbow triangles. In particular at least one original piece is rainbow.

That piece has area T with v(T) <= -1. If the dissection had n equal-area triangles with n odd, the same piece would also have area 1/n with v(1/n) = 0. The same real number cannot have valuation both <= -1 and 0. So n is not odd. The square cannot be dissected into an odd number of equal-area triangles.
