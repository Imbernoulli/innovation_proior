## Research question

We want an agent that learns reward-maximizing behavior in continuous action spaces — robot-like control where the action is a vector of torques, not one of a handful of discrete moves — directly from interaction, off-policy, and sample-efficiently. The dominant recipe couples a learned value function (a critic) with a deterministic policy (an actor) that ascends the critic's gradient. It works, but it is fragile: runs diverge, performance swings wildly across random seeds, and the learned value function drifts far above any return the policy can actually achieve.

The precise problem is this. A function approximator for the value function is never exact, so its output carries error. In value-based methods with discrete actions, it is well established that building a learning target by maximizing over a *noisy* value estimate produces a systematic *overestimation* — and because temporal-difference learning bootstraps each estimate from the next, that overestimation is fed back into the target and compounds. In the continuous-control actor-critic setting the policy is moved by a gradient step rather than an explicit maximization, so it is not obvious the same bias occurs — yet the symptoms (inflated values, brittle and divergent learning) look identical. A solution would have to: (1) determine whether overestimation is present when the actor is updated by a deterministic policy gradient against an approximate critic, and if so reduce it; (2) control the accumulation of bootstrapped error (variance) so the critic stays close to the true return; and (3) do all this without sacrificing the off-policy sample efficiency that makes the actor-critic recipe attractive — ideally as drop-in changes to the existing algorithm.

## Background

**Reinforcement learning and the value function.** An agent in state $s\in\mathcal S$ takes action $a\in\mathcal A$ under a policy $\pi$, receives reward $r$ and next state $s'$. The return is the discounted sum $R_t=\sum_{i=t}^{T}\gamma^{i-t} r(s_i,a_i)$. The action-value function $Q^\pi(s,a)=\mathbb E[R_t\mid s,a]$ is the expected return of taking $a$ in $s$ then following $\pi$. It obeys the Bellman equation $Q^\pi(s,a)=r+\gamma\,\mathbb E_{s',a'}[Q^\pi(s',a')]$, $a'\sim\pi(s')$.

**Temporal-difference learning with function approximation.** For large or continuous state spaces $Q$ is represented by a differentiable approximator $Q_\theta(s,a)$ and fit by temporal-difference (TD) learning (Sutton 1988; Watkins 1989): minimize $\big(y - Q_\theta(s,a)\big)^2$ where the target $y=r+\gamma Q_{\theta'}(s',a')$ is built from the estimate at the *next* state. Deep Q-learning (Mnih et al. 2015) makes this stable with two ingredients now standard across deep RL: an **experience replay** buffer (Lin 1992) that stores past transitions and samples random minibatches, decorrelating updates and enabling off-policy reuse; and a **target network** $Q_{\theta'}$ whose weights track the online weights slowly — either copied periodically or softly, $\theta'\leftarrow\tau\theta+(1-\tau)\theta'$ — so the regression target does not move under its own optimization.

**Overestimation from the maximum of a noisy estimate.** Standard Q-learning forms the target $y=r+\gamma\max_{a'}Q(s',a')$. Thrun & Schwartz (1993) observed that if the estimate carries error $\epsilon$, then in expectation $\mathbb E_\epsilon\!\big[\max_{a'}(Q(s',a')+\epsilon)\big]\ge\max_{a'}Q(s',a')$: the max of a noisy quantity is a biased estimator of the max of the true quantity, even when $\epsilon$ is zero-mean. The maximizing index tends to be whichever action's error happened to be largest, so the selected value is inflated. Under function approximation this error is unavoidable, and because TD bootstraps, the inflated target is propagated into earlier states and accumulates. Arbitrarily poor states can end up estimated as high value, driving bad policy updates and divergence.

**The deterministic policy gradient and the actor-critic coupling.** For continuous actions one cannot enumerate $\max_a Q$. The deterministic policy gradient (Silver et al. 2014) instead moves a deterministic actor $\pi_\phi$ in the direction the critic says value increases:
$$\nabla_\phi J(\phi)=\mathbb E_{s\sim p_\pi}\big[\nabla_a Q^\pi(s,a)\big|_{a=\pi_\phi(s)}\,\nabla_\phi\pi_\phi(s)\big].$$
The critic supplies the gradient; the actor performs a soft, local maximization of $Q$. This creates a tight loop: the actor is only as good as the critic's gradient, and the critic is trained on transitions generated by the actor. If the critic is wrong in a region, the actor is pulled toward exactly the actions the critic overrates, generating more data there and reinforcing the error.

**Accumulation of bootstrapped error.** Because each TD update fits an estimate to a target built from another estimate, the Bellman equation is never satisfied exactly; each update leaves a residual TD-error $\delta(s,a)$, so $Q_\theta(s,a)=r+\gamma\,\mathbb E[Q_\theta(s',a')]-\delta(s,a)$. Unrolling shows the learned value is the expected return minus the expected discounted sum of *future* TD-errors, so its variance grows with the variance of those errors and, for large $\gamma$, can grow rapidly across updates. A single minibatch step only reduces error inside that minibatch and says nothing about errors elsewhere in state space. Two-timescale stochastic-approximation analyses of actor-critic (Konda & Tsitsiklis 2003) require, for convergence in the linear case, that the critic be updated on a faster timescale than the actor — the value estimate should be near-converged relative to the slowly moving policy.

**Diagnostic observations about the existing system.** Three measurements about the standard continuous-control actor-critic motivate the work. First, tracking its value estimate against a Monte-Carlo estimate of the true value (average discounted return over many episodes from buffer states) while it learns on continuous-control benchmarks shows a clear, persistent overestimation building over a million steps. Second, the discrete-action remedy of decoupling selection from evaluation, when transcribed to actor-critic, fails to remove that overestimation. Third, examining the value estimate with and without slow target networks shows that with a *fixed* policy all target-update rates converge similarly (faster targets merely more volatile), but with a *learned* policy a fast-updating target diverges — pinpointing divergence to policy updates made against a high-variance value estimate.

## Baselines

**Q-learning with function approximation.** The base value-learning method: fit $Q_\theta$ by TD regression to $y=r+\gamma\max_{a'}Q_{\theta'}(s',a')$. Provably convergent tabularly, but the $\max$ over a noisy approximator is the source of overestimation, and there is no $\max$ available in continuous action spaces.

**Double Q-learning (van Hasselt 2010).** Removes the maximization bias by decoupling the *selection* of the greedy action from its *evaluation*. Maintain two independent estimators $Q^A,Q^B$ and update each using the other:
$$Q^A(s,a)\leftarrow Q^A(s,a)+\alpha\big[r+\gamma\,Q^B\big(s',\textstyle\arg\max_{a'}Q^A(s',a')\big)-Q^A(s,a)\big],$$
and symmetrically for $Q^B$. The action is chosen by $Q^A$ but valued by $Q^B$; since $Q^B$ did not select it, $\mathbb E[Q^B(s',a^*)]\le\max_{a'}Q^B(s',a')$ and the systematic inflation is gone. Its gap: the unbiasedness relies on the two estimators being genuinely independent.

**Double DQN (van Hasselt et al. 2016).** The deep, single-network realization of the same idea: reuse the existing target network as the second estimator — select with the online network, evaluate with the target network, $y=r+\gamma Q_{\theta'}\big(s',\arg\max_{a'}Q_\theta(s',a')\big)$. Cheap and effective with discrete actions. Its gap when carried to actor-critic: the role of $\arg\max$ is played by the slowly-changing policy, so the action the current policy selects and the action the target network would prefer are nearly the same — the two estimates are too similar to be independent, and the bias survives.

**Deep deterministic policy gradient (Lillicrap et al. 2015).** The state-of-the-art continuous-control actor-critic and the method to beat. It combines the deterministic policy gradient with deep-Q machinery: a deep actor $\pi_\phi$ and a single deep critic $Q_\theta$, both with soft-updated target networks $\pi_{\phi'},Q_{\theta'}$; an experience replay buffer for off-policy minibatch training; exploration by adding temporally-correlated Ornstein–Uhlenbeck noise to actions. The critic minimizes the mean-squared Bellman error to $y=r+\gamma Q_{\theta'}(s',\pi_{\phi'}(s'))$; the actor ascends $Q_\theta$ by the deterministic policy gradient. Its gaps: a single critic is subject to overestimation with no mechanism against it; nothing addresses the variance accumulated by bootstrapping or the tight actor-critic feedback loop; deterministic targets concentrate on narrow value peaks; and it is notoriously sensitive to hyperparameters and random seeds.

**SARSA / Expected SARSA (Sutton & Barto; van Seijen et al. 2009).** On-policy TD control whose target uses the action actually taken (or an average over the policy's action distribution) rather than a maximizing action: $y=r+\gamma Q(s',a')$, $a'\sim\pi$. Values learned this way are "safer" — they account for the consequences of perturbed actions rather than assuming the single best action is always executed. Not directly a competitor for continuous control, but the source of the idea that a value target can be deliberately smoothed over nearby actions.

## Evaluation settings

The natural yardstick is the suite of continuous-control tasks in the MuJoCo physics simulator (Todorov et al. 2012) accessed through OpenAI Gym (Brockman et al. 2016): locomotion and control environments such as HalfCheetah, Hopper, Walker2d, Ant, Reacher, InvertedPendulum, and InvertedDoublePendulum, each with continuous state and action vectors and the environment's native reward, used without modification. The protocol that supports reproducible comparison: run each task for one million environment time steps; periodically evaluate (every few thousand steps) by averaging episode return over several episodes with exploration noise switched off; report across many random seeds of both the simulator and the network initialization, because prior reproducibility studies (Henderson et al. 2017) showed deep-RL results swing dramatically with seed. Natural comparison algorithms are the standard continuous-control actor-critic and the state-of-the-art on-policy policy-gradient methods, run from a common benchmarking codebase. For diagnostics, value accuracy is measured by comparing the critic's average estimate over a fixed set of states to a Monte-Carlo estimate of the true value (average discounted return over many rollouts of the current policy from states drawn from the replay buffer).
