The thing that bothers me is not an algorithm but a definition. The most prominent objectives in modern machine learning are not minimizations but minimax problems $\min_{\mathbf x\in\mathcal X}\max_{\mathbf y\in\mathcal Y} f(\mathbf x,\mathbf y)$ — GANs (generator vs. discriminator), adversarial training (classifier vs. perturbation), multi-agent RL — and in all of them $f$ is nonconvex–nonconcave. In practice we attack them with gradient descent ascent (GDA), descending on $\mathbf x$ and ascending on $\mathbf y$, and the iterates sometimes settle to a point. And I cannot say what kind of point that is. In ordinary minimization the question has a crisp answer: gradient descent finds a local minimum, a local minimum is $\nabla f=0$ with $\nabla^2 f\succeq0$, and a local minimum always exists because a global minimum is one. For minimax there is no agreed local object at all. The naive candidate, the local Nash equilibrium studied by Daskalakis–Panageas, Mazumdar–Ratliff, and Adolphs et al. — demand $f(\mathbf x^\star,\mathbf y)\le f(\mathbf x^\star,\mathbf y^\star)\le f(\mathbf x,\mathbf y^\star)$ in a $\delta$-ball, giving stationarity plus $\nabla^2_{\mathbf x\mathbf x}f\succeq0$, $\nabla^2_{\mathbf y\mathbf y}f\preceq0$ — fails me three ways. It need not exist on a perfectly nice function: for $f(x,y)=\sin(x+y)$ every stationary point has $\nabla^2_{xx}f=\nabla^2_{yy}f=\pm1$ with the *same* sign, so it can never be simultaneously a local min in $x$ and a local max in $y$, and there is no local Nash anywhere. It is order-blind: its two block conditions are symmetric in $\mathbf x$ and $\mathbf y$ and the cross term $\nabla^2_{\mathbf x\mathbf y}f$ never appears, yet $\min\max$ explicitly puts $\mathbf x$ first and $\mathbf y$ second. And it is the wrong shape for GDA: Daskalakis–Panageas proved $\textsf{local-Nash}\subsetneq\textsf{GDA-stable}$, so GDA stably parks at points that are not local Nash, and nobody could name what those points were.

The reason local Nash answers the wrong question is that it localizes a *simultaneous*-game object. In the convex–concave world we could ignore the order, because von Neumann's minimax theorem gives $\min\max=\max\min$ (Sion later relaxed to quasiconvex–quasiconcave), so leader and follower are worth the same whichever moves first and Nash = saddle = optimum. But nonconvex–nonconcave $f$ has $\min\max\neq\max\min$ in general, so which player commits first genuinely changes the answer, and a symmetric notion is structurally incapable of capturing it. The applications are sequential: a GAN's discriminator reacts to the current generator; adversarial training wants a classifier robust to a perturbation chosen *after* the classifier is fixed. So the correct global object is the Stackelberg solution — the leader minimizing the follower's worst-case value $\phi(\mathbf x):=\max_{\mathbf y}f(\mathbf x,\mathbf y)$, with $\mathbf y^\star$ a global max of $f(\mathbf x^\star,\cdot)$ and $\mathbf x^\star$ a global min of $\phi$. This global minimax point always exists ($\phi$ is continuous on a compact domain), but finding it is NP-hard. So just as plain optimization trades the global minimum for a local one, I need a *local surrogate for the global minimax point* — and that object does not yet exist. I have to define it.

I propose the notion of a **local minimax point**. The naive move — localize both the inner $\max$ and the outer $\min$ to the same $\delta$-ball — just gives local Nash back, because matching the radii collapses the asymmetry. The fix is to let the leader's perturbation have radius $\delta$ but let the follower range over a radius $h(\delta)$ that I get to choose, subject only to $h(\delta)\to0$ as $\delta\to0$. Concretely: $(\mathbf x^\star,\mathbf y^\star)$ is a local minimax point if there exist $\delta_0>0$ and a function $h$ with $h(\delta)\to0$ as $\delta\to0$ such that for every $\delta\in(0,\delta_0]$ and every $(\mathbf x,\mathbf y)$ with $\|\mathbf x-\mathbf x^\star\|\le\delta$, $\|\mathbf y-\mathbf y^\star\|\le\delta$,
$$f(\mathbf x^\star,\mathbf y)\ \le\ f(\mathbf x^\star,\mathbf y^\star)\ \le\ \max_{\mathbf y':\,\|\mathbf y'-\mathbf y^\star\|\le h(\delta)} f(\mathbf x,\mathbf y').$$
The left inequality says $\mathbf y^\star$ is a local max of $f(\mathbf x^\star,\cdot)$: the follower is locally optimal at the committed $\mathbf x^\star$. The right inequality says that if the leader deviates to any $\mathbf x$ within $\delta$, the follower's *local* best response within radius $h(\delta)$ is still at least as good for the follower as the equilibrium value, so the leader cannot improve by deviating. The free $h$ is doing the real work. Because $h$ is arbitrary subject to $h(\delta)\to0$, the ratio $h(\delta)/\delta$ can be anything, even $\to\infty$: the follower's reaction neighborhood is allowed to dwarf the leader's step. That free ratio is the leader-commits / follower-reacts asymmetry; sending $h\to0$ keeps the notion a truly local, infinitesimal-neighborhood property of $f$, which is exactly what Evtushenko's 1974 "local" notion fails to be (his point qualifies only if it is the global minimax on *some* neighborhood $\mathcal W$, and enlarging $\mathcal W$ can flip the verdict using values of $f$ far away). An equivalent Stackelberg-flavored form: with $g_\epsilon(\mathbf x):=\max_{\|\mathbf y-\mathbf y^\star\|\le\epsilon}f(\mathbf x,\mathbf y)$, the point is local minimax iff $\mathbf y^\star$ is a local max of $f(\mathbf x^\star,\cdot)$ and there is $\epsilon_0>0$ with $\mathbf x^\star$ a local min of $g_\epsilon$ for *every* $\epsilon\in(0,\epsilon_0]$ — the "for every small $\epsilon$" being the same freedom as the arbitrary $h$. This is a strict weakening of local Nash: taking $h\equiv0$ collapses $\max_{\|\mathbf y'-\mathbf y^\star\|\le0}f(\mathbf x,\mathbf y')=f(\mathbf x,\mathbf y^\star)$, so the local Nash inequalities *are* the local minimax inequalities, hence every local Nash is local minimax; the converse fails at the origin of $f=-x^2+5xy-y^2$, where $\nabla^2_{xx}f=-2<0$ kills local Nash but the strong cross-coupling $5xy$ lets the follower's reaction overwhelm the leader's own negative curvature, so the origin is local minimax.

What makes the notion usable is that it comes with first- and second-order conditions, the analogue of $\nabla f=0$, $\nabla^2 f\succeq0$. Writing $\mathbf A=\nabla^2_{\mathbf x\mathbf x}f$, $\mathbf B=\nabla^2_{\mathbf y\mathbf y}f$, $\mathbf C=\nabla^2_{\mathbf x\mathbf y}f$ at the point, first order it is just stationary, $\nabla_{\mathbf x}f=\nabla_{\mathbf y}f=0$: $\nabla_{\mathbf y}f=0$ because $\mathbf y^\star$ is a local max of $f(\mathbf x^\star,\cdot)$, and $\nabla_{\mathbf x}f=0$ because the local-minimax bound applied at the follower's best response $\boldsymbol\delta_y^\star(\boldsymbol\delta_x)$ (whose norm vanishes with $\delta$) forces $0\le\nabla_{\mathbf x}f(\mathbf x^\star,\mathbf y^\star)^\top\boldsymbol\delta_x+o(\|\boldsymbol\delta_x\|)$ for both signs of $\boldsymbol\delta_x$. The second-order condition is where the order finally enters the algebra. After stationarity the expansion is $\tfrac12\boldsymbol\delta_x^\top\mathbf A\boldsymbol\delta_x+\boldsymbol\delta_x^\top\mathbf C\boldsymbol\delta_y+\tfrac12\boldsymbol\delta_y^\top\mathbf B\boldsymbol\delta_y$. Setting $\boldsymbol\delta_x=0$ gives $\mathbf B\preceq0$ — the follower's half. For the leader's half, suppose $\mathbf B\prec0$; then the inner objective in $\boldsymbol\delta_y$ is strictly concave, maximized at the interior point $\boldsymbol\delta_y^\star=-\mathbf B^{-1}\mathbf C^\top\boldsymbol\delta_x$ (norm $O(\|\boldsymbol\delta_x\|)$, so within the allowed inner radius), and substituting it back gives inner-max value
$$\tfrac12\boldsymbol\delta_x^\top\big(\mathbf A-\mathbf C\mathbf B^{-1}\mathbf C^\top\big)\boldsymbol\delta_x+o(\|\boldsymbol\delta_x\|^2),$$
where the two $\mathbf B^{-1}$ terms combine as $-\boldsymbol\delta_x^\top\mathbf C\mathbf B^{-1}\mathbf C^\top\boldsymbol\delta_x+\tfrac12\boldsymbol\delta_x^\top\mathbf C\mathbf B^{-1}\mathbf C^\top\boldsymbol\delta_x=-\tfrac12\boldsymbol\delta_x^\top\mathbf C\mathbf B^{-1}\mathbf C^\top\boldsymbol\delta_x$. The leader's local-min requirement makes this $\ge0$ for all small $\boldsymbol\delta_x$, so the necessary condition is that the *Schur complement* of $\mathbf B$,
$$\mathbf A-\mathbf C\mathbf B^{-1}\mathbf C^\top=\nabla^2_{\mathbf x\mathbf x}f-\nabla^2_{\mathbf x\mathbf y}f\,(\nabla^2_{\mathbf y\mathbf y}f)^{-1}\nabla^2_{\mathbf y\mathbf x}f\ \succeq\ 0,$$
is positive semidefinite; running the same evaluation forward, a stationary point with $\mathbf B\prec0$ and Schur complement $\succ0$ *is* a strict local minimax point, so for nondegenerate $\mathbf B$ the conditions match up to the boundary. Compared against local Nash's sufficient pair $\mathbf A\succ0$, $\mathbf B\prec0$, local minimax replaces $\mathbf A\succ0$ by $\mathbf A-\mathbf C\mathbf B^{-1}\mathbf C^\top\succ0$: the cross term $\mathbf C$ now enters, and the condition is no longer symmetric — the Schur complement is taken of the *follower* block $\mathbf B$ and baked into the *leader's* test (a local maximin would give $\mathbf B-\mathbf C^\top\mathbf A^{-1}\mathbf C\preceq0$). Since $-\mathbf C\mathbf B^{-1}\mathbf C^\top\succeq0$ when $\mathbf B\prec0$, the leader may carry negative curvature provided the follower's reaction supplies enough convexity — exactly the $f=-x^2+5xy-y^2$ case, now read off the Hessian as $\mathbf A=-2$, $\mathbf B=-2$, $\mathbf C=5$, Schur $=-2-25/(-2)=10.5>0$.

I should be honest about the boundaries. Local minimax can be *absent* ($f=y^2-2xy$ on a box: $f(x^\star,\cdot)$ is convex in $y$ so its local maxima sit at corners and no leader condition survives), and unlike minimization a *global* minimax point can be neither local minimax nor even stationary — for $f=0.2xy-\cos y$ the global solutions $(0,\pm\pi)$ have gradient $(\pm0.2\pi,0)\neq0$, because the global object minimizes a global inner max while local minimax minimizes a local one. Existence is recovered under regularity: if $f(\mathbf x,\cdot)$ is strongly concave near its maxima with all local maxima global, the implicit best response is well-defined and the global minimax point is also local minimax.

The payoff is that this definition finally explains GDA. Run GDA with a timescale ratio — leader step $\eta/\gamma$, follower step $\eta$ — giving the flow $\dot{\mathbf x}=-\tfrac1\gamma\nabla_{\mathbf x}f$, $\dot{\mathbf y}=\nabla_{\mathbf y}f$, whose linearization at a stationary point has Jacobian $\mathbf J_\gamma=\begin{pmatrix}-\tfrac1\gamma\mathbf A & -\tfrac1\gamma\mathbf C\\ \mathbf C^\top & \mathbf B\end{pmatrix}$, stable iff every eigenvalue has negative real part. Fixed $\gamma$ has no clean relation to the notion (explicit quadratics are local minimax but GDA-unstable, and vice versa), because the off-diagonal blocks have opposite signs so the dynamics rotate. But push $\gamma\to\infty$, i.e. $\epsilon=1/\gamma\to0$ with the leader frozen relative to the follower. The eigenvalues of $\mathbf J_\epsilon=\begin{pmatrix}-\epsilon\mathbf A & -\epsilon\mathbf C\\ \mathbf C^\top & \mathbf B\end{pmatrix}$ then split into two scales: $d_2$ values near $\mathrm{eig}(\mathbf B)$, and — after rescaling $\lambda=\epsilon\theta$ and block-triangularizing via the same Schur complement — $d_1$ values $\approx-\epsilon\,\mathrm{eig}(\mathbf A-\mathbf C\mathbf B^{-1}\mathbf C^\top)$. So for large $\gamma$ strict stability holds iff $\mathbf B\prec0$ and $\mathbf A-\mathbf C\mathbf B^{-1}\mathbf C^\top\succ0$ — exactly the strict local minimax conditions. With $\underline{\infty\text{-GDA}}=\liminf_\gamma$ and $\overline{\infty\text{-GDA}}=\limsup_\gamma$ of the stable sets,
$$\textsf{local-minimax}\subset\underline{\infty\text{-GDA}}\subset\overline{\infty\text{-GDA}}\subset\textsf{local-minimax}\cup\{\text{stationary},\ \nabla^2_{\mathbf y\mathbf y}f\ \text{degenerate}\},$$
so $\infty$-GDA $=$ local minimax away from the degenerate follower-Hessian boundary. This is not a trick: the static definition demanded the follower solve its local max for every inner radius before the leader's perturbation matters, and that is precisely the leader drifting infinitely slower than the follower, with the fast subsystem solving the inner problem and the slow subsystem descending the implicit $\phi$. The two-scale eigenvalue split is the linear-algebra shadow of that two-timescale picture, and it gives every nondegenerate stable limit point of GDA, at last, a meaning: those mysterious stable-but-not-Nash points were always the local minimax points.

```python
import numpy as np

def _as_2d(matrix):
    return np.atleast_2d(np.asarray(matrix, dtype=float))

def schur_complement(A, B, C):
    A, B, C = map(_as_2d, (A, B, C))
    return A - C @ np.linalg.solve(B, C.T)

def is_negative_definite(matrix, tol=1e-9):
    return np.max(np.linalg.eigvalsh(_as_2d(matrix))) < -tol

def is_positive_definite(matrix, tol=1e-9):
    return np.min(np.linalg.eigvalsh(_as_2d(matrix))) > tol

def is_stationary(grad_x_value, grad_y_value, tol=1e-8):
    return np.linalg.norm(grad_x_value) < tol and np.linalg.norm(grad_y_value) < tol

def second_order_test(A, B, C, tol=1e-9):
    """Strict Hessian test: B < 0 and A - C B^{-1} C^T > 0."""
    B = _as_2d(B)
    if not is_negative_definite(B, tol):
        return False
    return is_positive_definite(schur_complement(A, B, C), tol)

def is_local_optimum(grad_x_value, grad_y_value, A, B, C, tol=1e-8):
    return (
        is_stationary(grad_x_value, grad_y_value, tol)
        and second_order_test(A, B, C, tol)
    )

def gda_step(x, y, grad_x, grad_y, eta):
    return x - eta * grad_x(x, y), y + eta * grad_y(x, y)

def order_aware_step(x, y, grad_x, grad_y, eta, ratio):
    return x - (eta / ratio) * grad_x(x, y), y + eta * grad_y(x, y)

def flow_jacobian(A, B, C, ratio):
    A, B, C = map(_as_2d, (A, B, C))
    return np.block([[-(1.0 / ratio) * A, -(1.0 / ratio) * C],
                     [                 C.T,                  B]])

def linearly_stable(jacobian, tol=1e-9):
    return np.all(np.real(np.linalg.eigvals(jacobian)) < -tol)
```
