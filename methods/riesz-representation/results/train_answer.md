I will state and explain the positive Riesz representation theorem, which gives the canonical name Riesz representation for the correspondence between positive linear functionals on continuous functions and regular Borel measures. Let X be a compact Hausdorff space, and let C(X, R) be the real Banach space of continuous real-valued functions on X equipped with the supremum norm. Suppose I:C(X, R) -> R is a positive linear functional, meaning that whenever f is nonnegative everywhere on X, the value I(f) is also nonnegative. The theorem asserts that there exists a unique finite regular Borel measure mu on X such that I(f) equals the integral of f with respect to mu for every continuous f. Moreover, the operator norm of I is exactly I(1), which is also mu(X), so the total mass of the representing measure is precisely the norm of the functional.

The first observation is that positivity is much stronger than it appears at first glance. For any continuous function f, the uniform norm lets us sandwich f between negative and positive multiples of the constant function 1. Because I preserves order, applying I to that sandwich shows that the absolute value of I(f) is bounded by I(1) times the supremum norm of f. Therefore positivity already implies continuity, and the operator norm of I is at most I(1). Since the constant function 1 has norm 1 and I(1) is positive, the norm is exactly I(1). This means the analytic boundedness of I is encoded in the single order condition rather than being an extra hypothesis.

The real difficulty is constructing the measure from I. The naive idea would be to define the mass of a set E as I of its indicator function, but indicators of arbitrary sets are usually not continuous, so I cannot be applied to them directly. Instead, open sets are approximated from the inside by continuous functions. For an open set U, I define mu(U) as the supremum of I(f) over all continuous functions f satisfying 0 <= f <= 1 whose support is contained in U. This definition only uses values of I on continuous functions, and it is finite because every admissible f is bounded by 1, so I(f) <= I(1). Intuitively, these functions are smooth bump functions that live inside U and try to become the indicator of U.

For an arbitrary set E, I define an outer value mu*(E) as the infimum of mu(U) over all open sets U containing E. This is not a separate regularity property added at the end; it is forced by the fact that I only sees continuous functions and therefore only sees rough sets through their open neighborhoods. The construction then proceeds by showing that mu* is a genuine outer measure and that every open set is Caratheodory measurable. The compactness of X is essential here. If a continuous function is supported in a countable union of open sets, its support is compact, so only finitely many of those open sets are needed to cover it. A finite partition of unity decomposes the function into nonnegative continuous pieces supported in individual open sets, and positivity plus linearity of I gives countable subadditivity. Urysohn's lemma supplies the separating functions needed to show that open sets are measurable, and once open sets are measurable, all Borel sets are measurable because the Borel sigma-algebra is generated by open sets.

The resulting Borel measure mu is regular. Outer regularity is built into the definition through the infimum over open supersets. Inner regularity follows because every continuous test function inside an open set U has compact support K contained in U, and the value I(f) cannot exceed mu(K). Taking the supremum over all such test functions gives mu(U) as a supremum over compact subsets of U. For a general Borel set E, finiteness of mu(X) lets me apply outer regularity to the complement of E inside the compact space X, producing a compact subset of E whose mass approximates mu(E) from below. Thus the measure is determined both from outside by open sets and from inside by compact sets.

To verify that I is indeed integration against mu, I approximate a nonnegative continuous function f by simple functions built from its level sets. The lower and upper level sets of f are not necessarily open or closed in a way that gives continuous indicators, but their thickened versions are. By partitioning the range of f into small intervals and using Urysohn functions squeezed between compact inner approximations and open outer approximations of the level sets, I obtain lower and upper sums. Positivity of I squeezes I(f) between these sums, while the measure integral is squeezed by the same sums. As the mesh of the partition goes to zero, both bounds converge to the same value, proving that I(f) equals the integral of f with respect to mu. For a general continuous function, I write it as the difference of its positive and negative parts and use linearity.

Uniqueness is a consequence of the same continuous approximation of indicators. If two finite regular Borel measures give the same integral against every continuous function, then for any compact set K contained in an open set U, Urysohn's lemma provides a continuous function g with 1_K <= g <= 1_U. Equality of the integrals of g with respect to both measures implies that mu(K) is at most nu(U). Taking the infimum over all open U containing K and using regularity gives mu(K) <= nu(K), and symmetry gives equality on compact sets. Inner regularity then extends equality to open sets, and outer regularity extends it to all Borel sets. Therefore the representing measure is unique.

The theorem is foundational because it identifies the dual object corresponding to a positive functional. In finite dimensions, every linear functional is a dot product with a fixed vector. In this infinite-dimensional setting, the analog of that vector is a regular Borel measure, and the analog of the dot product is integration. The result explains why measure theory is the natural language for positive linear functionals on spaces of continuous functions and why regularity is not an optional technical condition but a consequence of the fact that the functional is only defined on continuous tests.

The following Python script illustrates the theorem in the concrete case where X is the unit interval [0, 1]. I take I to be Riemann integration, which is a positive linear functional on continuous functions. The script constructs the representing measure by evaluating I on continuous bump functions supported inside open sets, compares the resulting masses against the lengths of those open sets, and checks the norm identity I(1) equals the total measure of the interval.

```python
import numpy as np
from scipy.integrate import quad

def bump(t, a, b, c, d):
    """Continuous piecewise-linear bump: 0 outside [a,d], 1 on [b,c], linear ramps."""
    t = np.asarray(t, dtype=float)
    scalar = t.ndim == 0
    t = np.atleast_1d(t)
    result = np.zeros_like(t, dtype=float)
    left_ramp = (t > a) & (t < b)
    right_ramp = (t > c) & (t < d)
    flat = (t >= b) & (t <= c)
    if b != a:
        result[left_ramp] = (t[left_ramp] - a) / (b - a)
    if d != c:
        result[right_ramp] = (d - t[right_ramp]) / (d - c)
    result[flat] = 1.0
    return result.item() if scalar else result

def I(func):
    """Positive linear functional: Riemann/Lebesgue integral over [0,1]."""
    val, _ = quad(func, 0, 1, limit=100)
    return val

# The interval X = [0,1] is compact Hausdorff.
# Choose an open set U = (0.2, 0.7).
a, d = 0.2, 0.7
# Approximate 1_U from inside by a sequence of bumps whose support is contained in U.
widths = [0.15, 0.10, 0.05, 0.02, 0.005]
estimates = []
for w in widths:
    b = a + w
    c = d - w
    if b >= c:
        continue
    f = lambda t, b=b, c=c: bump(t, a, b, c, d)
    estimates.append(I(f))

print("Estimated mu(U) from inner bumps:", estimates)
print("Limit should be length of U:", d - a)

# Verify the norm identity ||I|| = I(1) = mu(X).
one_func = lambda t: np.ones_like(t)
total_mass = I(one_func)
print("I(1) =", total_mass)
print("mu(X) = length of [0,1] =", 1.0)

# Verify representation on a generic continuous function.
g = lambda t: np.sin(2 * np.pi * t) + 2.0  # nonnegative on [0,1]
functional_value = I(g)
measure_value, _ = quad(g, 0, 1, limit=100)
print("I(g) =", functional_value)
print("integral g dmu =", measure_value)
print("Difference:", abs(functional_value - measure_value))
```

In summary, the positive Riesz representation theorem under the canonical name Riesz representation tells us that every positive continuous linear functional on C(X, R) for a compact Hausdorff space X is exactly integration against a unique finite regular Borel measure, with total mass equal to the norm of the functional.
